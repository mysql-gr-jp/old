MySQL 8.0 : MySQL 8.0.12 リリース
投稿者： webmaster 投稿日時： 2018-7-27 8:12:00 (1197 ヒット)

MySQL 8.0.12 がリリースされました。
MySQL 8.0は現在の、MySQLの最新メジャー・シリーズです。

非常に多くの修正／追加機能があります。
変更点はこちらから：
https://dev.mysql.com/doc/relnotes/mysql/8.0/en/news-8-0-12.html

ダウンロード：
https://dev.mysql.com/downloads/mysql/8.0.html


また、MySQL 8.0.12 と同時に公開された、MySQL Workbench も、バージョンが 8.0.12 となって、このバージョンより GA となりました。
