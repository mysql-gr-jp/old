MySQL 8.0 : MySQL 8.0.1-dmr リリース
投稿者： webmaster 投稿日時： 2017-4-11 8:01:00 (721 ヒット)

MySQL 8.0.1-dmr がリリースされました。

これは、時期MySQLシリーズ（メジャーバージョン）である、MySQL 8.0 シリーズの開発途上版です。
（DMR: Development Milestone Release)

・新しいコレーション utf8mb4_ja_0900_as_cs　が追加されました。これを使用することにより、ハハパパ問題、およびいわゆる寿司ビール問題が解消されます。

・SQL標準で定められていたもののMySQLには実装されていなかった、CTE(Common Table Expressions;共通表式) が実装されました。WITH句が使用できます。


変更点（チェンジログ）：
https://dev.mysql.com/doc/relnotes/mysql/8.0/en/news-8-0-1.html

ダウンロード：
https://dev.mysql.com/downloads/mysql/8.0.html#downloads

MySQL 8.0 リファレンスマニュアル：
https://dev.mysql.com/doc/refman/8.0/en/
