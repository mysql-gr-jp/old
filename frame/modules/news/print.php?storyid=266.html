MySQL 5.7.6-m16 リリース

投稿日時 2015-3-9 7:06:00 | トピック： MySQL 5.7

MySQL 5.7.6 Milestone 16 がリリースされました。
これは、MySQL次期メジャーバージョン系列となる予定である MySQL 5.7 シリーズの、最新マイルストンリリースです。
前回のマイルストーン15から、約半年ぶりのリリースとなり、非常に多くの修正が加えられています。詳しくは、チェンジログをごらんください。


ダウンロード：
http://dev.mysql.com/downloads/mysql/


Linux環境の方は、Oracle MySQL チームが提供する以下のリポジトリも使用できます：
http://dev.mysql.com/downloads/repo/

チェンジログ：
http://dev.mysql.com/doc/relnotes/mysql/5.7/en/news-5-7-6.html
