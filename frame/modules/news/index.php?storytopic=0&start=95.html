MySQL 5.7 : MySQL 5.7.5-m15 リリース
投稿者： webmaster 投稿日時： 2014-9-25 7:05:00 (2201 ヒット)

MySQL 5.7.5 milestone 15 がリリースされました。

「milestone リリース」とは、安定稼働版（製品版やGAとも言う）の前段階のバージョンであり、大切な業務で使用する場合は、安定版である MySQL 5.6 シリーズの使用が推奨されています。

MySQL 5.7.5 では非常に多くの修正が加えられています。
修正内容は、以下のURLを参照ください。
http://dev.mysql.com/doc/relnotes/mysql/5.7/en/news-5-7-5.html


ダウンロードは、以下のURLより、「 Development Releases　」のタブをクリックするとリンクがあらわれます。
http://dev.mysql.com/downloads/mysql/

MySQL 5.6 : MySQL 5.6.21 リリース
投稿者： webmaster 投稿日時： 2014-9-23 6:21:00 (2552 ヒット)

MySQL 5.6.21 がリリースされました。

InnoDBを無効化するオプションが、本バージョンより「非推奨」扱いとなりました。ワーニングがでるようになります。
（--skip-innodb, --innodb=OFF, --disable-innodb などのオプション）
将来のMySQLのバージョンでは、これらのオプションはすべて廃止になる（＝InnoDBを無効化できない）予定とのことです。

その他、多くの修正が加えられています。詳しくはチェンジログをごらんください。

チェンジログ：
http://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-21.html

ダウンロード：
http://dev.mysql.com/downloads/mysql/
