MySQL : MySQL 4.1.19 リリース
投稿者： webmaster 投稿日時： 2006-5-5 10:11:28 (7732 ヒット)

MySQL 4.1.19 がリリースされました。

これまで mysql クライアントでは、接続時にパラメタでキャラクタセット名を指定していましたが、
\C の後に キャラクタセット名を入れることで、接続中でもキャラクタセットを変更できるようになりました。
（例： mysql> \C eucjpms ）
