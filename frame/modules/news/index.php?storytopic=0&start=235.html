MySQL 5.1 : MySQL 5.1.30-GA リリース！
投稿者： webmaster 投稿日時： 2008-11-28 3:59:58 (6080 ヒット)

MySQL 5.1 シリーズが、ようやく製品版（GA）になりました！！

http://dev.mysql.com/downloads/
からダウンロードできます。

MySQL 5.1 には以下のような新機能額見込まれています。
　　- パーティショニング（テーブルとインデックス）
　　- 行ベースレプリケーション および mixed レプリケーションreplication
　　- ジョブスケジューラー（ビルドイン）
　　- XPath のサポート
　　- New SQL diagnostic aids and performance utilities
　　- The return of the embedded library (libmysqld)

　ＭｙＳＱＬ5.1の新機能の詳細については、
http://dev.mysql.com/doc/refman/5.1/en/mysql-nutshell.html
　で見られます。

RC(リリース候補)版です。　今回のバージョンもまた、製品レベルのリリースではなくRC(リリース候補版)となりました。
