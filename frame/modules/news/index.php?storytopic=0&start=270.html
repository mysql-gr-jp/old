MySQL 6.0 : MySQL 6.0.4-alpha リリース
投稿者： webmaster 投稿日時： 2008-3-11 12:50:55 (9400 ヒット)

MySQL 6.0.4-alpha がリリースされました。

新しいトランザクショナルストレージエンジンである falcon の開発がこのメジャーバージョンで進行されている点で、注目の新バージョンです。　200項目以上にわたる修正がなされています。
4バイト utf8 (rfc3629) にもこのバージョンで対応したようですね。


○MySQL 6.0 についてはこちら:
http://www.mysql.com/mysql60/

○ダウンロードはこちら：
http://dev.mysql.com/downloads/mysql/6.0.html

MySQL : 【速報】Sun が MySQL社を買収へ
投稿者： webmaster 投稿日時： 2008-1-17 2:40:15 (12915 ヒット)

先ほど、Sun Microsystems 社が MySQL 社を買収することで両者が合意したとの発表がありました。


○Sun Microsystems Announces Agreement to Acquire MySQL, Developer of the World's Most Popular Open Source Database
http://www.sun.com/aboutsun/pr/2008-01/sunflash.20080116.1.xml
http://www.mysql.com/news-and-events/sun-to-acquire-mysql.html

もちろん、即買収完了というわけではなく、Sun の2008年度 3Q から　4Q にかけて買収完了を目指すというスケジュールのようです。

MySQLユーザの立場から見ると、Sun社がこの買収により、MySQLをどのようにしていくつもりなのかが気になるところです。　
続報に注目ですね。

------
追記：
　Sun Microsystems は 2008年１月より「2008年第3クォータ」が始まっているそうで、だとすると 3Q～4Q にかけて買収完了というのは意外と早そうですね。
