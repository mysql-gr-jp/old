MySQL 6.0 : MySQL 6.0.11-alpha リリース
投稿者： webmaster 投稿日時： 2009-5-25 15:49:08 (11269 ヒット)

　MySQL 6.0.11-alpha がリリースされました。
　これは、MySQL 6.0 シリーズ（現在アルファー版）の最新版です。


ダウンロードはこちらから：
http://dev.mysql.com/downloads/mysql/6.0.html


　
　今回も数多くの修正がなされています。今回もレプリケーションに関する修正が目立ちますが、それ以外でおもしろそうなものとしては、mysqlbackup コマンドの追加が挙げられます。
　mysqlbackup コマンドは、BACKUP DATABASE 文で作成したバックアップファイルの情報を表示するもののようです。

　また、今後のMySQL 6.0シリーズのリリース予定について、以下のアナウンスがなされています。
「9月までのマイルストンとしては、6.0の大部分の機能を実装したいと考えている。もっと詳細な情報は 6月末までに公表する。マイルストンに乗らなかった機能（オンラインバックアップ、falcon、コネクションプーリング、4 byte utf)も安定版リリースまでには実装する」

MySQL : MySQL 5.4.0-beta リリース
投稿者： webmaster 投稿日時： 2009-4-22 11:09:12 (4252 ヒット)

驚くべき発表がなされました。（買収以外の話で）

昨日、MySQLの新しいメジャーバージョン系列である MySQL　5.4 が発表されました。
バージョン番号は 5.4.0-beta 。　alphaを飛ばしてのいきなりの beta版です。

MySQL 5.4 シリーズは MySQL 5.1 をベースにして、スケーラビリティやパフォーマンスに大きな影響のある「ちょっとした」変更を加えたもののようです。

主な特徴としては：
　・スケーラビリティの改良　（InnoDB でのCPUスケール）
　・サブクエリの最適化
　・新クエリアルゴリズム（特にMySQL Cluster での join の改良）
　・ストアドプロシジャーの改良
　・プリペアドステートメントの改良
　・情報スキーマ(INFORMATION SCHEMA)の改良　（ODBCやJDBCからもっと情報が取れるように）
　・DTraceのサポート（Solaris上でのトラブルシュートが便利に）

とアナウンスされています。
GA版のリリース時期（見積）は年内（later this year）を予定しているとのこと。

　
MySQL 5.4 シリーズの詳しい情報はこちらから：
http://dev.mysql.com/doc/mysql-5.4-features/en/index.html


ダウンロードはこちらから：
http://dev.mysql.com/downloads/mysql/5.4.html
※ソースコードおよび、Solaris, Linux の64bit対応バイナリのみが公開されています
