MySQL 5.6.3-m6　がリリースされました。

これは、MySQL 5.6 シリーズの最新マイルストンリリースです。
マイルストンリリースなので、ベータ版レベルと考えれば良いでしょう。

MySQL 5.6 シリーズには多くの革新的な機能が含まれています。
・マルチスレッド・スレーブ
・オプティマイザの進化
・パフォーマンススキーマ項目の追加


○ダウンロードはこちらから（Development　release のタブをクリックしてください）：
http://dev.mysql.com/downloads/mysql/

○チェンジログはこちら（非常にたくさんの項目があります！）：

MyNA News : OSC2011-Okinawa に参画します(9/10)
投稿者： webmaster 投稿日時： 2011-9-1 9:19:07 (1135 ヒット)

9月10日(土)に沖縄で開催される
『オープンソースカンファレンス2011-沖縄』(OSC2011-Okinawa) に、
日本MySQLユーザ会として参加いたします。

ブース出展（MySQLよろず談話コーナー。お気軽にお立ち寄りください）のほか、
オラクルのかじやまりゅうすけさんが、『MySQLでNoSQL - 次世代のMySQLはRDBMSなのにNoSQLもあり!?』と題したセミナーを行います。
　MySQLのNoSQL的利用について、まとまった話を聞けるのは本邦初（当会調べ）ですので、ご興味の方は観光も兼ねて、ぜひご参加いただければと思います。

■セミナー内容:
　次期バージョンのMySQL5.6やMySQL　Cluster7.2で実装される、memcachedインターフェースなどのMySQLでのNoSQL実装について、開発ロードマップとあわせてご紹介いたします。
