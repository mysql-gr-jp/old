MySQL 5.6 : MySQL 5.6.21 リリース
投稿者： webmaster 投稿日時： 2014-9-23 6:21:00 (2944 ヒット)

MySQL 5.6.21 がリリースされました。

InnoDBを無効化するオプションが、本バージョンより「非推奨」扱いとなりました。ワーニングがでるようになります。
（--skip-innodb, --innodb=OFF, --disable-innodb などのオプション）
将来のMySQLのバージョンでは、これらのオプションはすべて廃止になる（＝InnoDBを無効化できない）予定とのことです。

その他、多くの修正が加えられています。詳しくはチェンジログをごらんください。

チェンジログ：
http://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-21.html

ダウンロード：
http://dev.mysql.com/downloads/mysql/

MySQL 5.6.20 がリリースされました。

リンクしている OpenSSL ライブラリのバージョンが 1.0.1g から 1.0.1h へと変更されています（セキュリティ上の修正）。（ただし、Oracleがビルドしたバイナリは、OpenSSLではなく yaSSL を使用しているので影響なし）
