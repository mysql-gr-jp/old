日本MySQLユーザ会は、2013年10月19,20の２日間の日程で東京で開催される
「オープンソースカンファレンス（ＯＳＣ）2013　東京／秋」に、
ブース出展、およびセミナー開催にて参画いたします。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
オープンソースカンファレンス2013　Tokyo/Fall
　　https://www.ospn.jp/osc2013-fall/

日程：2013年10月19日(土) 10:00～18:00
　　　　　　　　　　　　　20日(日) 10:00～17:00

会場：明星大学 日野キャンパス 28号館 2F
　　　(多摩モノレール 「中央大学・明星大学駅」から大学まで直結。会場まで徒歩6分）

費用：無料

内容：オープンソースに関する最新情報の提供
　　　・展示 - オープンソースコミュニティ、企業・団体による展示
　　　・セミナー - オープンソースの最新情報を提供

主催：オープンソースカンファレンス実行委員会

協賛：明星大学
　　　明星大学 情報学部

企画運営：株式会社びぎねっと
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝



■セミナーについて：
　日本MySQLユーザ会のセミナー枠では、メーリングリストなどでもお馴染みの yoku0825 さんがお話をしてくださいます。
『 あなたはどのmysqldがお好き？ ～MySQL, MariaDBからClusterまで～』
と題して、「MySQLファミリー」の様々なフォークについての内容です。
なかなか触れることのできない、貴重な比較情報となると思いますので、ご興味のある方はぜひ足をお運びください。　
日本MySQLユーザ会のセミナー枠は10/20（日）の朝10時からです。

「MySQL本流のMySQL, MariaDB, Percona ServerやDWHに特化したInfobright, InfiniDB、変り種としてのTokuDB、
名前は同じだけど毛色の違うMySQL Clusterに、独自の冗長構成を持たせたGalera Cluster。
MySQLだけじゃないMySQLをゆるーく紹介します。」（OSCサイトのセミナー案内文より）

2013年10月6日(日)に、広島市で開催される、オープンソースカンファレンス(OSC)2013-Hiroshima に、日本MySQLユーザ会（MyNA）は、ブース出展およびセミナー開催にて参画いたします。

－－－－－－－－－－－－－－－－－－－－－－－－
■開催概要
　オープンソースカンファレンス2013 Hiroshima
　　http://www.ospn.jp/osc2013-hiroshima/

日程：2013年10月6日(日) 10:00～17:00
会場：サテライトキャンパスひろしま（広島県民文化センター 5F）
　　　　広島市中区大手町1丁目5-3（広電最寄駅→紙屋町西 ・ 最寄バス停→紙屋町）
費用：無料
内容：オープンソースに関する最新情報の提供
　　　・展示 - オープンソースコミュニティ、企業・団体による展示
　　　・セミナー - オープンソースの最新情報を提供
主催：オープンソースカンファレンス実行委員会
協力：経済産業省 中国経済産業局
企画運営：株式会社びぎねっと


■セミナーについて
　　日本MySQLユーザ会（MyNA)の枠として、ユーザ会のやまさきよしあき氏が、12:00より、『MySQL最新動向と便利ツールMySQL Workbench』と題したセミナーを行います。
内容は、『今年リリースされたMySQL5.6、MySQL Cluster7.3の最新情報をご紹介します。また、MySQLデータベースの設計、開発、管理を効率的に行えるGUIツール、
MySQL Workbenchを合わせてご紹介します。』（OSCサイトより）ということで、お近くのかたは、ぜひ足をお運びくださいませ。
