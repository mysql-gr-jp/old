MySQL 5.6.3-m6 リリース

投稿日時 2011-10-4 12:29:35 | トピック： MySQL

MySQL 5.6.3-m6　がリリースされました。

これは、MySQL 5.6 シリーズの最新マイルストンリリースです。
マイルストンリリースなので、ベータ版レベルと考えれば良いでしょう。

MySQL 5.6 シリーズには多くの革新的な機能が含まれています。
・マルチスレッド・スレーブ
・オプティマイザの進化
・パフォーマンススキーマ項目の追加
