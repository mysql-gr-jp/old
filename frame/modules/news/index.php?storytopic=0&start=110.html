MySQL 5.5 : MySQL5.5.36 リリース
投稿者： webmaster 投稿日時： 2014-2-1 5:36:00 (2265 ヒット)

MySQL 5.5.36 が公開されました。

現在、MySQLは、MySQL 5.6 シリーズが最先端ですので、新規に利用を開始するなど、特に 5.5 にする理由がない場合は、5.6シリーズの使用をお勧めします。


ダウンロードはこちら：
http://dev.mysql.com/downloads/

チェンジログ（英語）はこちら：
http://dev.mysql.com/doc/relnotes/mysql/5.5/en/news-5-5-36.html

チェンジログ（日本語）：
http://openstandia.jp/oss_info/mysql/version-detail/20140131.html
