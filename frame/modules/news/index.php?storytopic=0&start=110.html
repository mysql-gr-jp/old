MySQL 5.5 : MySQL 5.5.43 リリース
投稿者： webmaster 投稿日時： 2015-4-6 5:43:00 (1163 ヒット)

MySQL 5.5.43 がリリースされました。

MySQL 5.5 シリーズは、現行「ひとつまえ」のMySQLバージョンシリーズです。これから採用される方は 5.6シリーズ以降の利用の検討をお勧めします。

MySQL 5.5 シリーズの通常のサポート終了(EoL)は、2015年12月となっています。Extended Support は 2018年12月。


ダウンロード：
http://dev.mysql.com/downloads/mysql/

チェンジログ：

MySQL 5.7 : MySQL 5.7.6-m16 リリース
投稿者： webmaster 投稿日時： 2015-3-9 7:06:00 (1087 ヒット)

MySQL 5.7.6 Milestone 16 がリリースされました。
これは、MySQL次期メジャーバージョン系列となる予定である MySQL 5.7 シリーズの、最新マイルストンリリースです。
前回のマイルストーン15から、約半年ぶりのリリースとなり、非常に多くの修正が加えられています。詳しくは、チェンジログをごらんください。

http://dev.mysql.com/downloads/mysql/


Linux環境の方は、Oracle MySQL チームが提供する以下のリポジトリも使用できます：
