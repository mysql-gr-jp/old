MySQL バージョン 5.0.15 がリリースされました　（バージョン 5.0.14 はリリースされませんでした）。

MySQL 5.0 系列はこれまで、 alpha → beta → rc と開発が進められてきましたが、今回の 5.0.15 で、これらの記号がとれました。
つまり 「MySQL 5.0 系列の最初のリリース」 になります。

もちろん、各種業務でみなさんが本格的に使用されるのはこれからでしょうから、様々なノウハウの積み上げもこれからなされていくことになります。
実際に使用してみての、情報をぜひ メーリングリスト にお寄せください。　その情報は 日本の MySQLユーザ共通の財産となることでしょう。

MySQL : MySQL 5.0.13-rc リリース！
投稿者： webmaster 投稿日時： 2005-9-27 9:06:57 (7114 ヒット)

MySQL 5.0 系列の最新バージョン、 5.0.13 がリリースされました。

5.0.12 までは beta バージョンでしたが、この MySQL 5.0.13 からは rc 版（リリース候補版）となります。
rc 版は、従来の MySQL バージョン （4.1, 4.0, 3.23 etc) では gamma リリース と呼ばれていたものです。

　MySQL AB が 「疑いなく、 MySQL ５.０ は MySQL AB の今までに最も意欲的なリリースです。」 と説明するように、今までの MySQL とは大きく変わったことを実感できると思います。 その 「バージョン 5.0」 が　ついに rc 版となったのですから、わくわくしますね。
