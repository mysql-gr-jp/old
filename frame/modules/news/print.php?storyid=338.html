MySQL 8.0.11 GA リリース！

投稿日時 2018-4-19 8:11:00 | トピック： MySQL 8.0

MySQL 8.0 がついに GA 版（正式リリース）となりました。
MySQL 8.0.11 です。

前回リリースのバージョン番号 8.0.4-rc から一気に飛んでいますが、これは、MySQLの周辺ツール（Connector や Workbench など）とバージョン番号を合わせるためとのことです。 （8.0.5～8.0.10 はリリースされていません）

MySQL 8.0 の最新情報（What's New）は、MySQL Server Blogに詳しいです。
