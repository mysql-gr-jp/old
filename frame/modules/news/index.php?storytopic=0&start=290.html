MySQL : MySQL 5.1.7-beta リリース！
投稿者： webmaster 投稿日時： 2006-3-7 12:00:00 (2484 ヒット)

MySQL 5.1.7-beta がリリースされました。

特に文字コードの大きな変更によりバージョン5.0系列への移行がまだあまり行われていないように見える中、次のバージョン系列である 5.1系列が はやくも、beta になりました。


mysql_upgrade プログラムが提供されるようになったり、
近いうちに CREATE TABLE 時の TYPE が使用不可になる（ENGINE句を使う）ことなどの修正が行われています。

MySQL : MySQL 5.0.18 リリース！
投稿者： webmaster 投稿日時： 2006-1-1 18:04:17 (17661 ヒット)

２００５年の大晦日に、MySQL 5.0.18 がリリースされました。

テンポラリテーブルに対する　DESC や SHOW 文が正しい結果を返さなかった問題や、NULLを含む LEFT JOIN によりサーバクラッシュを引き起こす問題などが修正されています。
ぱっとみたところ、残念ながら日本語キャラクタセットに関する目立つ修正はこのバージョンでは行われていないようです。

MySQL : MySQL 5.1.4-alpha リリース！
投稿者： webmaster 投稿日時： 2005-12-23 22:05:19 (3531 ヒット)

MySQL 5.0系列の開発が盛んに進んでいるかと思えば、
こちら、 バージョン 5.1 系列の開発も活発です。

先ほど MySQL 5.1.4-alpha が公開されました。

　　http://dev.mysql.com/downloads/

バージョン 5.1 系列は、まだ alphan版です。今後改良が進められていくと考えられますので、最新の情報に真っ先に触れたいユーザは、ぜひお試しください。

なお バージョン 5.1 系列 の Windows 版ではいまのところ、「インストーラなし版」しか公開されていません。インストーラ付きは将来のバージョンでサポートする意向とのことです。
