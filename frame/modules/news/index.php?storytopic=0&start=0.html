今年も、日本オラクル株式会社　MySQL GBUさん主催のイベント MySQL Innovation day 2019 が開催されます。
10月7日東京、8日大阪の2カ所での開催です。
海外からもプロダクトマネージャやコミュニティマネージャも来日し、MySQLの最新状態に触れられる場になるかと思いますので、MySQLに興味のある方は以下リンク先から申し込んで参加されてみてはいかがでしょうか。
