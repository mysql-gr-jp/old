２００５年の大晦日に、MySQL 5.0.18 がリリースされました。

テンポラリテーブルに対する　DESC や SHOW 文が正しい結果を返さなかった問題や、NULLを含む LEFT JOIN によりサーバクラッシュを引き起こす問題などが修正されています。
ぱっとみたところ、残念ながら日本語キャラクタセットに関する目立つ修正はこのバージョンでは行われていないようです。

MySQL 5.0系列の開発が盛んに進んでいるかと思えば、
こちら、 バージョン 5.1 系列の開発も活発です。

先ほど MySQL 5.1.4-alpha が公開されました。

　　http://dev.mysql.com/downloads/

バージョン 5.1 系列は、まだ alphan版です。今後改良が進められていくと考えられますので、最新の情報に真っ先に触れたいユーザは、ぜひお試しください。

MySQL : MySQL 5.0.17 リリース

MySQL 5.0.17 がリリースされました。 5.0系としては約１ヶ月ぶりのリリースです。
