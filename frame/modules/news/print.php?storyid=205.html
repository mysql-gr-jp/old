MySQL 5.1.63 リリース

投稿日時 2012-5-7 16:03:00 | トピック： MySQL 5.1

MySQL 5.1.63 がリリースされました。


MySQL 5.1シリーズは、ひとつ前のメジャーバージョンですので、
これからMySQLを導入される方は、最新メジャーバージョンである、
MySQL 5.5 シリーズの使用をお勧めします。
