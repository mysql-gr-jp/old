投稿者： webmaster 投稿日時： 2014-7-31 6:20:00 (931 ヒット)

MySQL 5.6.20 がリリースされました。

リンクしている OpenSSL ライブラリのバージョンが 1.0.1g から 1.0.1h へと変更されています（セキュリティ上の修正）。（ただし、Oracleがビルドしたバイナリは、OpenSSLではなく yaSSL を使用しているので影響なし）
その他いくつもの修正が加わっていますので、詳しくはチェンジログを確認ください。


チェンジログ：
http://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-20.html
