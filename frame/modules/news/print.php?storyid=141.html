MySQL 5.4.1-beta リリース

投稿日時 2009-6-26 9:51:36 | トピック： MySQL

MySQL 5.4.1-beta がリリースされました。

MySQL 5.4 シリーズは、MySQL 5.1 をベースにして、主にハイスペックのサーバ環境でのスケーラビリティ向上を目指しているバージョンです。

今回リリースされた MySQL 5.4.1-beta は 前回の 5.4.0 からの実質的な変更はありませんが、前回は 「ソースコードと、Solaris, Linux の64bit対応バイナリのみ」の公開だったのに対して、今回は Windows を含む多くのプラットフォーム向けのバイナリが公開されています。
