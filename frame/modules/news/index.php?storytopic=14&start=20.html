MySQL 5.6 : MySQL 5.6.17 リリース
投稿者： webmaster 投稿日時： 2014-3-28 6:17:00 (3247 ヒット)

MySQL 5.6.17 がリリースされました。

MySQL 5.6 シリーズは、MySQLの各種メジャーバージョン（5.5, 5.6, 5.7など）の中で、最新のプロダクションリリース（ベータ版などではないもの；GA）ですので、今後MySQLを新たに導入される人には、この MySQL 5.6 シリーズの使用がお勧めされます。

○チェンジログ
http://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-17.html

○最新版のダウンロードはこちらから
http://dev.mysql.com/downloads/

○MySQL 5.6 について
http://dev.mysql.com/doc/refman/5.6/en/mysql-nutshell.html
MySQL 5.6.11 から、Windows パッケージは "full" インストーラーと "web" インストーラ尾の２つが提供されています。　web インストーラーは、「ダウンロード・オンデマンド」方式でファイルを取得しながらのインストールが行われるものです。

○MySQL 5.6 のインストールとアップグレードについては、こちら
http://dev.mysql.com/doc/refman/5.6/en/installing.html
