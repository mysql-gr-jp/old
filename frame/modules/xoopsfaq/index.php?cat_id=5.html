InnoDBのトランザクション分離レベル
InnoDBのトランザクション分離レベルは4つ
READ UNCOMMITTED
READ COMMITTED
REPEATABLE READ
SERIALIZABLE

標準は REPEATABLE READ。
InnoDBの REPEATABLE READは、phantom read は起きない実装。
