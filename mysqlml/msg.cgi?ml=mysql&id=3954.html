> gcc 2.7.2.3 でした。
> まさか、gcc をアップデートしないとコンパイルできないんでしょうか？(^^;

マニュアルを見ると、以下のように書いてありますよ。
素直に gcc をバージョンアップしましょう。

gcc >= 2.8.1, egcs >= 1.0.2, SGI C++, SunPro C++ は働くことが
知られているコンパイラのいくつかです。 libg++ は gcc 使用時には
必要ではありません。gcc 2.7.x は、`sql/sql_base.cc' のような、
完全に正当な C++ ファイルのいくつかをコンパイルできないという
バグがあります。gcc 2.7.x だけしかなければ、MySQL をコンパイル
できるように gcc をアップ グレードする必要があります。 
MySQL 3.23.x のコンパイル時は、gcc >= 2.95.2 を推奨します。
