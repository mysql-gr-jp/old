Subject: [mysql 05845] PHP3 が MySQL の関数を認識しない。

まさちと申します。ひとつお教え下さい。
当方PHPよりMySQLに作ったDBにアクセスを試みたんですが、
以下の内容のエラーが出てしまいます。

Fatal error: Call to unsupported or undefined function mysql_connect() in 
/usr/local/apache/htdocs/mysqllab/results.php on line 17

過去ログで調べましたところ、どうやら
「MySQLのPHP用モジュール（php-mysql）はインストールされていない」
のではないかと言う気がします。「PHPのconfigureで、 --with-mysql 」を
指定した記憶がありませんし、そもそもphpはPostgreSQLを使うために
インストールした物ですので、これ用のconfigurationにしかなってないと
思います（phpからPostgreSQLへのアクセスは問題ありません）。

そこで率直に言って、PHP3にMySQLの関数を認識させるのには
どうすればいいのでしょうか？過去ログからは自分が理解できるような
回答は見つけることが出来ませんでした。
「PHPのconfigureでの --with-mysql 」を後から補うことは出来るのでしょうか。
よろしくお願いします。


ちなみにphpinfo()によるconfigurationレポートでは
         Configure command: ./configure '--with-pgsql' '--enable-track-vars' 
'--with-apxs=/usr/local/apache/bin/apxs' '--enable-i18n' '--enable-mbregex'
         php3.ini file path is set to: /usr/local/lib
となってます。
またPHP3およびMySQLはソースからインストールしました。
