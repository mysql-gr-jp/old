Takahiro Iwatani <iwatani@xxxxxxxxxx> wrote:

> 　私はPostgreSQLを使用していたので、各DBの所有者を設定するのではないかと思っているのですが、
> MySQLでそのような部分を見つけることができませんでした。
> 　MySQLでDBの所有者の設定はどのように行えば宜しいでしょうか？

MySQL では、次のようにします。

1. MySQL にユーザを登録 (OSのユーザとは別)
2. そのユーザがどのデータベースに対してどのような権限を持つかを登録

1 は mysql データベースの user テーブルに対して行ない、2 は mysql
データベースの db テーブルに対して行ないます。

個々のテーブルを操作する代わりに grant 命令を使えば、この２つを一度に
登録できます。
