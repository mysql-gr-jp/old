福嶋です。

私の環境は、Win98＋アパッチ＋ＰＨＰ＋ＭｙＳＱＬ　です。
Win98のせいか、ＤＢはユーザーをあまり管理していないようです。
例えば、ＤＢを作ると「grant何とか」としないと利用できない
はずですが、「grant何とか」しなくても利用できたりします。
参考までに、私のコマンドの例を示します。（ヒントになるのでは？）

立ち上げるとき
mysqladmin  リターン
mysql -u root -p  リターン
パスワード入力になるので、パスワードを入力
ＤＯＳプロンプトがmysqlになる。
立ち上げ完了

show databases;  これでＤＢが表示される。
use XXXXX;       テーブルを作るＤＢをuseする。
あとは、
create table YYYYYY
(
------------------
);

確認するため、
show tables;     テーブルが表示される。

この後、普通なら「grant何とか」とするのですが、
私の場合、しなくても利用できています。
