'休み'を 3.5 として ORDER BY してはいかがでしょうか？

Records: 7  Duplicates: 0  Warnings: 0

mysql> SELECT
    ->   day
    -> FROM
    ->   test

    ->   day
    -> ;

    ->   day
    -> FROM
    ->   test

    ->   CASE day WHEN '休み' THEN 3.5 ELSE day END
    -> ;
