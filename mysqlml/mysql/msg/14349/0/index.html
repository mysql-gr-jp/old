平成15年頃に購入した書籍に添付されていた MySQL 3.23.49 というバージョンの MySQL を利用しています。
 
サーバーOS     Windows2000 Server
MySQL ver     3.23.49
インストール先   C:\mysql
起動方法        サービスに登録して、システム起動時に自動的に起動するように設定
my.ini             C:\Windows に記述 ( 内容は下記に記載しました )
 
クライアントOS     Windows Xp Professonal
ODBC ver          3.51.02
その他               myodbc3.dll を更新して部分的に日本語化してます ( 設定は下記に記載しました )
 
------------ my.ini の内容  --------------
[mysqld]
default-charactor-set=sjis
basedir=C:\mysql
--------------------------------------
 
---- MyODBC クライアントのオプション設定 -----
Return Matching Row                   True
EUC変換する                                True
Change BIGINT Columns to INT     True
--------------------------------------
