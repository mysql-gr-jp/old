泉田隼人> 接続先はlocalhostなのでpingなどは問題なく動作します。
MySQLではホストごとにアクセス権を設定できるので
mysqlコマンドでもJavaプログラムで指定しているのと同じホスト名を指定して
確認してみる必要があると思います

localhost=hoge.comとして
mysql -h localhost -uユーザー名 -pパスワード
と
mysql -h hoge.com -uユーザー名 -pパスワード
で異なるアクセス権が設定されているかもしれません
