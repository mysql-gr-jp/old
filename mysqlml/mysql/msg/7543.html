標題の通り，DB間を越えてテーブルを結合することは可能なのでしょうか．
例としては，DB1とDB2というふたつのデータベースを用意していて，以下のようにテーブルを組みます．

DB1----- t_name (氏名テーブル）

         ID    NAME
        ------------
          1    野茂
          2    イチロー
          3    松井
          4    石井
          5    長谷川

DB2----- t_team （チームテーブル）
    |
    |    ID    teamname
    |  -----------------
    |     1    まりなーず
    |     2    ヤンキー酢
    |
    |--- t_member （チームメンバー）

         NID    team   member
        ---------------------
          1       1       2
          2       1       5
          3       2       3

という状態で， t_member と t_team と t_name を結合させて次のようなテーブルを作りたいのですが，SQLは
どのように発行したらいいのでしょうか？ 
t_name テーブルだけ別のDBに入っているのでややこしくなっているのです．

         NID    teamname    NAME
       ---------------------------
          1     まりなーず  イチロー
          2     まりなーず　長谷川
          3     ヤンキー酢  松井
