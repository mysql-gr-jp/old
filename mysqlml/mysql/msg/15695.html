OTN MySQL User Forum Tokyo

〜MySQLの最新技術動向から技術概要、トラブルシューティングまで一気にお聞きいただける絶好の機会〜

来る4月3日、MySQL User Forum Tokyoを開催いたします。

米国本社より、開発部門責任者バイスプレジデント Tomas Ulin(トマス・ウリ
ン)によるキーノート及びセールス・コンサルティングディレクター Philip

ション、開発エンジニアによるInnoDBのパフォーマンス・チューニング、最新
製品概要、そしてトラブルシューティングなど、MySQLの世界を一同にご紹介し
ます。

MySQLエキスパートが一堂に会しご説明する絶好の機会です。開発者、DBAの方
はもとより、ITインフラでのMySQL活用をお考えの方、是非ご参加ください。

製品概要レベルから構築・運用・開発、そして現場で役立つ技まで。これらの
スキルアップにお役立てください。

  * MySQLによるコスト効率の高いデータベース・アプリケーションを構築
  * MySQLデータベースのパフォーマンスを向上
  * MySQLトラブルシューティング
