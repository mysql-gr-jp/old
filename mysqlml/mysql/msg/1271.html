# 前の質問の「col3 のデフォルトを col1 にする」の続きの話だと思ってまし
# た… (__;

> 
> Table list
>   NAME	 	AGE	 F/M	 type
>   shigeki	25	 M	  1 ←つまりAってこと
>   yamanaka	30	 F	  2         Bってこと
>   				 ~~~
>   				 
> Table type
> 
>  1 	2	 3	 4 
>  A      B 	 O	 AB 

これは select 時のテーブルの結合(join)でできます。テーブル type の項目名
がテーブル list と同じく type だとすると、

	SELECT 〜 FROM list LEFT JOIN type USING (type);

とかでできると思います。
