rie wrote:
> 以前、temporary table を使って、グルーピングしてから
> MAX値をとりだす方法がMLに流れていましたが、
> PHPではどのように、MySQLの temporary table を作成するのでしょうか？

普通にmysql_queryを発行するだけです。
$result = mysql_query("CREATE TEMPORARY TABLE hogetable select * from a");
$result = mysql_query("select * from hogetable");
