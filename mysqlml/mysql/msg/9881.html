以下は、全てWindows配下で行います。

◎テキストファイル"SCR.sql"をつくり、mysql.exeを実行する。

 SCR.sqlファイルの中身は以下の通り。
 -------------------------------
 select * from name;
 --------------------------------

 コマンドを打つ。（batファイルで可）
 mysql -u userID -pPassword DBname < SCR.sql > \\Winxp\temp_mysql\test1.csv 


◎また、SCR.sqlを作らずとも、以下のようにしてもよい。
 mysql -u userID -pPassword DBname -e "select * from name;" >
  \\Winxp\temp_mysql\test1.csv 

但し、出力されるファイルはtab区切りのCSVファイルとなります。
どうしてもカンマ区切るのCSVを得たい場合は、ローカルサーバに落してから
ファイル転送することになります。

mysqldumpでも可能とマニュアルにはありますが、試していません。
