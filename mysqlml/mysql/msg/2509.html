若林です。

  At Wed, 27 Sep 2000 20:01:41 +0900,
  Hiroyuki Hasegawa <admin@xxxxxxxxxx> wrote:

> こんばんわ。はせがわです。
> 
> MySQL-3.23.22から、MySQL-3.23.24にバージョンアップしようとして
> 
> rpm --rebuild MySQL-3.23.24-1.src.rpm
> rpm -Uvh /usr/src/redhat/RPMS/i386/MySQL-3.23.24-1.i386.rpm

私の場合。

rpm -i MySQL-3.23.24-1.src.rpm
rpm -ba /usr/src/redhat/SPEC/MySQL.spec
rpm -i /usr/src/redhat/RPMS/i386/MySQL-3.23.24-1.i386.rpm
	
> を行ったら、リビルドはできてますが、-Uvhを行ったとき
> file /usr/bin/mysql_config from install of MySQL-3.23.24-1 conflicts with file from package MySQL-devel-3.23.22-1
> と出て成功しません。

MySQLとMySQL-develが依存関係でconflictを起こしてるということならば、

rpm -e MySQL-devel-3.23.22-1 ; rpm -i MySQL-3.23.24-1.i386.rpm

です。
