> PostgreSQLを検討したのですが、外部結合をサポートしていない
> ため断念しました。（Unionで実現可能ですが...）
> 
> MySQLは外部結合をサポートしていますか？

こんなことができればいいんでしょうか？

	mysql> select * from A;
	+------+------+
	| id   | a    |
	+------+------+
	|    0 |    0 |
	|    1 |    1 |
	|    2 |    2 |
	+------+------+
	3 rows in set (0.00 sec)
	
	mysql> select * from B;
	+------+------+
	| id   | b    |
	+------+------+
	|    1 |    1 |
	|    3 |    3 |
	+------+------+
	2 rows in set (0.00 sec)
	
	mysql> select * from A left join B using (id) ;
	+------+------+------+------+
	| id   | a    | id   | b    |
	+------+------+------+------+
	|    0 |    0 | NULL | NULL |
	|    1 |    1 |    1 |    1 |
	|    2 |    2 | NULL | NULL |
	+------+------+------+------+
	3 rows in set (0.00 sec)
	

> その他、注意点はありますか？

MySQL は結合関係は弱いです。おそらく PostgreSQL よりも…。
