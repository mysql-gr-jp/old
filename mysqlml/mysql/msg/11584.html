休憩がてら flashMyAdmin をインストールしました。
ムダにアニメーションしまくってるのがいいですね～(^.^)

「Server Name」で [サーバIP]:[ポート番号] を指定すると
任意の MySQL Server へ接続できます。

日本語は残念ながら文字化けします。
  MySQL4.0 (ujis)
  MySQL5.0 (utf8)
どなたか興味のある方は日本語化にチャレンジしてみてください。
