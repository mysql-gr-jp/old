PHPの公式のビルド済みWindowsバイナリで使用しているmysqlクライアント
ライブラリは、phpinfoの出力でmysqlの項を確認すれば判りますが、
PHP4.3.9の場合「Client API version 	3.23.49」と、古いスタイルの
パスワードハッシュを使用するバージョンのものです。また、このmysql
クライアントライブラリは、PHPに静的に組み込まれてビルドされています。

先に示したマニュアルの記述では
「サーバを 4.1 にアップグレードしたときに 4.1 より前のクライアントと
互換性を保つ方法」という解説に該当します。
