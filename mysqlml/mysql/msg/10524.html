MaxDB インストレーション　マネージャーのリリース

新しいMaxDB インストレーション　マネージャーがリリースされました。
インストレーション　マネージャーを使うとMaxDBのインストレーション
が容易に行えます。ほんの２−３のクリックで。

インストレーション　マネージャーは直観的なで簡単なグラフィカラルな
インターフェースを提供してインストレーションのプロセスを
容易にします。デスクトップ・データベースをセットするか更には
カスタマイズされたエンタープライズ用のシステムにも使用できます。

マネージャー使用に関しては３ページのドキュメントが以下に
あります。
http://dev.mysql.com/doc/maxdb/instupgrade.html

インストレーション　マネージャーはMaxDB7.5版ようにプロトタイプ
としてリリースされています。最終版は7.6版と同時に出荷されます。

プロトタイプの特徴は:

MaxDB ソフトのインストレーション
MaxDB ソフトのアップグレード、インスタンスのアップグレードはまだ。
MaxDB のセーフ・プロダクション版をインストール
自動ディインストレーションはまだなし、手動で。
LinuxとWindows 32 ビット、LinuxのAMD/Opteron 64ビットをサポート

ダウンロードの他にMaxDB のインストレーションCDを無料で
以下のサップショップから注文できます。

www.sap.com/shop -> Knowledge Shop -> Documentation -> Others

製品番号は 50070581 (サーチの際お使いください)。
CD には新しいMaxDB PHPドライバーも含まれます。
