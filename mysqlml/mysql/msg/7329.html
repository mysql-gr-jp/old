aimitsu katoh <akimitsu@xxxxxxxxxx> さんは書きました:

ak> もうプログラムのコーディングも終わっているので出来ればMySQL側で DATETIME型のデータを【20030320105328】
ak> というフォーマットで扱いたいのですが、可能でしょうか？
ak> どなたか是非ご教授ください。

FROM_UNIXTIME(unix_timestamp) 
文脈(数値/文字列)に依存して、'YYYY-MM-DD HH:MM:SS' または YYYYMMDDHHMMSS 形式の timestamp 文字列を返します: 
mysql> SELECT FROM_UNIXTIME(875996580);
        -> '1997-10-04 22:23:00'
mysql> SELECT FROM_UNIXTIME(875996580) + 0;
        -> 19971004222300

これ、使えますか？
