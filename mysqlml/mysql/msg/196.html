>   その設定の方法がわからず困っています^^;;

あらら(^_^;

まず、hostテーブルで「安全なホスト」を登録します。うちでは
INSERT INTO host VALUES ('%','%','Y','Y','Y','Y','Y','Y');
としてたりしますが、閉じたネットワーク内なので。

次に、userテーブルで各ユーザーの権限を設定します。
INSERT INTO user VALUES ('%','kengo','password','Y','Y','Y','Y','Y','Y','Y','Y','Y','Y');
パスワード無しなら第３フィールドは''、パスワードを設定する時はpassword()
関数を使って設定します。
＃userテーブルは '%','%' を認めてくれないみたいです。

dbテーブルやfuncテーブルは初期値のまま何もしてませんが、これで
問題なく使えています。

で、設定したらmysqladminでreloadしてやります。
