> また、MySQL Reference Manual for version 4.0.12 には、
> LOAD DATA LOCAL INFILE....となっているのに、
> LOAD DATA INFILE...でないとできないのはなぜでしょうか。

LOAD DATA LOCAL INFILE を使用するには、サーバとクライアントの両方で 
--local-infile オプションが必要です。

> また、自分のホームディレクトリでつくったtextfileをデータとして入力するには、
> /usr/local/mysql/varに移してからでないとできないのでしょうか。

そんなことはありません。ファイルが置かれているディレクトリのアクセス権
の問題だと思います。
