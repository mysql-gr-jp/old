ちょうど　私も　Windows で
perl DBI が　動いて　これは　使えるな　とおもってい
たところで　(^^ゞ

多分　DBD-mysql がないせいだと思いますが


私の場合は　active perl 518(perl 5.005_003) で
ちゃんと　動作しています
UNIX のときのように　mysqlclient を入れなきゃならない
などのこともなく　すんなり動いてくれました

まず　perl をとってきます
perl (ActivePerl) 
http://www.activestate.com/ActivePerl/download.htm

このあと
インストールして
http://www.activestate.com/packages/zips/
から DBI.zip と DBD-mysql.zip をダウンロードしてきて
展開して　ppm install DBI.ppd などとするか

または　インターネット上なら
ppm install DBI
ppm install DBD-mysql
で　ダウンロード＆インストール　してくれます

あとは　DBI　スクリプトをきちんと記述すれば
うごいてくれます

perl-TK (TK.ZIP) などと組み合わせて使えば　windows上のGUIで
いろいろ遊べそうですね

なお　perl日本語化に付いては
http://www.shonan.ne.jp/~kipp/perl/jperl_win32.html
を参照なされば　詳しく書いてます
