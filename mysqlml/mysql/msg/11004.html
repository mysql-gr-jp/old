MySQLがエンタープライズ用の「MySQL Network」を発表
迅速なオープン・ソースのデータベースを展開

全て入った解で高性能のビジネス・クリティカルなMySQL
アプリケーションを高い費用効果で展開

LinuxWorld Conference & Expo、ボストン-- 2005年2月15日

世界で一番人気のあるオープンソースのデータベース　ベンダー
のMySQL ABは、今日エンタープライズがビジネス・クリティカル
なアプリケーションのためにMySQLを採用、展開、サポートする
新しい方法を発表しました。この新しい「MySQL Network」は
大規模なIT部門で、MySQLのコストと性能の恩恵を求めており、
同時にオープンソースの実装のためのビジネス
の要求をしっかりと持っているところを対象にしています。

MySQL Network は分かりやすく費用効果の高い解で以下を
提供します。

　　*エンタープライズ・レベルのプロダクション・サポートと知識ベース
　　*自動のアップデートやアラート
　　*柔軟なオープン・ソースのライセンス
　　*保障や免責
　　*認定されてプロダクションに適すソフトウエアー

MySQL ABのCEOのMarten Mickosは「MySQL Networkは企業の時間を
節約するだけでなく、MySQLを使用するアプリケーションを簡単にして
リスクも軽減します。広範なオープンソースのコミュニティのサポート
を受けて我々のデータベースはIT業界で一番速く信頼性の高い
 ものの一つとなりました。現在世界中で実際に使用されて
いる600万件のインストレーションがあります。我々の会社は
ここまで成長し、MySQL　Networkを通してエンタープライズのカスタマー
の高度な要求を満たすことができるようになりました。」と語った。

PriceGrabber.com, Blue Frog Mobile, Soft Express と
Silver State Schools Credit Unionを含む数社は既に
MySQL NetworkのサブスクライバーでMySQLの開発者から
タイムリーなサービスを受け、最良のソースから認定
ソフトを受け取っています。

PriceGrabber.comのファウンダーでchief product officer の
Tamim Mouradは「MySQL NetworkはMySQLのコストと性能の
恩恵をもたらすだけでなく、プロダクション・レベルの
サポート、ベンダーの責任の明確化と認定ソフトを提供
してくれます。PriceGrabber.comは1999年以来ビジネス・クリティカル
な分野にMySQLを使っており、MySQL Networkは開発者
の生産性を最大化すると確信しています。」

MySQL Network:

エンタープライズがプロダクション・レベルのMySQLアプリケーション
を成功裏に展開できるのに必要なもの全てを含む

MySQL Networkは4段階（基本、金、銀、プラチナム）の年間
サブスクリプションで 企業の大きさやニーズに従って
カスタマイズされます。一番高度な MySQL Network プラチナム
は次の要素を含みます。

*エンタープライズ・クラス　プロダクション・サポート
- オープンソースを使用する際のエンタプライズのIT部門の疑問や
心配を払拭するためMySQLはプロダクション・レベルの
サポートをMySQLの開発者から直接提供します。プラチナム
サブスクライバーは制限なしの 24x7 の電話サービス
と経験豊かなエンジニアリング・チームへのウエブによる
アクセスを提供します。これには30分以内の緊急反応を含みます。

*新しい認定ソフト - 新しいMySQL Pro 認定データベースのサーバーは
良くテストされ信頼性が高く、ビジネス・クリティカルな使用に
耐えると保障されます。

このソフトはMySQLのQAチームによってコマーシャル版と
オープンソース版のツールによってテストされているだけでなく
その他オープンソースの開発者コミュニティによってもリビューされて
います。その上 Coverity 社と Klocwork Softwareによって
解析されています。この２つはソースコードを解析に有名な
会社で、クラッシュを引き起こすバグや性能の問題やセキュリティの
問題などををテストします。

MySQL Network 認定ソフトはオープンソースのGPLを通じてライセンスされて
います。これでデータベースの開発と展開に柔軟
で幅の広いオプションを提供できます。

*保障と免責による新しい保護。MySQL ABは自分の
全ての技術の開発者でありオーナーでもあります。オープンソース
業界でユーニークで強固な法的な保護を提供できます。ある条件
を満たすサブスクライバーは包括的な知的財産問題への免責と
被害をカバーする項目を追加できます。これには特許、コピーライト、
トレードマーク、トレードシークレット侵害を含みます。

*新しいMySQL 知識ベース - MySQLの開発者とサポートが書いた
何百という記事の総集データベースです。検索可能な知識ベース
は大抵の問題への解答を提供します。その他、トラブルシューティング
性能向上へのヒントや「ベスト・プラクティス」への情報も
提供します。

*新しい MySQLアップデート・アドバイザー - 自動のメッセージ
システムでカスタマーの環境に適した新しいMySQL認定の
ソフトをアラートで知らせます。アップデート・アドバイザー
は開発者とDBAにインストレーションとアップグレードの
プロセスに関して情報を提供して手助けをします。アラート
は電子メール、ページャーまたはSMS メッセージで配布されます。
完全なアップデートやアラートのリストは普通のウエブ・ブラウザーで
閲覧できます。

*新しいテクニカル・アラート・アドバイザー - テクニカル・アラート
システムはカスタマーに特定のコンピューティング環境
に関する問題を、アプリケーションに影響
が出る前に通告します。また完全にカスタマイズできる他、
これらのアラートはＯＳのセキュリティの問題、新機能、
性能向上や重要な知識ベースの記事などを含みます。

価格とアベイラビリティ
MySQL Networkは現在11 の主なプラットフォームに適用されます。
Red Hat Enterprise Linux, SUSE Linux Enterprise
Server, Microsoft Windows Server 2003, Sun Solarisと
HP-UXを含みます。年間のサブスクリプション・モデルで
提供され、データベース・サーバーあたり、 US $595から
$4,995 （EUR だと495 から3,995)。詳細は http://www.mysql.com/network.
