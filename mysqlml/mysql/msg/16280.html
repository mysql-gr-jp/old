MySQL 5.7のrpmではデフォルトでパスワードが生成される＆初期状態でEXPIREされている関係で、
↑のURLの通りにやればmysql57-community-mroongaがインストールできますが、
mysql-community-serverがインストールされていない状態からmysql57-community-mroongaで
依存関係の解決に任せてインストールしようとするとMroongaのインストールスクリプトが転けます。
（俺がいつもそうしていました。。）

修正リクエストは投げてあるので、次にrpmファイルがビルドされた時にはこの問題はFIXされるはずです。
（真っ新な状態からyum install mysql57-community-mroongaだけでINSTALL PLUGINまで全て終わる）
https://github.com/mroonga/mroonga/pull/78
