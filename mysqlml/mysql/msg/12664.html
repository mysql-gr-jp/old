ところで自分が１つボケかましてたのを見つけたので訂正しておきます。

> ここでMySQL4.1以降だったら SELECT (上記のSELECT文) AS t WHERE `SUM` <> 0 で

「ここでMySQL4.1以降だったら SELECT * FROM (上記のSELECT文) AS t WHERE `SUM` <> 0 で」
の書き間違えでした。
ついでに書き添えると、MySQL4.0でも計算結果を一旦テンポラリテーブルに
INSERT～SELECTしてそこからSELECTで`SUM`<>0の行だけ抽出するようにすれば
DBサーバ内で抽出が行えてDBサーバ～Webサーバ間の通信量を削減できます。
