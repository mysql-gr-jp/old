製品

MySQL (5.0.4 Beta)リリース
これは5.0版の２度目のベータ版です。このリリースには
Stored Procedure、 View、 Trigger、Information Schema (Data Dictionary),
その他を含みます。その他の変更の一部は

* -show-table-type オプションをmysqlshowに追加して,SHOW FULL TABLES
用にテーブルのタイプを示す列を表示するように。
* コンフィギュレーションのために-with-big-tablesコンパイルの
オプションを追加
* 新しいコンフィグレーションのための!include and !includedir
が実装されオプション・ファイルをを含有したりオプション・ファイル
の検索を可能に。

現在全てはバグ修正と安定度を上げることに集中されています。
今から慣れるためにベータ版を使ってみましょう。

変更の詳細は
http://lists.mysql.com/announce/277
ダウンロードは
http://dev.mysql.com/downloads/mysql/5.0.html

MySQL Query Browser (1.1.7) のリリース
MySQL Query BrowserはGUIクエリのシェルでウエブ・ブラウザーの
様な 使い勝手の良いユーザインタフェースからSQLのクエリを
実行します。

この版での変更は

* MySQL 5.0.3とそれ以降の版に対するViewのサポート
* MySQLのクエリー警告とノートの表示
* 安定度の改善

変更の詳細は
http://lists.mysql.com/announce/277
ダウンロードは
http://dev.mysql.com/downloads/mysql/5.0.html

MySQL コミュニティ版(4.1.12)のリリース

4.1版の保守リリースです。最近発見されたバグの修正
が主です。このリリースには

* libeditの版を2.9にアップデート
* InnoDB: SHOW INNODB STATUSのアウトプットの長さの限界
を超えた場合、最後を削るのではなく最初を削る。
* InnoDB: FOREIGN_KEY_CHECKS=0の時 ALTER TABLE と RENAME
TABLE は参照する列と参照される列がタイプ・コンパチでないと
無視。 foreign keyにある列の文字列を変換可能。
データを変更するまえに全てのテーブルを変換すること。

変更の詳細は
http://lists.mysql.com/announce/282
ダウンロードは
http://dev.mysql.com/downloads/mysql/4.1.html

MySQL Migration Toolkit (1.0.6)のリリース
MySQL Migration Toolkitはプロプリエタリなデータベースを
MySQLにマイグレートするのに使う有益なフレームワークです。JDBCを介して
OracelやAccess やその他のデータベースをマイグレートする
機能が備わっています。MySQL Migration Toolkitは
MySQL 4.x と MySQL 5.0にマイグレートできます。この版は
主にバグ修正で、以下の変更を含みます。

* MS Access DOUBLE と BYTE のデータタイプを追加
* MS Access COUNTER (AutoNum)のセーフな扱いとMySQLの
auto increment flagへの変換。

変更の詳細は
http://forums.mysql.com/read.php?104,26459
ダウンロードは
http://dev.mysql.com/downloads/migration-toolkit/1.0.html

Point-in-Time データ・リカバリー
mysqldumpの様なツールはMySQLのデータやテーブル
のスキームをバックアップするには非常に有益です。しかし、
バックアップは一日一度だけ、または決まった頻度
でしか行われません。最後のバックアップの後何時間
もたってからデータがなくなったら、リカバーできません。
続けてデータを守ることができる方法があります。その一部
をこの記事で取り上げます。

記事を読む
http://dev.mysql.com/tech-resources/articles/point_in_time_recovery.html

パートナー・コナー
MyDB スタジオ

MySQLのMyDB スタジオ はグラフィカルなGUIで開発者や
DBAにMySQLのサーバー管理を簡素化する環境を提供
します。これにはバックアアップ、チューニング、
やSQLクエリの生成や実行を含みます。 MySQLのMyDB スタジオ
はPHP の開発者にPHP用のSQLクエリジェネレーター
とPHPコードジェネレーター(native, PEAR, ADOBb)を提供します。
自前のクエリのテンプレートをEMICで生成できます。
MySQLのMyDB スタジオは英語とフランス語で提供されて
おり、MySQLドキュメンテーションにリンクして
ドキュメンテーションが存在します。

詳細は
http://www.mydb-studio.com/
