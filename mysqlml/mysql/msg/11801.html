とみたです。

MySQL/Ruby 2.7-beta を公開しました。
http://tmtm.org/ja/mysql/ruby/README_ja-2.7-beta.html

プリペアドステートメントに対応しました。こんな感じで使えます。

my = Mysql.new(hostname, username, password, databasename)
st = my.stmt_init
st.prepare("insert into tblname (col1,col2,col3) values (?,?,?)")
st.execute("abc",123,Time.now)
st.prepare("select col1,col2,col3 from tblname")
st.execute
st.fetch  # => ["abc", 123, <Mysql::Time:2005-07-24 23:52:55>]
st.close

・従来はクエリの結果はすべて文字列で返されましたが、MySQL の型に対応し
  た Ruby のオブジェクトで返されるようになります。

・execute() で指定するパラメータはクォートする必要がありません。

・また、クエリの速度が若干速くなるようです。

よろしければお試しください。
