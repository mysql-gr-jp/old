マニュアル 3.4.3 ISQP MySQL services の第2段落

    If you are an ISP that allows customers to install and administer
    MySQL for themselves on your machine with no assistance from you,

    する事を 許可しているのであれば(あなたの手助けは無しで)、 あなたかあるい
    は顧客のどちらかに MySQL ライセンスが必要です。

と訳されてますが、「neither 〜 nor 〜」なんで「どちらにも MySQL ライセン
スは必要ではありません」じゃないかと…。
