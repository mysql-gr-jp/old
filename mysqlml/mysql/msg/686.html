1.solaris x86 パッケージ

> マシンが reboot してしまう現象に当たっています。

いろいろやっているうちに解決しました。
ujis,sjis のパッケージを up しておきました。

2. FreeBSD パッケージ
ujis, sjis の FreeBSD-3 用のバイナリをパッケージ形式にしたものと、
ports ファイルを up しました。

FreeBSD-2.x の方は、ports から作成できると思います。

ports を展開すると Makefile がありますが、
このなかの FreeBSD-3 独自のところをコメントアウトして、
make CHARSET=ujis build
make CHARSET=ujis package
とすればよいはずです、、、(README.jaみてください)

# こちらには FreeBSD-2.x の環境がないので、
# FreeBSD-2.x ご使用の方、試していただくとうれしいです

/var/db/mysql がすでに存在するとpackage install中に止まるので
/var/db/mysql を rename して make package してください。
