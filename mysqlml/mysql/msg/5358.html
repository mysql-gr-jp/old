ADODB.Commandと、そのプロパティのParameters、
メソッドのCreateParameterあたりを見てみて下さい。

Recordsetの取得は、（とりあえずは）以下のようにするとできます。

Set cmd = New ADODB.Command

（cmdへの、Connection,SQL文,パラメータの設定処理）

Set rs = New ADODB.Recordset
rs.Open cmd, , adOpenForwardOnly, adLockReadOnly
