次のようなテーブルを作成して、
create table t1(ts datetime);
このテーブルに、300件/秒でデータをインサートして
取り出す時にインサート順に取り出すにはどうすれば良いのでしょうか。
ソートオプションは、何かあるのでしょうか。

次のＳＱＬでやってみたところ、インサート順になっているような気はするのですが。
select * from t1 where date_format(ts, '%H:%i:%s')='12:12:12' ;
