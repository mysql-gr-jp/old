なんせ初心者なんで、どう説明したらいいか難しいんですが…
SELECT文で例えば

select * from database where name like '%my%';

としたとき”my”を含むものにヒットしますが、この文の
%my%の部分って他にどんなものがありますか？

”my”ってのは今、ブラウザからの入力されたものを”my”ってとこに埋
め込んで検索をしているのですが、たまに”%”とか入力してくる人がいる
んです。けっきょくこの場合って

select * from table2 where name like '%%%';

なりWHEREで絞り込んだ意味がなくなってしまいます。この場合って
”%”って文字を検索する場合って

select * from table2 where name like '%"%"%';

でいいんですよね？
