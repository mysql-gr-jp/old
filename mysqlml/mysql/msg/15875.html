後藤と申します。

InnoDB なら INSERT INTO の側でトランザクション隔離レベルを
READ COMMITTED にすれば競合しないと思われます。

    SET SESSION TRANSACTION ISOLATION LEVEL READ COMMITTED;
    INSERT INTO WORK( コード、名 )SELECT コード、名 FROM 得意先マスタ WHERE コード = 1;

以上です。
