はじめまして、一連のスレをみていて思ったのですが、そもそもの話以前のサー
バと現在のサーバとで MySQL の文字コード設定は実際はどうなっていたのか？と
いうことです。

両方のサーバで MySQL にログインして、以下のコマンド実行して比較してみてく
ださい。まずは、MySQL の設定内容に差異がないのかどうかから確認していった
方がいいと思いますよ。

SHOW VARIABLES LIKE 'char%';

また、両方のサーバでの /etc/my.cnf を比較してみることをお勧めします。

私は、PHP、MySQL ともに UTF-8 でしか使ったことがないので、あまり参考にな
らないかもですが、MySQL の文字コード設定は以下のように設定してました。

[client]
default-character-set = utf8

[mysqld]
default-character-set = utf8
skip-character-set-client-handshake
character-set-server=utf8
collation-server=utf8_unicode_ci
init-connect=SET NAMES utf8

もし、MySQL の設定に差異がないようであれば、PHP の設定ファイルの方を確認
すればよろしいかと思われます。
