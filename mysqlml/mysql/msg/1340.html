やっと、動きました。
mysql-ruby が windows 上で。(^^)/

ruby-1.4.2 を VC で自分でコンパイルして、
あと mysql.c の少しの変更と、
mysql-ruby の Makefile の変更で。

#===
mysql.c の windows 用の変更

uint -> unsigned int へ。
Data_Wrap_Struct(cMysqlRes, 0, (void)(*mysql_free_result), res) へ。

#===
ruby extconf.rb 後に出来た Makefile の修正
(VC6.0, mywinclients-3.22.28 つかってます)

LOCAL_LIBS =  rubymw.lib -link /LIBPATH:$(topdir) /EXPORT:Init_$(TARGET) /nologo  /NODEFAULTLIB:libc.lib
LIBS = lib\mysqlclient.lib lib\strings.lib lib\zlib.lib wsock32.lib kernel32.lib  user32.lib gdi32.lib winspool.lib comdlg32.lib advapi32.lib shell32.lib ole32.lib oleaut32.lib uuid.lib odbc32.lib odbccp32.lib
