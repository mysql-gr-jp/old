マニュアルを読むと、

INDEX のうち、値を一意にするという性格を持つものが UNIQUE INDEX で、
UNIQUE INDEX のうち、NOT NULL 属性である最初のものが、PRIMARY KEY である。

ということだと思います。

ちなみに、以下の３つは同じことです。

	CREATE TABLE hoge (id INT PRIMARY KEY);
	CREATE TABLE hoge (id INT NOT NULL UNIQUE);
	CREATE TABLE hoge (id INT NOT NULL, UNIQUE INDEX (id));
