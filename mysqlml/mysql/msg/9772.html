上田さんが「> mysqlshow」で作成したデータベースが表示
されないという現象は、「> mysqlshow」とだけ入力した
からではないでしょうか？
私の場合、
    「> mysqladmin -uroot -p create bankindb」
として、データベースを作成したところ、bankindbは、
「> mysqlshow -uroot」と入力しないと、表示されま
せんでした。
    ----------------------------------
    C:\mysql\bin>mysqlshow
    +-----------+
    | Databases |
    +-----------+
    | test      |
    +-----------+

    C:\mysql\bin>mysqlshow -uroot
    +-----------+
    | Databases |
    +-----------+
    | bankindb  |
    | mysql     |
    | test      |
    +-----------+
    ----------------------------------

同様に、、、
「mysql> show databases」コマンドでの表示についても、
mysqlの接続で、「> mysql」として起動するときと、
「> mysql -u root」として起動する時とでは、結果が異
なります。
(大文字・小文字は関係ありません。。。というか、それ
 はすぐ確認できたと思いますが。。。)
    ----------------------------------
    C:\mysql\bin>mysql
    mysql> show databases;
    +----------+
    | Database |
    +----------+
    | test     |
    +----------+
    1 row in set (0.00 sec)

    mysql> quit
    Bye


    C:\mysql\bin>mysql -u root
    mysql> show databases;
    +-----------+
    | Database  |
    +-----------+
    | bankindb  |
    | mysql     |
    | test      |
    +-----------+
    3 rows in set (0.00 sec)

    mysql> quit
    Bye
    ----------------------------------


いかがでしょうか？
結果を教えて下さい。
