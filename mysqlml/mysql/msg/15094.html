MySQLでは管理やクエリの発行に以下のツールが用意されています。無償です。

MySQL GUI Tools
    * MySQL Administrator 1.2
    * MySQL Query Browser 1.2

http://dev.mysql.com/downloads/gui-tools/5.0.html

ER図をあつかうモデリングツールとしてはMySQL Workbenchがあります。 こちらは無償版と、有償版があります
。

MySQL Workbench 5.1.18a
http://dev.mysql.com/downloads/select.php?id=8

GUI Toolsは現在開発が止まっており、今後はAdministrator/Query Browserの機能は
MySQL Workbenchに統合されます。そのファーストバージョンがMySQL Workbench 5.2として上記URLの
Upcoming Releasesからダウンロードできます。

MySQL Workbench 5.2.10 beta
