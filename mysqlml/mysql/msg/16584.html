こんにちは。Mroongaプロジェクトの堀本です。

Mroonga 10.02をリリースしました！

リリースアナウンス:
   http://mroonga.org/ja/blog/2020/04/29/mroonga-10.02.html

MroongaはMySQLで日本語全文検索を実現するストレージエンジンです。高速で
あることや位置情報検索をサポートしていることなどが特徴です。詳細につい
はドキュメントをご覧ください。

  * Mroongaの特徴 ― Mroonga v10.02 documentation
    http://mroonga.org/ja/docs/characteristic.html#what-is-mroonga

変更点:
   http://mroonga.org/ja/docs/news.html#release-10.02

さて、今回の主な変更点は次のとおりです。

  * WITH_WEIGHT フラグをサポートしました。

    * このフラグを使うには10.0.2以降のGroongaが必要です。
    * このフラグを使用して以下のように重み付きベクターを挿入できます。

      CREATE TABLE tags (
        name VARCHAR(64) PRIMARY KEY
      ) DEFAULT CHARSET=utf8mb4;

      CREATE TABLE bugs (
        tags TEXT COMMENT 'flags "COLUMN_VECTOR|WITH_WEIGHT", type "tags"',
        FULLTEXT INDEX bugs_tags (tags) COMMENT 'table "tags", flags "WITH_WEIGHT"'
      ) DEFAULT CHARSET=utf8mb4;

      INSERT INTO bugs VALUES ('{"package": 100, "priority": 5}');

  * CentOS7のMariaDB 5.5のサポートをやめました。

  * CentOS MySQL 5.6.48, 5.7.30, 8.0.20をサポートしました。

それでは、Mroongaでガンガン検索してください！
