Windows9x 系は クライアント OS ですのでサーバ用途としての

XP Pro は10台まで OK です。根拠はここ↓

http://www.apache.jp/misc/windows.html

これは Apache が MS とミーティングした時の概要ですが、ネット
ワーク同時接続ということからいえば mysql も絡んでくる事と思い
ます。

但し「ミドルウェアや業務アプリケーション開発の試験用」であれ
ば問題有りません。これは、MSDN の一定レベル以上では SDK/DDK
以外に OS も配布されますが「評価の目的のみに使用を許可する」
事項が記載されているからです。製品版について MS に確かめた
訳ではありませんが、多分同様に扱われるのでは、と思います。
勿論実運用では取得クライアント数以上で接続するとライセンス
違反です。それでも動いてしまうので、知らず知らずのうちに
ライセンス違反している所が沢山有るかもしれませんね。

尚、2003 Server からは取得ライセンス以上の接続はガードロック
がかかっているようだ、と聴いております。
