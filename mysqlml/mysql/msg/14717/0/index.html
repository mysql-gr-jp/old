nullなのに、0を返していたというのが事実なら、MySQL 3.23.55に問題が
あったのではないでしょうか？
 
nullの場合に、0としたいなら、coalesce関数を使うのが簡単だと思います。
 
select sum(coalesce(id,0)) from tbl1;
 
 
なお、coalesce関数は、主要なRDBMSで、同じ機能で実装されています。
