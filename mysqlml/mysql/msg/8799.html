MySQL ヒント
・ MySQLの性能を計測する
MySQL Administratorを使用できる前にあなたのMySQLの性能評価をする方法
を述べます。モニタリングは常に繰り返し行われ持続するものです。データベース
の実行パターンでどうのようなパターンがOKで、もしくは小さい問題から大きな問題
に関してどのような兆候があるか知る必要があります。

以下にシステムをモニターするのに使用できる項目をあげます。

- mysqladmin extended (絶対値)
- mysqladmin extended -i10 -r
- mysqladmin processlist
- mysql -e "show innodb status"
- OS data. vmstat/iostat
- MySQL エラーログ
- Innodb tablespace 情報.

・ MySQLのStored Procedures

Functions」
