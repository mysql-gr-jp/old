藤原です。

  CREATE TABLE Syntax について、ちょっと重箱の隅をつついたような内容で
  す。


CREATE TABLE Syntax
http://dev.mysql.com/doc/mysql/en/CREATE_TABLE.html
http://dev.mysql.com/doc/mysql/ja/CREATE_TABLE.html

を見ていたら、columnに対する type: に、

  | CHAR(length) [BINARY | ASCII | UNICODE]
  | VARCHAR(length) [BINARY]

はあるのですが、

  CHARACTER SET charset_name

を付加するのが見当たりません。
これがないと、細かく指定ができません。
でも、後の文章中にはちゃんと出てきます。


また、table_option: の中に、

  | [DEFAULT] CHARACTER SET charset_name [COLLATE collation_name]

はあるのですが、

  [DEFAULT] CHARASET=charset_name

という書き方は出てきません。COLLATEのところは、間に = があってもなくて
もOKなようで、

  [COLLATE [=] collation_name]

とすべきでしょうか。

= は、だいたいあっても無くても良いことが多いようだ。


何も困ることはないのだが....
