ターミナル上の日本語入力は問題なしです。vimなどの入力も問題なしです（自
分でコンパイルしたものです）。
jtcshを使ったのは、とりあえず既にHDD上にあったからで、jbashはHDD上にはな
かったという理由からです。
.inputrcも作っています。
set convert-meta off
set output-meta on
set input-meta on
set show-all-if-ambiguous on

もちろんtcsh用に　.tcshrcもあります。
unset nokanji
set dspmbyte=utf8

別RESに書いたように4.1.5gammaでは問題なかったので、どうも4.1.3betaに問題
ありだったようです。
