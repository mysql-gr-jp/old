On Wed, 23 Jan 2008 18:28:22 +0900
福田　晃章 <t.fukuta@xxxxxxxxxx> wrote:

> バックアップの方法ですが、mtsqldump で実行してテキストファイルに保存しておけば良いのでしょうか。
> mysqldump -u root -p [ファイル名] [データベース名]

出力先ファイル名を指定するのは -r です。

ファイルの中身が文字化け等していないかちゃんと確認しておきましょう。

> リストアは、下記の方法でやろうと思っています。
> mysql -u root -p [データベース名]  < [ファイル名]
> 
> また、現在利用している文字コードは、sjis ですので、5.0のバージョンでも sjis 
> ですか。
> それとも 5.0のバージョンでは cp932 という文字コードを使っても良いのでしょうか。 

Windows なら cp932 の方が無難です。

mysql に --default-character-set=cp932 を指定し、さらにデータベースを作
成する際に、charset=cp932 をつけておけば、文字化けはしないと思います。
