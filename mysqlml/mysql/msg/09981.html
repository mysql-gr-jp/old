まず、インストールはフォルダを丸ごとコピーで十分です。

>i> 試してませんが、サービスのインストール、アンインストールは
>i> mysqld-nt.exe --install
>i> mysqld-nt.exe --remove
>i> みたいですよ。

windows 2000,XPでのサービスの inst uninst は、これでいいですね。
さらに、inst 時にサービスをスタートさせる場合は、
>net start mysql
uninst 時に停止させる場合は、
>net stop mysql
です。
ちなみに、サービスは自動起動になります。

windows9X系では、inst したmysql等を直にたたきます。
スタート時 (スタートアップで十分です)
>c:\mysql\bin\mysqld.exe
シャットダウン時
>c:\mysql\bin\mysqladmin shutdown

# ライセンスを取得しているのなら、
# インストールはどんな方法でも問題ないと思うのですが。
