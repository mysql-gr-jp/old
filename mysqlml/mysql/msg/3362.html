> ユーザーには生年月日を入力させシステムにて現在年齢を算出させる様
> 設計しようと考えております。

色々やり方はあると思いますが、

| mysql> select DATE_SUB(CURDATE(),INTERVAL TO_DAYS('1967-1-1') DAY);
| +------------------------------------------------------+
| | DATE_SUB(CURDATE(),INTERVAL TO_DAYS('1967-1-1') DAY) |
| +------------------------------------------------------+
| | 0034-03-13                                           |
| +------------------------------------------------------+
| 1 row in set (0.01 sec)

で、とりあえずできるようです。
＃ 厳密には、-1 日か。
