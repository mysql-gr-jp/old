Ishiiさんこんにちは。池田と申します。

SELECT * FROM TEST_TBL WHERE 1;

このクエリはオプティマイザにより、

SELECT * FROM TEST_TBL;

としてWHERE句は切り捨てられて処理されると思います。

一般的に各SELECT文が内部でどのように処理されるかを知るには、
EXPLAINを使用するのが便利です。

例：EXPLAIN SELECT * FROM TEST_TBL;
