MySQLへのマイグレーション
http://www.mysql.com/news-and-events/press-release/release_2005_12.html

MySQL AB は'MIG 4'技術にもとづく新しいデータベース
マイグレーション・ツールとコンサルティングの
パッケージを提供します。ユーザはビジュアルな
オープンソースのツールをダウンロードしてMySQLの
開発者から直接のカスタムのコンサルティングを受ける
ことが出来ます。

カリフォルニア州サンタクララ市2005年4月19日発

オープンソースのデータベースの開発社のMySQL AB
は今日新しいビジュアルツールとマイグレーション
のコンサルティングのパッケージを発表しました。
これはMySQLの４フェーズの"MIG 4"を使用して、
既存のデータと技術とアプリを最大限に活用して
お手ごろな値段でデータベースの解を提供するようにデザイン
されています。

ベータ版の新しいビジュアルなGUIベースの MySQL マイグレーション
スイートがMySQLのウエブサイトからGPLライセンスで無料
で入手できます。現在Oracleと Microsoft Accessをサポート
します。近日Microsoft SQL Server, Sybase, DB2と Informix
をサポートします。詳細はhttp://www.mysql.com/products/migration-suite/.

MySQLのサービス担当のUlf Sandbergは「高度な機能を持つMySQL 5.0が出荷
に近づき、ユーザにデータベースのマイグレーション、性能チューニング
とスケールアップを助けるツールとサービスを提供します。
マイグレーション・スイートとジャンプスタートの
コンサルティング・パッケージその線に沿ったものです。
包括的な'MIG 4’方式で長年の経験に基づき実世界の
アプリをMySQLにマイグレートします。」と語りました。

MySQL マイグレーション・ジャンプスタート- 5日間 US $8,500 (通常 $10,000)
6月30日までに申し込むと$1,500節約できます。MySQLは
旅費及び他の経費をチャージしません。

5日間でMySQLのコンサルタントはプランを立てて
Microsoft SQL Server, Oracle, Sybase, DB2, Informixや
その他のプロプリエタリのデータベースからMySQLへアプリを
マイグレートします。ベテランのMySQLのコンサルティング
のチームは知識と技術をクライアントに技術トランスファーし
時間と経費を節減しマイグレーションの工数を正しく見積もり
プロジェクトのリスクを削減します。

この時期限定のオファーの他MySQL は短期及び長期の性能チューニング
や最適化、設計、アーキテクチャその他の「ベストプラクティス」
に関するコンサルティング・サービスを提供します。MySQLの
プロフェッショナル・サービスの詳細は
http://www.mysql.com/consulting.
