ただこれですと mysql のインストールフォルダの変更に
関してのもののようで、こちらの趣旨とは少々違うようです・・・

こちらでも鋭意調査中ですが、
CREATE DATABASE で新規にデータベースを構築する際
あくまでも「データフォルダ」下にディレクトリが追加されるのみで
例えば、

A というデータベースは 「C:\Bukken\麻布十番」
B というデータベースは 「D:\Bukken\神田佐久間町」

という風にはできないと考えるべきなのでしょうか？

例えば ADO 何かですと、以下みたいに新規に任意のディレクトリや
ファイル名を指定してデータベースを構築できるのですが・・・

＃ Delphi のソースで恐縮です。m(_ _;)m

procedure CreateDatabase;
var
  objCAT: OleVariant;
begin
  objCAT := CreateOleObject( 'ADOX.Catalog' );
  objCAT.Create
    ( 'Provider = Microsoft.Jet.OLEDB.4.0;' + 
      'Data Source = C:\Bukken\麻布十番.mdb' );
  objCAT := Unassigned;
end;
