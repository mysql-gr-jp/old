1.
ドキュメントが古いのかなんなのか、いまどきPKGからインストールすると"mysql_install_db
が済んだ状態"でインストールがされるような気がする。だから、あとはもうMacintosh
HD
直下のライブラリの中にあるPreferencePanesにMySQL.prefPaneを入れて、シス
テム環境設定で出てくる水色の丸いMySQLをクリックして「Start MySQL Server
」をクリックすれば新しい方のMySQLが起動する、ような気がする。
2. /etc/my.cnfがある場合(確かMacOS X Serverは標準状態では/etc/my.cnfが
なかった記憶がある)、標準と自前と両方が参照してしまってわけわからんこと
になるかもしれないので注意。
3. せっかく4.0系から5.0系にデータを持っていくのだから漢字コード関係の醍
醐味を存分に楽しもう。
4. ちなみにあたしゃ標準と自前を並列で入れたことはないので、スッと入って
パッと動くかどうかは知らない。
5. Macユーザたるもの、マニュアル/ドキュメント類を見てはならない。まずダ
ブルクリック、次にダブルクリック、そしてクリッククリッククリック。それ
で使えないようなソフトは使わない。
