> 　3.23は機能を追加する等を含めて開発版としてリリースされているのだと思います。
> 　その機能が（他の機能との整合性などで）安定したら3.24になるということですよね？

MySQL の場合は、Linux カーネルとはバージョンの付け方が異なっていて、
同じマイナーバージョンで alpha → beta → gamma → 無印(安定版) となります。

例えば(バージョンはデタラメですが)、3.23.1～9 は alpha, 3.23.10～15 は
beta, 3.23.16～20 は gamma, 3.23.21～ は安定版というような感じです。

# 私は Linux 方式よりもこっちの方が好みです。
