古畑と申します。

私も同じ経験をしたことがあります。

Portsのdatabases/mysql40-serverからインストールした場合、
同時にインストールされるmysql40-clientのコンパイル時に
文字コードセットの指定(WITH_CHARSET=sjis)が引き継がれないようです。
クライアントがlatin1しか知らないので、statusで表示した時に
サーバ、クライアント共にlatin1と表示されているのでしょう。
ここらあたりはきちんと確認したわけではないのですが…。

ということでPortsのdatabases/mysql40-clientを
# make WITH_CHARSET=sjis WITH_XCHARSET=all install
でインストールし直してみてください。

私はこれで解決できました。
