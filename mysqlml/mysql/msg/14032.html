富士和電子 伊藤浩樹 どの。
意味はよくわかりませんが、
--defaults-extra-file=path
http://dev.mysql.com/doc/refman/4.1/ja/option-files.html
もためしてみてください。

> ./bin/mysqld_safe --user=mysql --defaults-file=/etc/my5.cnf&
> として起動しようにも
> 
> Starting mysqld daemon with databases from /usr/local/mysql/data
> STOPPING server from pid file /usr/local/mysql/data/INTRA.FUJIWA-NET.COM.pid
> 070601 09:36:09  mysqld ended
> となり起動が出来ない状態です。
> 
> my5.cnfの抜粋
> [mysqld]
> basedir         = /usr/local/mysql
> datadir         = /usr/local/mysql/data
> pid-file        = /usr/local/mysql/data/mysql.pid
> port            = 23306
> socket          = /tmp/mysql.sock
> 
> これに設定しているpidファイル名も違うことから読んでいないと判断できそう
> です。
> 
> ソースからconfigureオプションで指定し構築したいと思います。
>
