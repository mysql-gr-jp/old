森山さんとも被りますが、一番確実な現状把握の手段は、
Access の VBA を使って、次の SQL を MySQL Server に発行し、
character_set_* と、collation_* の値を確認することだと思います。

SHOW SESSION VARIABLES;
SHOW GLOBAL VARIABLES;

ちなみに上記SQLは phpMyAdmin 上ですと、
システム変数のセッション値、グローバル値を取得するために発行されます。
