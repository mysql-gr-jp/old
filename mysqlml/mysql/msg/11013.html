うーん「exe ファイルを追加で入れたくない」のだと、どうやって
Web ベースで検索、表示させる方法があるのか、WebDAV 以外には
思いつきませんね。

少なくとも私、bon さん、ohiromas さんの示した解決方法はペケ
ですよね(私以外の方の方法では結果を引き取って加工するプログ
ラムが必要です)。
Windows で Web ベースでは「exe ファイルを追加で入れる」のは、
cgi を入れるのと同じです。何も cgi って Perl や Ruby で無い
といけない訳ではなく、C++ とかの言語でもいいんです。
またスクリプト系言語で cgi(又はそれに類したもの)を作る事も
同様な訳ですから．．．

しかし運用の狙いを汲み取ると WebDAV だけではどうも解決しない
様な感じがします。WebDAV + namazu ではそれぞれが独立した機能
でうまく運用で満足できるかどうか不明ですね。

どうしても Web ベースとなると、やはりその辺をうまく整理して
くれる実行プログラム(exe とは限りませんが、cgi や Java サー
ブレットなど)が必要になると思いますが、如何でしょうか。
