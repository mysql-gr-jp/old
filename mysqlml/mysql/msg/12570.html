MySQL 5.0.15では、接続したらまずクライアント側の文字コードセット
を教える必要がある場合が多いです。

私の場合ODBC設定のInitial Statement欄に以下のように書いています。
もちろんVBAでselect前に以下のクエリを出してもいいはずです。

   set names sjis; (またはset character set sjis;)


コマンドラインからだとうまくいくのは、my.iniの[mysql]欄にdefault
-character-set=sjisが書いてあり、暗に同等のクエリが実行されてい
るからだろうと思います。
