Subject: [mysql 15821] データベースレベルの権限変更を接続済みクライアントに即座に反映させたい

長永と申します。

以下のマニュアルページにて、「データベースの権限での変更は、次の USE db_name ステートメントを発行するときに発効する」とありますが、
USE db_name 以外のステートメント発行時にも GRANT/REVOKE の変更内容を既存のクライアント接続に発効させることはできないでしょうか？

MySQL :: MySQL 5.1 リファレンスマニュアル :: 4.7.7 権限の変更が反映するタイミング
http://dev.mysql.com/doc/refman/5.1/ja/privilege-changes.html

具体的には、あるデータベースに対するあるユーザー権限が REVOKE されたら、
その権限による既存の接続をすべて即座に遮断したい、というのが実現したい内容です。

ワークアラウンド的な方法でも結構ですので、ご存知の方がいらっしゃいましたらご教授いただけると嬉しいです。
