こんにちは。

> httpd.conf　の、AddDefaultCharset ISO-8859-1
> の部分を、AddDefaultCharset none　に変更したら、直りました。

これはあくまで Apache2 の場合ですね。
Apache2 だと Windows だろうが Linux だろうが、ここをコメント
アウトしないとまともに日本語表示されません。

一部が表示されないとのことですから、原因は別のところにある
のではないかと思います。

原因の切り分けは、phpMyaAmin を用いて、MySQL に格納されてい
る文字が化けていないかどうかから調べたほうがいいでしょうね。

次に、その結果から、php のコーディングを調べた方がいいでしょ
う。データベース中で化けていないのでしたら格納時は問題ない
訳です。表示部分だけ調べればいいですね。化けているのでしたら
コーディングより環境の問題から洗い直し、でしょう。

MySQL のバージョン 4.0x -> 4.1x で文字化けしたなら、これが
解決方法でしょう。
