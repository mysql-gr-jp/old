とみたです。

「一週間でマスターする SQL for Windows ～ MySQLで始めるデータベース入門」
という本が 7/11 に出版されました。
http://book.mycom.co.jp/book/4-8399-1058-8/4-8399-1058-8.shtml

毎日コミュニケーションズさんから、ユーザ会に１冊寄贈されましたので、
抽選で１名様にプレゼントします。

内容はちゃんと読んでませんが、1週間でデータベースの基礎から Webアプリ
ケーションの作成までやるみたいです。全ページフルカラーで見易いです。

希望者の方は、私宛て（ML宛てではなく）にその旨のメールを送ってください。
締め切りは 8/17 です。当選者は ML で発表します。

抽選方法は、私のところに届いたメールを cksum に喰わして、一番数字の大
きかった人が当選です。Received ヘッダ等の不確定要素があるので、数字が
大きくなるようにメールの内容をあれこれ考えるのはたぶん無駄です (^^;
