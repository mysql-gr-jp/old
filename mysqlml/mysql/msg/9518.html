ハイライト
組み込み用 MySQL
組み込みサーバー用のライブラリーは小型機器や組み込み用のデータベース
に最適です。ISVやリセラー(VAR)はソフトのアプリケーションや、ターンキー
のハード・ソフトからなるアプライアンスや、ルータ、キオスク、
その他多くに組み込み用 MySQLを搭載しています。
詳細は
http://www.mysql.com/products/embedded/

MySQL ケーススタディ: MTA
Matanuska Telephone Association (MTA) はアラスカ最大の電話協会
で１万マイル四方をカバーして3万8千人のメンバーが参加しています。
MySQLは通信をつかさどる一番重要なデータを管理しています。通信方法に
は無線、高速インターネット、長距離、ビジネスコミュニケーション
を含みます。

http://www.mysql.com/it-resources/case-studies/mta.php

製品情報
MySQL 4.0.20 がリリースされました
この版は現在の製品版の保守用です。この版には以下
の機能を含みます。

MATCH ... AGAINST ( ... IN BOOLEAN MODE)での語句の
検索は語句の部分的なマッチは行わない
タイムスタンプのフィールドの長さが１９であれば、
タイムスタンプは「YYYY-MM-DD HH:MM:SS」として
表示されます。これはMySQL 4.1でつくられたテーブルが
MySQL 4.0で容易に使用できるためです。
追加オプション--replicate-same-server-id.
多くの改善とバグ修理
詳細は
http://lists.mysql.com/announce/191
ダウンロード
http://dev.mysql.com/downloads/mysql/4.0.html

MySQL Connector/J 3.0.12 がリリース
MySQL Connector/J 3.0.12, はType-IV all-Java JDBC
ドライバーです。リリースの内容は

'failOverReadOnly' を追加し多くのバグ修理のほかに
フィールド　オーバーの際の接続の状態 (read-only/writable)
をユーザがコンフィギュレーションをできるようになりました。
Unsigned attribute を TYPE_NAME コラムでDatabaseMetaData.getColumns() output
に追加しました。
'url' パラメターを MysqlDataSource and MysqlConnectionPool
DataSource に追加して、他のプロパティを内部のアップサーバーから
パスすることを可能にしました。
多くの改善
詳細は
http://lists.mysql.com/announce/192
ダウンロード
http://dev.mysql.com/downloads/connector/j/3.0.html

3.51.07版のMySQL Connector/ODBC

ションに

他に多くの変更

http://lists.mysql.com/announce/188
ダウンロードは

7.5.00.12版MaxDB がリリースされました。
この版は製品版のMaxDBの保守用です。この版はステーブルなSQL
Database Connectivity (SQLDBC)の機能を含む最初の版です。
SQLDBCはアプリケーションがデータベース内でSQLのコマンドを
実行したりデータにアクセスしたり変更を可能にします。 SQLDBC は
7.3.00版以降をサポートします。
詳細は
http://dev.mysql.com/doc/maxdb/changes/changes_7.5.00.12.html
ダウンロードは
http://dev.mysql.com/downloads/maxdb/7.5.00.html

MySQL SET データタイプ
多分一番使われていないMySQLのデータタイプはSETデータタイプでしょう。
しかしながら、MySQL SET データタイプは正しい使用時には大変
有益です。この記事はDatatypeSET データタイプの詳細とどのようにして
活用するかに迫ります。
詳細は
http://dev.mysql.com/tech-resources/articles/mysql-set-datatype.html

PHP とMySQLの架け橋の ext/mysqli , パート１:
概略
これはPHP5用のMySQLの新しい拡張機能についての3回シリーズの
最初です。PHP5 は4.1以降のMySQL の機能をサポートします。
90年中ごろからext/mysqlはPHP とMySQLの架け橋の役割を果たして
きました。Georg RichterはPHP5用のMySQLの新しい拡張機能を
開発しました。PHP5 は4.1以降のMySQL の機能をサポートします。
詳細は
http://www.zend.com/php5/articles/php5-mysqli.php
