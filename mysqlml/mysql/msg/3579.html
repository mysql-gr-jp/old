c:\mysql
にインストールしたのでしょうか。
ディフォルトではそうですね。

c:\mysql\bin\winmysqladmin.exe
これをダブルクリックします。

初めて起動するときはユーザー名とパスワードを 求めてきますので、
root
rootのパスワード
を入れてください。
（求めてこない場合はそれでもOK）

右下にアイコンが表示されますので、
右クリック→Show me

WinMySQLAdminのmy.ini Setup タブをクリック。
左の部分のラジオボタンで mysqld をチェックします。
右側の設定部分は全て消去し、

---------------------------------
[mysqld]
port=3306
skip-locking
default-character-set=sjis
set-variable=key_buffer=16M
set-variable=max_allowed_packet=5M
set-variable=thread_stack=128K
set-variable=flush_time=1800

[mysqldump]
quick
set-variable=max_allowed_packet=16M

[mysql]
no-auto-rehash
default-character-set=sjis

[WinMySQLadmin]
Server=C:/MYSQL/bin/mysqld.exe
--------------------------------
こんな風にかきこんでください。

最後に左の Save Modificationをクリック。
はい→OK

WinMySQLAdmin上のどこでもいいので、
右クリック→Hide Me

これで設定完了です。

最後に右下のWinMySQLAdminアイコンを
右クリック→Win 9x→ Shutdown Both→はい

これでWinMySQLAdminを終了。

そしてもう一度
c:\mysql\bin\winmysqladmin.exe
をダブルクリックして起動します。
今度はパスワードなど求めてこないはずです。
そしてすぐに右下のタスクバーに入るはずです。

あとは、mysql -u root -p でMS-DOSからアクセスすると、
mysql client もSJISに設定されているので、両方がSJISになるはずです。
MySQLサーバーをShutdownする場合は、同じように右下のアイコンを
右クリック→Win 9x → Shutdown Both
とします。

Hope this helps!
