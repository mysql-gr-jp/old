"A.K.I." <aki@xxxxxxxxxx> wrote:
 |
 | FreeBSD上のApacheとmod_log_mysqlで同じエラーログを大量に吐いてます。

このエラーは http://www.mysql.jp/mysql/TIPS/clients.html によると

1. ujis と sjis をサポートしない libmysqlclient が入っている
2. その libmysqlclient を使ってコンパイルされた MySQL モジュール
   （この場合は mod_log_mysql ですね）が使用されている

ことが原因にあるので、FreeBSD なら適切な configure のオプションを
指定して ports から入れ直せばエラーは出なくなるような気がします。

## 試したわけではないので外していたらすみません。

FreeBSD packages の MySQL って多言語サポートが入ってない状態で
コンパイルされてるのかな？
