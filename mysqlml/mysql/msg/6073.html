---------------------------------------------------------------------------
   String result = new String(native, "XXXX");
   (※ nativeは、MySQLサーバから受け取ったバイトデータで、resultは
       JDBCドライバの利用側に返す値)

   byte[] native = result.getBytes("XXXX");

       nativeは、MySQLサーバに送信するバイトデータです)
---------------------------------------------------------------------------

  このような処理に、有る条件の時に何か処理を加えれば良いと思うのですが、
