MySQL AB はOracleとそのほかに攻撃を開始
2004年10月27日
http://searchoracle.techtarget.com/originalContent/0,289142,sid41_gci1020060,00.html

Robert Westervelt

MySQL AB は大規模エンタープライズに適する製品版の
データベースをリリースしました。

4.1版はサブクエリを通じての高度なクエリ機能、より
速くよりセキュアなクライアント・サーバ通信、ウインドウ・Linux用
のインストレーションとコンフィギュレーションのウイザード、
国際文字との地理的なデータのサポートを提供します。

市場調査会社のForrester ResearchのアナリストのNoel Yuhannay
によればローエンドではMySQLはシェアーを伸ばしていると
述べました。大規模なDBMSによくある機能を追加することで
この先2-3年のうちにMySQLの大規模な展開が予想されると
も述べました。

Yuhannaは続けて、「MySQLは性能のスケーラビリティの
解と機能アップを図っている。それは必ずもっとカスタマー
を獲得するのに繋がる。今までにエンタープライズの環境
をコントロールしてきた他のベンダーと対等に戦える
ことになる。」と語りました。

大きなデータベースの会社はMySQLを無視しているわけではありません。
Oracle, Microsoft とIBM は価格を下げて、データ管理の自動化
の機能を追加しています。

Yuhannaによれば、企業は無料のオープンソース版をダウンロードして
自社の環境で使用できるかテストしています。

MySQLは主にミッション・クリティカルでないアプリに使用されて
きました。しかし、企業が最初のテストの段階を過ぎれば
大部分が完全にサポートされた版に乗り換えるとYuhanna
は推測します。

企業がMySQLやその他のオープンソース DBMSに向かうのは
大部分価格のせいだとInterstate SoftwareのCEOのJohn Horn
は言います。MySQLは予算の少ないベンチャー企業の間で
人気がありますとohn Hornは続けました。

Alpha Software社の会長のRichard RabinesはMySQL は
開発者にも人気があり、Oracleやその他の大手と十分競争
ができると述べました。Alpha社は過去3年に渡りMySQLをつかって
おりそれ用のアプリケーションのツールのスイートを
開発しています。

「MySQLの使命はOracle やその他の大手と戦うことだ。MySQL
は高性能のデータベースエンジンを開発した。私の社員は簡単に迅速にウエブの
アプリを開発できる。」とRabinesは語りました。

もっとMySQLをミッションクリティカルなアプリに採用するため
にMySQLはサブクエリとderived テーブルを最新版で追加しました。
この機能を使ってユーザはデータセットをサーチできます。

MySQLはクライアント・サーバープロトコルをアップグレードしました。更に
prepared statementをサポートし、クエリの実行と警告のメッセージ
を改善しました。

MySQLはLinuxとWindows用に新しいGUI インストーラーとコンフィギュレーション
ウイザードを提供します。また、OpenSSL (GPL)を利用して暗号化された
クライアント・サーバーのプロトコルを開発しています。

開発者はMySQLのサーバーのライブラリーも改善しました。小さくして
組み込み用のサードパーティのソフトと迅速に作動 できるようにしました。
MySQLは現在utf8と ucs2 のUnicodeを使用してローカルな言語の
サポートが必要なアプリに対応しています。

製品版のMySQL 4.1は以下で動作します。Linux, Windows, Solaris,
Mac OS X, FreeBSD, HP-UX　とIBM AIX operating systems.

MySQL データベースは2重ライセンスです。

開発者と会社はオープンソース版(GPL) かコマーシャル版を選ぶことが
できます。これは展開する要求によります。コマーシャル版の値段はサーバー毎
に$595からです。
