私はLinuxのSlackware 3.4ですが、同様の状況でした。mysqlコンソールで日本語を
使うにはホームディレクトリに.inputrcというのを置いて、

set convert-meta off
set meta-flag on
set output-meta on

と書けばよい、ということだったように思います。これ、最初に結構悩んでどなた
かのWebページにチラッとサラッと書いてあっただけで見つけるまでに数日間動かせ
ませんでした。
 上記のTIPSで間違いなければ、どこかのめだつところにドーンと置いてあった方が
迷う方を減らせるんじゃないでしょうか。おそらく動いちゃえばあとはたいてい
PerlだのPHPだのCだのなんだのからキックしているから、あまりmysqlコンソールで
グリグリSQL叩いている方は少ないんでしょうねぇ。私はグリグリSQL叩いています
。
