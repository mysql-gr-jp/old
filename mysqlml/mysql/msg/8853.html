==>ハイライト
<==

2003/2004 MySQL アプリケーション　オブザイヤー

* 今年フロリダ州オルランドで４月14-16日に開催される
MySQLユーザコンファレンスで2003/2004MySQL アプリケーション　
オブザイヤーを授与します。この賞はMySQLを使用して
開発された優秀なアプリケーションを発表し、その技術を
開発した会社や個人を賞するものです。

*MySQL ケーススタディ: Enercon 社がMySQL社のMaxDB使用で大きく経費節減
数十億ドル規模のエネルギー会社のEnercon社はクローズドソース
（オープンソースに対して）300ユーザから750ユーザへ
の移行に初期投資とTCO（total cost of ownership）
がかかりすぎることからオープンソースへ移行しました。

==>製品<==

*MySQLのサブクエリ、パート１
4.1版でサブクエリを追加しました。この機能は2003年4月以降
テスト用に提供しています。連載記事で、Jeremy Coleが実際
的なサブクエリの使い方を例を提示しながら説明します。
まだ、サブクエリを使っていないあなたに最適な記事です。

*MySQL Administrator 1.0.1a がリリースされました
新しいグラフィックベースの管理クライアント、MySQL Administrator
がリリースされました。 MySQL Administratorはあなたのデータベース
の管理を容易にして、データベースがどのように作動しているかを
示します。ダウンロードページからLinux・ウインドーのどちら
の版もダウンロードできます。

*MySQL 4.0.18 版がリリースされました。
MySQL 4.0.18版がリリースされました。ダウンロードページから
さまざまなプラットフォーム用にソースコード・バイナリー形式
どちらもダウンロードできます。

4.0.18版の機能の一部を以下に示します。

新しいユーティリティの`myisam_ftdump'がバイナリ・ディストリビューション
に追加されました。`myisam_ftdump'はネットウエアーでも作動します。
キーの最初の部分が定数の際Optimizerがよくチューンニングされます。

*MySQL Connector/J 3.1.1 アルファがリリースされました
Type-IV all-Java JDBC　ドライバーの Connector/J 3.1.1
がリリースされました。3.1.1版は開発ツリーの新しいフィーチャー
リリースで MySQL-4.1または MySQL-5.0に最適です。
Connector/Jダウンロードページからソースコードおよび
バイナリーコード形式で入手可能です。

*ForresterとMySQLによるオープンソース戦略のウエブセミナー
日: 3月16日(火)
時間: 10:00am （太平洋時間）(1:00pm 東部時間)

MySQLとForrester Researchは1時間の無料ウエブセミナーで企業が
いかにオープンソースを使用しているかを教授します。
Forrester Researchの副社長のTed Schadler氏がオープンソース
の誤った常識をくつがえし、いかに企業がTCOを下げるため
にオープンソースを使用しているか講演します。このセミナー
であなたの会社でどのようにオープンソースの戦略を立てる
かのヒントをつかめます。

*MySQLが KOMCOM (2月25-26日) と CeBIT (3月18-24日)に参加
MySQL GmbHがドイツのハノーバで開催されるKOMCOM Nord ショー

==>トレーニング<==

新しいMySQLトレーニング・コース：MySQLと使用したデータベースへの初級コース
データベースの基本を学び、MySQLを使用始めます。
MySQLを使用してSQLの基本を学びます。一般的なデータベースを
学びたいかた、MySQLを使ったことのない方に最適です。

現在３つのコースがワシントンDC、ストックフォルムとサンフランシスコ
で予定されています。
