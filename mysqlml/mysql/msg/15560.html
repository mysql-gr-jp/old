あさかわ

VertrigoServってなんだろうと思ってググってみました。

Apache/PHP/MySQLなどをWindowsに一発でインストールできる「VertrigoServ」
http://gigazine.net/news/20080324_vertrigoserv/

ということで。

Winodwsに、Apache MySQL PHP をインストールする為のお手軽インストーラという事がわかりました。

原因は、このお手軽インストーラが、完璧じゃないという「ダケ」の事ではありませんか？
VertrigoServインストールの話であって、MySQLの話じゃないですね。

Windows+アプリケーションインストールの不具合問題です。

MySQL+PHP をお勉強されたいのでしたら

まずはUNIX系を先にやってみる事をお勧めします。
もともとUNIXのツールですから。

UbuntsuならOSのインストーラで全部入ります。

どうしてもUNIXが嫌でwindowsでやりたいというのならば。
VertrigoServに拘らずに、他のインストーラを使って見るというのもあるでしょう。

借りた本が、VertrigoServだったからといっても、インストール部分以外は、なんでも同じです。
また、本と現在のバージョンがちがうのかもしれません。

そういう事を考えると、一度は、apache php MySQLを、個別に入れて理解した方がいいです。

お手軽パッケージをつかうと、そのパッケージが内部で何をしてるのかわかりません。
一発でインストールできた場合は、ラッキーですが、そうじゃない場合は悲惨です。(実際そうでしょ？)

そうすれば、お手軽パッケージでハマッタ時の助けになるはずです。

1個インストールに成功すると、2個目は楽になるでしょう。

ちなみに、私は、インストーラでハマル&windowsでハマルのは嫌なので。
Apache PHP MySQLは、UNIXでソースをコンパイルして入れる派です。
結局この方が余計な事(インストーラが想定してる事etc)調べないで済むので楽なのです。
