「ケンタッキー」「パンタロン」で
like '%パ%' か like'%ケ%'の一文字ではうまく検索できますが、
like '%パン%' か like'%ケン%'でダメですし。

like '%パa%' か like'%ケa%'のはうまく検索できて、
like '%パab%' か like'%ケab%'ではだめな状態だったりするようです。

ただ単純に、大文字・小文字の区別がついていないと言う訳でもないような
気がするのですが。3.23.55とかでは問題ないようですし。
