そこで、3.23.28にアップグレードしようと
・MySQL-3.23.28-1.i386.rpm
 ・MySQL-client-3.23.28-1.i386.rpm
 ・MySQL-bench-3.23.28-1.i386.rpm
 ・MySQL-devel-3.23.28-1.i386.rpm
 ・MySQL-shared-3.23.28-1.i386.rpm
の各パッケージを得ました。

これでアップグレートしたのですが、今度は
ldd /usr/libexec/apache/mysql.so
としたときに
ldd: /usr/libexec/apache/mysql.so: No such file or directory
となり、mysql.so自体が消滅してしまいました。

3.23.28-1に特化したmysql.soを復活できれば
解決すると思いますので、模索してみます。
