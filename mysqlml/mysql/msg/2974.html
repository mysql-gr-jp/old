> perlでmysqlを使用するときのシングルクウォートのエスケープの
> 仕方を教えてください。

> 例えば、
> TEST というテーブルオブジェクトが存在し、単一のテキストカラムを持つとす
> る。
> そのカラムに「'90」なるデータを入力する際には、
> INSERT INTO TEST VALUES ('\'90');ではだめでしょうか？

多分 perl が \ を食っちゃってると思うので、\\ と書けばいいと思います。

	INSERT INTO TEST VALUES ('\\'90');
