お世話になっております。

[06339]> このクエリを送信しても単一の結果しかかえって
[06339]> 来ません＞＜
（中略）
[06339]> String query = "select * from Asa order by
[06339]> rand() limit 1";

limitオプションにパラメータを一個しか指定しないと
「検索結果の先頭からパラメータ値分の件数を表示しろ」
という意味になります。
従ってこの場合は、
「検索結果の先頭から一件を表示しろ」という意味になるので
当然「単一の結果しか返ってこない」と思われます。
