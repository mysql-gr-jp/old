SELECT
 FROM_DAYS(TO_DAYS(mail_tb.mail_datetime))      CL_DATE
,SUM(IF(HOUR(mail_tb.mail_datetime) = 0, 1, 0)) CT_0
,SUM(IF(HOUR(mail_tb.mail_datetime) = 1, 1, 0)) CT_1
,SUM(IF(HOUR(mail_tb.mail_datetime) = 2, 1, 0)) CT_2
...

という形で、あとはFROM,WHERE,GROUP BYをよろしく組めば
いけるのではないでしょうか。
