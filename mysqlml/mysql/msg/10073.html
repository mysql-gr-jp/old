事例: Pivex 社MySQLを使用して旅行・余暇市場に展開
利益上昇
Pivex社はMySQLを使用して Folio Exchangeを開発しました。
これは大手の旅行卸会社のアメリカン航空・バケーション、Pleasant
Hawaiian HolidaysやOutriggerホテルチェーン の$200ミリオン以上の
請求書の発行・料金の受け取りに使われます。 Folio Exchange は完全な
ターンキー・システムでホテルと旅行卸会社間の請求書の発行や料金の支払いに
かかる経費の90%を削減します。
MySQLの信頼性、高性能、低経費とプラグできるMySQLエンジンの
アーキテクチャのおかげでPivex社の Folio Exchangeは
成功しています。
詳細は
http://www.mysql.com/it-resources/case-studies/mysql-pivex-casestudy.pdf

MySQL Cluster はMySQL 4.1.4 Max版のバイナリーリリースとしてリリース
MySQL Clusterはクラスターの分散データベース管理システムを提供します。
高度にスケーラブルで、ミッションクリティカルなデータベースの
アプリを99.999%のアベイラビリティで提供します。MySQL Cluster は
共有部分がなくフェールオーバー、ノードリカバリー、
同期データ・リプリケーション、 単一障害点なし
などの機能を提供します。
ドキュメントは
http://dev.mysql.com/doc/mysql/en/NDBCluster.html
クイックスタートガイド
http://dev.mysql.com/doc/mysql/en/MySQL_Cluster_quick.html
ダウンロード
http://dev.mysql.com/downloads/mysql/4.1.html
