データベースのコンサルティングをMySQL から手軽にお手ごろな価格でご利用頂けます。 

パッケージになったコンサルティングの解は
アプリの設計、パーフォーマンスチューニングとマイグレーションに最適です。

カリフォルニア州クプティノ市発

世界で一番人気のあるオープンソースの開発会社の MySQL ABは
今日新しい３つのコンサルティングパッケージを発表しました。
これはIT部門に対してMySQLのベストプラクティスを元にすばやく
データベースを設計し高性能の実装を助けることを目的と
しています。

MySQL から認定されたコンサルタントがカスタマーのサイトで提供します。
比較的短いコンサル期間でアプリケーションを詳細に解析して
よく問題になるイシューに対する解決方法をアドヴァイスします。
例えば、アプリケーション・アーキテクチャや設計、パーフォーマンス
チューニング、とレガシーのデータベースをMySQLへのマイグレーションなど
です。

MySQL ABのサービス担当副社長のUlf Sandbergは「これらの
パッケージは何ヶ月ではなく何日のうちにクライアントの
皆様に結果をもたらすことができます。
短期間にクライアントの皆さまのMySQLデータベースの
アプリケーションを理解し改善するのにお役にたてます。
それにこれは全ての費用が含まれています。出張費もMySQL
持ちです。」と語りました。

この特別プログラムでMySQLのカスタマは３つの異なったコンサルティング
のパッケージを選択できます。

MySQL アーキテクチャと設計： 2 日でUS $5,000か EUR 4,000
２日でMySQL認定のコンサルタントはクライアントのITチームと
一緒に仕事をして、アプリケーションをリビューし、現在と
将来のデータベースのアーキテクチャを決定するかの
解析を行います。 これはアプリケーションのアーキテクチャが
正しくなされているかを保障することで開発の時間を短縮できます。
また正しいアーキテクチャーであれば、ビジネスの要求が
変更してもアーキテクチャーのやり直しが最小限ですみます。

MySQLパーフォーマンスチューニングとオプティマイゼーション：
3 日でUS $7,500 または EUR 6,000
３日で、MySQLのコンサルタントはクライアントが性能改善の
ためにボトルネックを突き止めるのお手伝いをします。同時に重要な
アーキテクチャーと性能のチューニングについて説明します。
典型的な性能改善は50-100%ですが、カスタムのチューニングを
行うとそれ以上になることがあり、高いハードウエアの買い替えを
しなくてもすむようになることが多いです。

MySQL マイグレーションのジャンプスタート：　5日でUS $10,000 または EUR 8,000
この5日間で、MySQLはクランアントがOracle, Sybase, Microsoft SQL Server,
DB2, Informix, やその他のデータベースからアプリケーションをMySQLへ
マイグレートする計画とその実行を助けます。ベテランの MySQL コンサルティング
チームは今までの豊富な経験を元にクライアントプロジェクトにその
ノウハウを伝授します。それで、時間やコストが削減されマイグレーションの
仕事量とプロジェクトの大きさを正確に把握することで、リスクを
最小限に抑えます。

このパッケージの他にMySQLは長期のカスタムのコンサルサービスも
提供します。MySQLのプロフェッショナルサービスの詳細は
こちらまで。http://www.mysql.com/consulting.
