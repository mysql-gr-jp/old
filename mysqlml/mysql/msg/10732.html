>　データを書き換えるとテーブル上にあるTimestampの情報が書き換えた日時に
>替わっ
>てしまいます。Timestampを変更せずにデータを書き換えるSQL文を教えて下さい。

timestamp フィールドは、insert 文や udpate 文で何も指定しないと自動的に現在
時刻に置き換わります。
これを元のデータのままにしておきたいときには、そのフィールドを'='で指定しま
す。次のようにすればよいのです。

	udpate TABLE set DATE = DATE;
