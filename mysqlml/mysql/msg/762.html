デフォルトではlocalhostからの接続しか許されていなかったと思います。

$mysql mysql
>select * from user;
とかやるとだれがどこからどんな権限で接続できるか確認すればできます。
接続したいユーザのレコードの先頭フィールドにホスト名を書けばアクセス
できるようになります。
たとえば、
update user set Host="Ｂ" where User="だれか";
とかでＢからＡにアクセスできます。
ただし上記の方法だとこのユーザはローカルから入れなくなるので追加のほうが
いいかも知れません。
