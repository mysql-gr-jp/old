製品
MySQL (5.0.13 Release Candidate) のリリース

5.0の最初のrelease candidateがリリースされました。
このリリースにはStored Procedure, View,
Trigger, Information Schema (Data Dictionary), XA などを含みます。
そのほかには:

* Windows上でのリプリケーション・サーバーの接続タイムアウト
の検知の改善
* Key_read_request, Key_reads,
Key_write_requestsやKey_writes などのstatus 変数のカウンターが
unsigned longがunsigned longlongに変換されロールオーバーなし
に大きな変数をハンドルできるように。
* 以前はLEAST()とGREATEST()関数は全てのパラメータが
NULLの時に限りNULLを変換。現在はOracleのようにどの1つのパラメータが
NULLでもNULLを返還。
* 幾つモノバグ・修理

今全ての力はバグ修理とMySQL５.0の安定化に注がれています。
いまから MySQL 5.0 Release Candidate を使用しましょう。

完全な変更のリストは
http://lists.mysql.com/announce/320

ダウンロードはここから
http://dev.mysql.com/downloads/mysql/5.0.html

iPod nano とＵＣのパスを当てましょう。
http://dev.mysql.com/mysql_5_contest.html
