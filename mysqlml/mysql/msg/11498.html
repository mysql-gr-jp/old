池田です。

> また、character set system　のグローバル値が　utf8 となっているのですが、これはどこかで設定できるのでしょうか？

character_set_systemはMySQLがメタデータを格納する際に使用する文字コードのことで、
これは常にutf8です。変更できません。たぶんこれはver4.1からの仕様だったかと。

メタデータというのは、"SHOW ほげほげ" で得られる各種変数や、"SHOW CREATE TABLE ほげ"
などで得られるテーブル定義、ver5.0系で追加されたinformation_schemaデータベースなど、
MySQLが内部で保持しているデータのことです。

蛇足ですが、つまりテーブル名やカラム名に日本語も使えてしまうということです。
