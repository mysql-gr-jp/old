ツール

TOOL 1. Instance ManagerはリモートのMySQLサーバーを管理できますか？
はい。 MySQL Instance Managerの詳細についてはこちらをご覧下さい。
http://dev.mysql.com/doc/mysql/en/instance-manager.html

TOOL 2. Windows上のInstance ManagerでLinux上のMySQLを管理できますか？
MySQL Instance Managerは現在のところUnix系オペレーティングシステムでのみ
使用可能となっています。つまりWindowsは含まれません。Windows版につきまし
ては現在積極的に開発中ですので、近々ご利用になれるでしょう。

TOOL 3. GUI (Graphical User Interface)ツールはLinux上で使用可能ですか？
MySQL Administrator (データベース管理者向け)、 MySQL Query Browser (開発
者向け）といったGUIツールはダウンロードすることができます。各ツールの詳
細につきましては http://www.mysql.com/products/administrator/ 及び http:
//www.mysql.com/products/query-browser/ をご覧下さい。既存ツールの機能強
化が進められていると同時に、更なるGUIツールが現在開発されています。

MySQL Administratorフォーラムはこちら
http://forums.mysql.com/list.php?34.
MySQL Query browserフォーラムはこちら
http://forums.mysql.com/list.php?108.

TOOL 4. MySQL AdministratorはMySQL 5.0の新機能に対応していますか？
はい。

TOOL 5. MySQL Query BrowserはMySQL 5.0の新機能に対応していますか？
はい。

TOOL 6. MySQL GUIツールをストアドプロシージャやビューの作成に使用できま
すか？
ストアドプロシージャ等の開発やデバッグを容易にする、さらなる機能の開発が
進められています。GUIツールはより高い頻度でリリースされ、新機能やバグ修
正を含んだ新しいバージョンが頻繁にリリースされます。

TOOL 7. バックアップ/リストア、及びmysqldumpはいつストアドプロシージャに
対応しますか？
MySQL 5.0の製品版（GA版）がリリースされる前には完成予定です。

TOOL 8. mysqldumpはいつストアドプロシージャ、トリガー、ファンクションの
定義をダンプできるようになりますか？
MySQL 5.0の製品版（GA版）がリリースされる前には完成予定です。
