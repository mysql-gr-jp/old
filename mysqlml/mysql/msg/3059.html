> $ mysql -hdb.hoge.com -uhogeuser -pogepass
> と接続すると以下のエラーが出て接続できません。
> ERROR 2003: Can't connect to MySQL server on 'db.hoge.com' (111)

1. db.hoge.com で、MySQLが動いていない。
   -> db.hoge.com 上で、mysql で再度確認
2. db.hoge.com がホスト名として正しくない。
   -> ping db.hoge.com で確認
3. db.hoge.com でMySQLのポートを閉じている。
   -> telnet  db.hoge.com 3306
      で、MySQLのバージョンのゴミが表示されればOK
      そうで無い場合は、tcpd (hosts.allow)などの設定を見直す。

という事が考えられると思います。

> マニュアルを見て設定したつもりなのですが、何か誤りがあるのでしょうか？

セキュリティ上（ユーザ登録）の問題だと、他のエラーメッセージがでます。
