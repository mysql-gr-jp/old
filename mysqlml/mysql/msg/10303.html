2005年 MySQLユーザーコンファレンスへのご招待

2004年10月5日

コンファレンス・テーマ: "MySQL どこにでも"

組み込み式用の解から、ウエブ・コマースや重要なビジネスのアプリまで、
MySQL はどこにでも使われています。世界中で5百万以上のインストレーション
です。この人気をよんでいるオープンソースのデータベースは
産業のリーダであるGoogle, NASA, Sabre, やSuzukiなどに採用され
多くの人はMySQLがソフトウエアー産業に革命的影響を与える
キーの役割を果たしていると思っています。新しいユーザをMySQLに
紹介し、開発者やITのプロフェッショナルに最新の成果を伝えるために、
MySQL AB はO'Reilly メディアと組み３回目のMySQLユーザコンファレンス
をカリフォルニア州サンタクララで、2005年の4月18-21日に開きます。

このコンファレンスは、現在までのMySQLのコミュニティの集まりで、
IT産業の有名人に参加頂き、MySQLの新しい製品、詳細なセッションや
チュートリアル、最新のオープンソースの技術の展示、最高のデータベース
やオープンソースの専門家を網羅します。現在確認されているスピーカは
GnomeのファウンダーのMiguel de Icaza、MySQLのファウンダーの
Monty WideniusとDavid Axmark, CEOの Marten Mickos, アーキテクチャ・
ディレクターのBiran Akerとコミュニティ・マネージャーのArjen Lentzです。

2005年の MySQLユーザーコンファレンスで発表を希望される
個人と企業はプロポーザルを11月1日までに提出してください。

トピックは：

    * アドミンとセキュリティ
    * アーキテクチャーと技術
    * ビジネス・ケースと事例
    * データウエアハウスとデータマート
    * Java
    * LAMP (Linux, Apache, MySQL, PHP/Perl/Python)
    * MaxDB
    * マイグレーション
    * .NET
    * 性能チューニングとベンチマーク
    * スケーラビリティと高アベイラビリティ

参加登録とプレスの登録は2004年12月に始まります。
