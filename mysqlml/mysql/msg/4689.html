こんなのどうでしょう？へんてこなSQLですけど^_^;

select *
from Table1
order by (case when 金額 = null then 999999 when 金額=1 then 90 else 金額
end);

この場合　金額=NULL　のときは　999999　で
　　　　　金額=1 　　のときは　90　　　で
　　　　　上記以外は金額そのままで

で並べ替えができると思いますが、まちがってたらごめんなさい。
