> DROP DATABASE のみを禁止することはできないように思われます。


そこにかいてありますよ。


	 データベース レベル

データベース権限は既存データベース内の全てのオブジェクトに適応します。これらの権限は mysql.db とmysql.host テーブル内に格納されています。GRANT ALL ON db_name.* と REVOKE ALL ON db_name.* はグローバル権限だけを供与、廃止します。

	 テーブル レベル

テーブル権限は既存テーブル内の全てのカラムに適応します。これらの権限は mysql.tables_priv テーブル内に格納されています。GRANT ALL ON db_name.tbl_name と REVOKE ALL ON db_name.tbl_name はグローバル権限だけを供与、廃止します。
