ただ、私の書き方が悪かったのですがやりたかったことは、

型	フィールド長	項目名			Option
数値	4		年			Key-U-N-A
数値	2		月			Key-U-N
数値	2		日			Key-U-N
文字列	24		当番者の氏名		N
・
・
等のようにエクセルで作成したデータ（又はCSV）から、

CREATE TABLE temp1 (
  年 smallint(4) unsigned NOT NULL auto_increment,
  月 tinyint(2) unsigned NOT NULL,
  日 tinyint(2) unsigned NOT NULL,
  当番者の氏名 varchar(24) NOT NULL,
  PRIMARY KEY  (年,月,日)
) TYPE=MyISAM;

のようなCREATE文を作成したかったのです。
データのインポートはいろんなツールでできるのですが、
テーブルの仕様をエクセルで作成し、
テーブル作成ができれば項目が多い場合便利だと思ったので。
テーブルの仕様変更まで考えると大変そうですが、
作成のみであれば、PerlやPHPで
CSVファイルアップロード→テーブル作成のようにして
すぐ作れそうですが、ものぐさですみません(汗)
