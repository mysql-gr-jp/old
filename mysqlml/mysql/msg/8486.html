Subject: [mysql 08486] 3.23.40 和訳マニュアルの chm 化

初めまして，増田と申します．

Windows + Python + DB +Zope で開発をしていて，これまで PostgreSQL を使
っていたのですが，最近 MySQL への移行をはじめました．

手始めに， Windows 環境の方がドキュメントを参照しやすいように
http://www.mysql.gr.jp/jpdoc/
の 3.23.40 の texi ソースから Windows ヘルプ形式を作成してみました．

差し支えなければ公開したいと考えています．ユーザ会様の方でダウンロード
できるようにしていただければと思っています．

いかがでしょうか．

--- 補足 ---
ドキュメントの作成は， html help workshop で行いました．

[on cygwin, after cvs checkout from 3.23 trunk]
(texinfo ソースのマージ)
cat texi.?? > mysql3.23
(章，ノードごとに分割するよう指定してコンパイル)
texi2html -split_chapter -split_node mysql3.23
(prechm.py は Python ソース配布に含まれるスクリプトで， html から stp, 
hhc, hhk ファイルを作成します． 正しくインデクスを生成できるように， 
一部改変して使用)
python prechm.py -v 3.23jp mysql3.23
(インデクス情報が全て EUC-JP なので， SJIS に変換)
python kf.py mysql3.23.stp
python kf.py mysql3.23.hhc
python kf.py mysql3.23.hhk

この後， HTML Help Workshop でヘルプをコンパイルしました．
