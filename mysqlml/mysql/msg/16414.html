こんにちは、Mroongaプロジェクトの林です。

Mroonga 7.04をリリースしました！

  http://mroonga.org/ja/blog/2017/06/29/mroonga-7.04.html

7.03からの変更点:
  http://mroonga.org/ja/docs/news.html#release-7-04

MroongaはMySQLで日本語全文検索を実現するストレージエンジンです。高速で
あることや位置情報検索をサポートしていることなどが特徴です。詳細が気に
なる方はドキュメントをご覧ください。

  * Mroongaの特徴 ― Mroonga v7.04 documentation
    http://mroonga.org/ja/docs/characteristic.html#what-is-mroonga

今月はDebian 9 (stretch)向けのパッケージの提供をはじめました。
以下のようにして簡単にインストールすることができます。

  $ sudo apt install% sudo apt-get install apt-transport-https
  $ sudo apt-get update
  $ sudo apt-get install -y --allow-unauthenticated groonga-keyring
  $ sudo apt-get update
  $ sudo apt-get install -y -V mariadb-server-10.1-mroonga

mariadb-plugin-mroongaというパッケージもありますが、そちらはMariaDBに
バンドルされているやや古いバージョンのMroongaです。
mariadb-server-10.1-mroongaパッケージで最新のMroongaを体験してみてくだ
さい。
