tinyint も enum も１バイトなのでディスク使用量は同じですね。

tinyint は単純な整数比較ですむけど、enum は比較的複雑な処理をしないと
いけないので、tinyint の方が速いような気がします。未検証ですが。

一応 bool という型もあるようです。実際には tinyint の別名です (^^;

あと、4.1.x では true, false というリテラルもあります。それぞれ 1, 0 
と同じです。
