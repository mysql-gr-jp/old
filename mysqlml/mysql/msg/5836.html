LAST_INSERT_ID()関数を利用すれば良いのではないでしょうか？
最後の作成された ID はそれぞれのコネクション毎にサーバーに維持されます。
よって、他のユーザに変更される事はありません。

<%
    '新規掲示を登録
    sql = "insert into BBS (掲示内容) values (掲示内容)"
    conn.Execute(sql)
    'refnoを更新
    sql = "update bbs set refno = bbsno"
    sql = sql & " where bbsno = LAST_INSERT_ID()"
    conn.Execute(sql)
%>
