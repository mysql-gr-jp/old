成松です。菅原さんご指摘ありがとうございます。

mysqldumpのバージョンを上げ、「--compatible=mysql40」 を付け加えることで上手
くできました。

以下手順です

mysqldump ( Ver 9.08 Distrib 4.0.14, for Win95/Win98 (i32) )
から実行すると
mysqldump: ERROR: unknown variable 'compatible=mysql40'
となってしまうので、mysql 4.1をインストールして

mysqldump ( Ver 10.9 Distrib 4.1.9, for Win95/Win98 (i32) )
で実行すると　TYPE=ISAM が書き出されました。
