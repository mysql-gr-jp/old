私としては、glibc の iconv や Perl 5.8 の名前に合わせてほしいです。
「cp932」、「eucjp-ms」かな。さらに別名も指定可能ならいいのですが。

$ iconv --list |egrep -i '932|euc.*jp.*ms'
CP932//
CSIBM932//
EUC-JP-MS//
EUCJP-MS//
IBM-932//
IBM932//
MS932//

$ perl -e 'use Encode;print join("\n",Encode->encodings(":all"));' |egrep -i '932|euc.*jp.*ms'
cp932
(Perl 5.8 は EUC-JP-MS をサポートしていません)
