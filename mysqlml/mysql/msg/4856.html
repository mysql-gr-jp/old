Windowsのほうはわからないのでこっちだけ。
ほとんどのPerlモジュールに共通することですが、perl Makefile.PLを実行するときにオプションを指定しないと、Perlの標準のライブラリのディレクトリにモジュールをインストールします。ですので、どの資料を見てもrootでの設定方法しか載っていないんだと思います。

root以外のユーザでインストールする場合にはそのユーザで書き込みできるディレクトリにインストールし、あとはuse libなどで、インストールしたディレクトリからモジュールを読み込むようにすればOKです。具体的には

% perl Makefile.PL PREFIX=~/perl
% make 
% make test
% make install

等としてあげればいいと思います。
