> 疑問に思ったのですが、selectやinsertにかかった時間を
> ミリ秒単位でログ等に吐き出すことはできないのでしょうか。

私の場合、プログラムに以下のようなのを入れてます。
(javaからJDBCで使う時)

// 開始時間取得
long startTime = System.currentTimeMillis();

/* select や　insertの処理 */

// 終了時間取得
long stopTime = System.currentTimeMillis();

 // 所要時間
 System.out.println( (stopTime - startTime) + "ms");
