==============
MySQL 株式会社は今秋、アジア初となる MySQL Users Conference Japan 2007 を開
催いたします。

2日間の日程で基調講演、日本語または英日同時通訳による十数のセッション、出展
ブースを予定しております。

セッションには、2007年 4月に米国で開催した MySQL Conference & Expo 2007 から
特に人気の高かった MySQL 高可用性ソリューション、パフォーマンスチューニン
グ、また日本のユーザ様による事例解説などを揃え、既に MySQL をお使いの方から
導入検討中の方までお楽しみいただける内容を計画しております。

是非、皆様にご参加いただけますようお願い申し上げます。

詳細は、決まり次第下記ページでお知らせいたします。
http://www-jp.mysql.com/news-and-events/news/article_1325.html

会期: 2007年 9月11日(火)、9月12日(水)の2日間
会場: 日本科学未来館 http://www.miraikan.jst.go.jp/j/guide/map_index.html
入場料: 事前申込みの方は無料、当日参加の方は5,000円。事前申込みは 7月に開始
予定です。
