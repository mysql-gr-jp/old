INSERT INTO TEST VALUES ("'90");
では、だめですか？？

私は、専らMySQL は、Perl + DBD::Mysql + DBI で
使用していますが、扱うデータ中に、ダブルクォートが
出てくる頻度に比べて、シングルクォートが圧倒的に
多いので、

$mysql = DBI -> connect(db_name,user,pass);
$sql = qq/UPDATE TEST SET OS="WIN'95" WHERE ID=1 || ID=2/;
$opr = $mysql -> prepare($sql);
$opr -> execute;

こんな感じで、利用しています。
