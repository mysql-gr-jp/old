Subject: [mysql 12546] MySQL の mysql-bin ファイルと DB の移動について

はじめまして、石川と申します。


MySQL 5.0.12 を使用しているのですが、
現在、使用しているスライスがもう少しで一杯に
なってしまい、DB領域を移したいと考えています。

◆mysql-bin.000001のファイルについて
現在稼動しているDBで以下のファイルが作成されています。
【ファイル名】
mysql-bin.000001
mysql-bin.000002
mysql-bin.000003
　　　：
　　　：

このファイルはmy.cnfの中で設定されて、
出力されているようですが切り替わったものについては、
移動させても問題ないでしょうか？

【my.cnfの設定】
# Replication Master Server (default)
# binary logging is required for replication
log-bin=mysql-bin


◆MySQLのDBデータ移動について
/var/lib/mysqlにあるDBを
別のディレクトリに移動したいと考えています。

スタートスクリプトに書かれている、
「datadir」を書き換えてディレクトリを変えようと思ったのですが、
起動時にmy.cnfも読み込んでいるようでしたので、
my.cnfのファイルも書き換える必要があると思います。
この為、/var/lib/mysql配下全てを別ディレクトリ移動させ、
/var/lib/mysqlからその別ディレクトリに対して
シンボリックリンクをはる方法が無難かと思うのですが、
他に何かよい方法はありますでしょうか？
