おはようございます、たまに出てくる小岩です。

>テーブルの設定で、
>
>たとえば
>colum｜   col1     col2      col3         col4
>value｜   hoge     foo     col1と同じ    col2と同じ
>			 ~~~~~~~~      ~~~~~~~~~
>	       		このようにデフォルト値として、
>	       		col1やcol2の値を入れたい場合
>	       		どのようにSQL文を記述すればいいのでしょうか？

http://www.database.ne.jp/jpdoc/mysql-3.22.21/Docs/manual.ja_Reference.html
#CREATE_TABLE にあるように、CREATE TABLE文で
  col_name type [NOT NULL | NULL] [DEFAULT default_value] [AUTO_INCREMENT]
            [PRIMARY KEY] [reference_definition]
の中のDEFAULを使えばいいんじゃないですかね。
CREATE TABLE sample(
col1 char(10) default 'hoge', 
col2 char(10) default 'foo'
);
みたいな。すでに作ってしまったtableならALTER TABLE文で
ALTER TABLE sample change col1 col1 char(10) default 'hoge';
でいけるような気がします。
