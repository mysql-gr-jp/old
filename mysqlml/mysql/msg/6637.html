どうまえです

In message "[mysql 06636] サーバー上の MySQL への接続ができません",
At Fri, 29 Nov 2002 11:42:40 +0900,
名執　聡 Wrote.
> MySQLコマンドでのコマンドラインで
> mysql -u mysql -h ＩＰアドレス または mysql -u mysql --host=ＩＰアドレス
> と実行しますと
> ERROR 1130: Host 'アクセスポイント名.ap.so-net.ne.jp' is not allowed to
> connect to this MySQL server
> のエラーが表示されます（現在使用しているプロバイダはSo-netです）。

MySQLのアクセス権の設定だと思われます。


> レンタル業者の方に問い合わせてみると
> 「こちらは、以下のコマンドにて接続を確認いたしました。mysql -u root -p -h
> localhost
> (rootのパスは設定されておりません)

localhost からは接続できるみたいなので、一旦 localhost から接続
して、GRANT 文で正しいアクセス権を定義すればよいでしょう。

http://www.mysql.gr.jp/jpdoc/3.23.x/manual.ja_toc.html#GRANT
このあたりに書いてあります。


セキュリティには十分ご注意ください...
下手すると誰にでもアクセスできるDBになったり、通信内容が他に漏れ
たりしますので。
