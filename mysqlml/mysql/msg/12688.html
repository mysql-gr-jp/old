製品
MySQL 5.1.5 コミュニティ版のリリース(Alpha)

これはMySQL 5.1の最初のアルファ版です。
このリリースでは次の新機能と変更を含みます。

*追加：XML functions ExtractValue()と UpdateXML()
ExtractValue()はXPathのエクスプレッションにマッチするXMLのフラグメントを
返還。
* 追加： --base64-output オプションをmysqlbinlogに。base64エンコーディング
を使用して全てのバイナリログの内容を印刷
* 追加： --port-open-timeout オプションをmysqld に。開封できない
時にサーバーがTCP/IPのポートがフリーになるのを待つ秒数のコントロール
を可能

変更の詳細
http://lists.mysql.com/announce/342

MySQL Connector/PHP のリリース

The MySQL Connector/PHP は mysql/mysqli PHP 拡張
から成り立っています。これは最新版のMySQLのクライアントのライブラリー
の版にあわせてビルドされています。php.netから
のmysql/mysqli PHP 拡張の利点だけでなく、MySQLのクライアント
ライブラリーへの最新のバグフィクスがあるだけでは
なく全てのMySQLのサーバーの4.1.16+と5.0.18+ のすべて
の機能をサポートすることです。

変更リスト
http://lists.mysql.com/announce/340
ダウンロード
http://dev.mysql.com/downloads/connector/php/

MySQL Connector/J 5.0.0 (Beta)のリリース

MySQL Connector/J 5.0.0 はType IV all-Java JDBCの新しい版です。新たしい
版の番号はJDBCがサポートするMySQLのサーバーの版と連携する
ようになっています。新しい機能と変更は

* url subprotocol :jdbc:mysql:mxj://…を介したConnector/MJX 結合
のサポート
* XA Distributed Transactions via javax.sql.XADataSourceの実装の
com.mysql.jdbc.jdbc2.optiona.MysqlXADataSource を介したXA
Distributed Transaction
* Statement.cancel() とStatement.setQueryTimeout().

変更
http://lists.mysql.com/announce/339
ダウンロード
http://dev.mysql.com/downloads/connector/j/5.0.html
