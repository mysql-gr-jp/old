池田様　有り難うございました。おかげさまで解決いたしました。

mysqlimportではYuGo様の記載にも有るとおりエスケープシーケンスとして解釈
してしまう為　不可（ワタシの場合）でしたので、

shell> mysql -u root --password=*** DB名 -e 
"load data infile '/var/lib/mysql/CSVファイル名' 
into table テーブル名 fields terminated by ',' enclosed by '\"'
ignore 1 lines "

として意図する結果を得られました。

> 
> <引用>
> load data infileの場合はこのようなことは発生していません。
> mysql> load data infile './BUSYOPF.CSV' 
> 　　　into table BUSYOPF 
> 　　　fields terminated by ','
> 　　　enclosed by '"' ignore 1 lines;
> </引用>
> 
> これを-eのあとにつけます。最後のセミコロンは不要です。
