> priceが最大のレコードを１件表示すれば良いんですよね。

MLに流したのは、ドキュメントをただ写しただけの例だったので、
本当にやりたかったことはこういうことなんです。

データが下のようにに並んでいて、
# keywordはidごとに無限にあります。
+----+----------+
| ID | keyword  |
+----+----------+
| 1  | AAA      |
| 1  | BBB      |
| 1  | CCC      |
| 2  | DDD  　　|
| 2  | EEE      |
+----+----------+

を、
+----+----------+----------+----------+
| ID | keyword  | keyword  | keyword  |
+----+----------+----------+----------+
| 1  | AAA      | BBB      | CCC      |
| 2  | DDD      | EEE      |          |
+----+----------+----------+----------+
こうして、
select ID from table where keyword = 'AAA' and keyword = 'BBB';

というようなことをやりたかったのです。
この手順を１発でSQLでやりたかったのです。

わかりますでしょうか？
