データベースの構築に取り組んでおります。

さて問題なのですが、いろいろと検索、実験を繰り返し、
my.cnfの設定に関して以下のようにしようと思っています。

[client]
default-character-set=ujis
[mysqld]
old-passwords
default-character-set=ujis
init-connect=SET NAMES binary
[mysql]
default-character-set=utf8

環境
MacOSX Server 10.4.2
MySQL  4.1.10a
PHP  4.3.11

やりたい事は
・XOOPSを動かす。
・AppleScriptからの接続

で、AppleScriptからの接続という問題があるため、[mysql]
だけを今まで検索にかかった解決策であるbinaryからutf8にしよ
うとしているのですが、多分こういう設定にした場合が、検索
に引っかかる、文字の自動変換をしているじょうたいなのだと
思いますが（違うのか？）このとき時折見かける自動変換には
問題があるらしい書き込み。これはどのような問題があるとい
うことなのでしょうか？。
