C API の mysql_row_tell() と mysql_row_seek() は、検索結果がすべてクラ
イアント側にあることが前提なので、大量の結果の中から数件だけ取り出すに
は、効率が悪いと思います。

サーバサイドのカーソル機能があればいいのかもしれませんが…。

> PHPからいろいろ試したのですがうまく使えません。
> そもそもPHPからこれらの関数は使えないのでしょうか？

PHP のことはわかりません (__)
