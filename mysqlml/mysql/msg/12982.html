以下のように文字化けしたエラーメッセージが表示されます。

※mysqlに接続するたびに文字化けする部分は変化します。

本来なら以下のようになるはずだと思うのですが・・・
mysql> use
ERROR:
USE must be followed by a database name

引数に入力していない何かが勝手に入り込んでいるように思えます。

4.0.20〜4.0.27までをインストールし、調査した結果
4.0.20,4.0.21は正常にメッセージがでるのですが、4.0.22以上は文字化けしてメッセージが表示されるようです。


原因、影響等まったく分かりません。
