MySQLはMacintosh OS X用のオープンソースのデータベース
ツールをリリース

2005年6月7日

Visual QueryとAdministrationツールは以下から
ダウンロードできます。 http://dev.mysql.com

サンフランシスコ発：Apple Worldwide Developer Conference
オープンソースのデータベース開発会社のMySQL ABは
今日 Macintosh OS X用の２つのビジュアルなデータベースツール
を発表します。MySQLのデータベース開発者、DBAとユーザの
使い勝手を増加するためにリリースされました。ベータ版
のMySQL Query BrowserとMySQL Administratorツールは
以下からGPLのライセンスで入手できます。 http://dev.mysql.com.

MySQLと新しいMacintoshのツールは来週のサンフランシスコ
で開催されるApple Worldwide Developers Conferenceで
デモされます。

MySQL ABのSoftware Engineering担当副社長のMaurizio Gianola
は「Apple Macintosh はMySQL開発でもっとも人気のある
プラットフォームの１つです。Mac OS X用のオープンソース
のデータベースのクエリとアドミのツールを提供できて嬉しいです。」
と語った。
eBay, AOL Time Warnerと Expediaなどをクライアントとして持つ
LiveWorldのSolutions Development担当副社長の Bruce Dembecki
は「MySQLは私達の負荷の大きいApple Xserveサーバー
で速く、信頼性の高く動作することを証明しました。一番要求の大きい
MACのユーザが期待するくらい設定や管理が簡単です。毎日
我々のディスカッション・ボードではたくさんのヒット
があります。MySQLとMacintoshは問題なくスケール・アウト
します。」と語った。

MySQL Query Browser

The MySQL Query Browserはビジュアルなツール群で、MySQLの
データベースのクエリを生成、実行、最適化をする
ツールです。ウエブのような環境でドラッグ・アンド・ドロップ
の統合ツールです。これがあれば容易で生産的な方法で
MySQLのデータベースに格納された情報にアクセスし解析
することができます。

MySQL Administrator

MySQL AdministratorはDBAが容易にMySQLのアプリを
管理・モニターできる有能なツールです。直感的
なグラフィカルなユーザーインターフェースでDBAはMySQLの
オペレーションをより良く把握できます。これには
サーバーのコンフィギュレーション、ユーザアドミ、
リプリケーションの状態のモニタリング、バックアップ・リストアー
やログの閲覧を含みます。

他のプラットフォーム

Macintoshの他MySQL Query BrowserとMySQL
Administrator はLinuxとWindows上でも動作します。

ダウンロードと詳細は
http://www.mysql.com/products/query-browser/
とhttp://www.mysql.com/products/administratorへ。
