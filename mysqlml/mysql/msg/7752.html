横山と申します。
win2000にmysql 4.0のインストールを行っております。
mysqlの操作になれるためいろいろ設定を変えていたのですが、
把握仕切れなくなったので、再インストールを行いました。

コンパネのアプリの削除からmysqlを削除してもう一度
mysql4.0をインストールしました。
最初にrootのパスワード設定をおこなうため

> mysqld

> mysqladmin -u root password newpassword

> mysql -u root -p
とした後パスワードをnewpasswordとしたら
ERROR 1045: Access denied for user: 'root@xxxxxxxxxx' 
(Using password: YES)

とはねられました。
> mysql -u root 
でパスワードなしおよび
> mysql -u root -p
でパスワード無しでenterを押すとmysqlのプロンプトになります。
rootのパスの設定の仕方が参考にしたサイトによって
多少ことなっていたので以下のものをためしましたがダメでした。

> mysqladmin -u root password 'newpassword'
あるいは
> mysqladmin -u root password "newpassword"

rootのパスワードの設定で私が犯しているミスがどこにあるのか
わからず投稿に至りました。
なにかアドバイスをいただければ幸いです。
