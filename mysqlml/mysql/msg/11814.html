製品
MySQL (5.0.9 Beta)のリリース（但し現在は5.0.10版
のリリースがあります。http://lists.mysql.com/announce/301）

このベータリリースは5.0シリーズのプロダクション前の
リリースです。このリリースではStored Procedure, View,
Trigger, Information Schema (Data Dictionary), XA とその他
多くをサポートします。

追加のアップデートは：

* InnoDB: 種々の最適化. デバッグ版でないコードから
到達不可能なデバックコードを除去。GCCにブランチ・プリディクターへ
のヒントを追加。assertionのスペースを最小化。
* BIT コラムの処理を改善し更に信頼性を増加。
* mysqldump -- xml がNULLコラムを正しく処理
* 種々のバグ処理

全てのエネルギーはバグの修理と安定化に注がれています。

全ての変更リストは
http://lists.mysql.com/announce/298
ダウンロードは
http://dev.mysql.com/downloads/mysql/5.0.html

MySQL マイグレーション・ツールキット(1.0.11)のリリース

MySQL マイグレーション・ツールキットは強力なフレームワーク
で、プロプリエタリーのデータベースをMySQLにマイグレートします。
ツールキットは:

1. JDBCドライバーのあるどのデータベースからもマイグレートする
「一般マイグレーション」
2. MS Access, MS SQL Server とOracleからマイグレートする
「最適化マイグレーション」があります。

このリリースにアップデートなさることをお勧めします。
修理されたバグは：

* MySQL4.0とのコンパチ問題を解決
* 接続が格納されない
* Oracleのリバース・エンジニアリングコードでのエラーを無視
* Procedureから Routineへの名前変更

ダウンロード
http://dev.mysql.com/downloads/migration-toolkit/1.0.html

MySQL コミュニティ版(4.1.13)のリリース

これはMySQL コミュニティ版 のリリースで新機能とバグ修正を
含みます。

* セキュリティの改善: zlibデータ問題を解決するパッチを
適用。この問題はバッファーのオバーフローとコード実行の際
にセキュリティの問題を生じる恐れがあった。
* メモリー・テーブルは500バイトまでのインデクスをサポート
* -log-slow-admin-statements サーバー・オプションを追加
してOPTIMIZE TABLE, ANALYZE TABLEやALTER TABLE などの
slow administrative ステートメントを支援
* EXPLAINの出力のtable, typeとrows columnsはNULLの
値を取れる。

詳細は
http://lists.mysql.com/announce/299
ソースとバイナリーのダウンロードは
http://dev.mysql.com/downloads/mysql/4.1.html

MySQL サーバーをCプログラムに組み込む

MySQLのサーバーのライブラリーをアプリに組み込む際は
MySQLがユーザーのコンピュータにインストールされている
必要はありません。小さな機器やキオスクなどに有益です。組み込み
のMySQL サーバーはアプリと共に速く実行するという利点があります。
というのもネットワークを介して接続することがないからです。
ライブラリーはすでに、コンパイルされたプログラムにリンクされて
組み込まれているからです。組み込み型のサーバーはMySQLの
ディストリビューションの一部です。

記事を読む
http://dev.mysql.com/tech-resources/articles/embedding-mysql-server.html
