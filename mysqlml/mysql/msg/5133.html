VB6.0でMySQLにSQLをおくれれば良いのですか？
VB6のHELPを参考に1プローシジャ作ってみました。

MySQL側には'Test'データベース
UniqueIDと、Memoの２つのフィールドを持つTestTABLE(既にUniqueID:1～4のレコー
ド作成済み)
root名でパスワード無しアクセス可能

Win2k側でODBCデータソースアドミニストレータにて'MYSQLODBC'名にてシステムDSN
でMyODBCを登録済み。

VB側ではフォーム上にCommand1のボタンを配置し、参照設定にて'Microsoft DAO 3.6
Object Library"をチェックして以下を追加
Command1のClickイベントプローシジャ-----------------------------------
Private Sub Command1_Click()
    Dim WS As WorkSpace
    Dim CN As Connection

    Set WS = CreateWorkspace("WS01", "root", "", dbUseODBC)
    Set CN = WS.OpenConnection("Connect01", _
                              dbDriverPrompt, , _

"ODBC;DATABASE=Test;UID=root;PWD=;DSN=MYSQLODBC;OPTION=16835")
    CN.Execute "UPDATE TestTABLE Set Memo='ID1memo' WHERE UniqueID='1'"
End Sub
