>その辺全く分からないのですが、MySQL のバージョンは OS 依存
>なんですか？
>
>つまり、自分で勝手に MySQL の最新バージョンは入れられない？

      技術的にはインストールする知識があればより最新のバージョンにで
    きるでしょう．先のメールは単に「収録されている MySQL のバージョン」
    ですから．ただし Red Hat Enterprise Linux など商用のものをディス
    トリビューター以外で配布された MySQL をインストールした場合は
    MySQL に付随するサポートはディストリビューターからまったく受けられ
    なくなると思います．
