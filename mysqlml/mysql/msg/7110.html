現在Redhat7.3に入れ替えてやっているのですが、
（RedHat8.0は自宅にあるので、そちらでもまた試してみます）
やはり同じ状態になっていまいました。

設定内容ですが、
httpd.confに

LoadModule mysql_auth_module  modules/mod_auth_mysql.so

AddModule mod_auth_mysql.c

の二行を追加して（8.0の場合はAddModuleはありません）

Apacheを起動すると起動する事ができます。

この後にhttp.confの最終行に

Auth_MySQL_Info [ホスト名] [ユーザ名] [パスワード]
Auth_MySQL_General_DB [データベース名]

を入れてApacheを再起動すると

Invalid command 'Auth_MySQL_Info', perhaps mis-spelled or defined by a module no
t included in the server configuration

というエラーメッセージが出ます。

それ以外に今のところ設定しているところはないと思います。
基本的にはRedHat8.0の時もほぼ同じ設定です。
