藤森です。

最近、クリーンインストールした Windows 98 で、
今までうまくいっていた Access 2000 からの
エクポートがエラーになってしまいました。(サーバは Linux)
色々調べたところ、Microsoft Jet の問題らしく
SP3 -> SP5 に変更してうまくいきましたので、 
必要なものをまとめてみました。

1. MDAC 2.6 RTM (2.60.6526.3)
 http://www.microsoft.com/data/
2. Microsoft Jet SP5
 http://support.microsoft.com/support/kb/articles/q239/1/14.asp
3. MyODBC 2.50.36 for Windows95/98 (or for NT)
 http://www.mysql.com/downloads/api-myodbc.html
4. winclients+myodbc+com-conv_ujis-20001204
 http://www.softagency.co.jp/mysql/Win/myodbc.html

# 3,4 は myodbc-2.50.29-jp でも大丈夫かもしれません。(未確認)
