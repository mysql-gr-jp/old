Subject: [mysql 04488] ログファイルへの吐き出し処理について

はじめまして、MYSQLを使ってバッチファイルのログファイルを吐き出す部分で悩ん
でおります。
バッチに以下のように記します。
mysql aaa < del_file.sql >> log.out
処理を実行させそれをログファイルに書き込む処理を入れたaaaには、次のように記
してあります。
tee log.out
SELECT NOW();
DELETE FROM customerinfo WHERE extractFlg='1';
SELECT NOW();
notee
quit
上記のしょりが終わったあとのログは、
NOW()
2001-10-16 15:19:44
NOW()
2001-10-16 15:19:44
Outfile disabled.
になりDELETE文の処理をうまく吐き出すことができません。
DELETE文を直接MYSQLに打つと、
例えば、Query OK, 2 rows affected (0.00 sec)なメッセージがでます。これをうま
くlog.outのなかに吐き出すようにしたいのですが、どうしたらいいでしょうか？
アドバイスをよろしくお願いします。
