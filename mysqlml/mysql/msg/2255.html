２日前に同じような質問をしたばかりの初心者の服部です。

僕の場合psコマンドでは表示されてませんが、しっかりサーバーは作動していま
す。
MySQLサーバーの立ち上げからやり直してみればどうでしょうか？
% mysqld -u mysqladm  (自分でコンパイルした場合）
か
% mysqld -u mysql （RPMを使った場合）　でサーバーを起動します。

もし、すでにサーバーは起動されてるんとちゃう？みたいなメッセージが返って
きたら、クライアントを起動します。
% mysql -u mysqladm      (自分でコンパイルした場合）
か
% mysql -u mysql         (RPMを使った場合）

上のが動かないときはルートで試してください。

% mysql -u root		(rootにまだパスワードを与えていない場合）
か
% mysql -u root -p	（すでにrootにパスワードを与えた場合）

MySQLサーバーをインストールした後にmysql_install_dbというコマンドを打った
ことがありますか？　これはデータベースを初期化するためのものです。
（どっかのバージョンから、別にしなくてもよいとも聞いたのですが・・・試し
て損はないでしょう。でも、これを実行するのはインストールした後に一回だけ
でいいです。）
