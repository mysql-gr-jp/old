Subject: [mysql 11485] Dドライブにデータ領域が変更できない

はじめて、メーリングリストにメールします。
取引先サーバーの設定ができなくて困っています。

WindowsServer２００３StandardEditionに
Mysql4.0.18－Winをインストールしました。

データ領域をDドライブに変更したくて下記の作業をしました。
参考ページは
http://dev.mysql.com/doc/mysql/ja/windows-prepare-environment.html
です。

ｍｙｓｑｌをインストール
C:\mysql\dataを削除
D:\mydataに削除したｄａｔａフォルダをコピー
mi.iniをC:\WINDOWSの直下に作成、テキストは以下のとおり

[WinMySQLAdmin]
Server=C:/mysql/bin/mysqld-nt.exe

[mysqld]
# set basedir to your installation path
basedir=C:/mysql
# set datadir to the location of your data directory
datadir=D:/mydata/data

user=max
password=erp

[mysqld]
default-character-set=sjis

[mysql]
default-character-set=sjis

[mysql-dump]
default-character-set=sjis

winmysqladmin.exeを起動すると、赤信号です。
winmysqladminの中は

StartCheckのページには
there is a mi.ini file      yes
Veryfying the my ini keys
MySQL Server path key        OK
datadir                               OK
basedir                              OK

Serverページ、Variablesページは真白です。

何が、いけないのでしょうか。一旦アンインストールしてレジストリーも
クリアして再びインストールしても結果は同じでした。

どなたか、アドバイスをお願いいたします。
