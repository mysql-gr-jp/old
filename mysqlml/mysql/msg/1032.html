  やったことは以下の通りです。

  1)ActivePerl 5.003_003(Api518e.exe)をインストール
  2)DBI.zipを解凍し, ppm install DBI.ppd でインストール

  4)テストスクリプトを動作させる。→ 正常に検索結果が得られた。
  5)japerl518_bin_diff.zipを解凍しperlのインストールディレクトリに解凍し
    てできたbinとlibをコピーする。
  6)これでできたJperlで4)項のスクリプトを動作させる。
    → parse exception エラーとなる。
