文字通り、OS の readline ライブラリの代わりに mysql の readline ライブラ
リを使う/使わない、という選択かと思います。

しかし、気になるのは、CentOS 4.4 という事ですが

>configure: error: Could not find system readline or libedit libraries

/usr/lib/libreadline.a 
/usr/lib/libreadline.so.*

あたりが見付からないでしょうか。

もし、これがあって、且つ configure がそれを見付けられないとすると、ひょっ
とすると /usr/lib が LD_LIBRARY_PATH に入っていない、という可能性があるかも。
