>   取り敢えず、VineLINUX2.1(インストール時にapache, php, mysql等は、

> 
>   そこで質問ですが、apache, php, mysqlのインストールや設定の順番が、
> ありますか?
はい
mysql をコンパイルして出来上がるライブラリがないと
php は　mysql モジュールを作りません
apache のDSO　を使うのであれば　apache を最初にコンパイルする必要
があります


ということで
apache DSOタイプなら
１）mysql のコンパイル
２）apacheのコンパイル
３）php のコンパイル
　　　./configure --with-mysql \
       --with-apxs=/usr/local/apache/bin/apxs 　他のオプション

apache に直接phpを組み込むなら
１）mysql のコンパイル
２）apacheとphpを　php のソースディレクトリ上から
　　いっしょにコンパイル
    ./configure --with-mysql \
      --with-apache=../apache_1.3.x　他のオプション

って感じになります

  もうひとつは php　をcgi モードとして　というのもありますね
　これも　mysql が最初になります
