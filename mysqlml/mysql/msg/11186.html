バージョン3.23.58では副問い合わせはできないようです。
「Starting with MySQL Version 4.0.4, you can also perform UPDATE operations that cover multiple tables:」

バージョン4.022では以下のSQLで実行出来ます。
UPDATE テーブル１ 
 LEFT JOIN テーブル２
 ON テーブル１.ID = テーブル２.ID
 SET テーブル１.商品名 = テーブル２.商品名
