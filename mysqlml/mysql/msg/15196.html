坂井です

　今月18,19日に開催されたデブサミ(http://codezine.jp/devsumi/2010)で
　来場くださった皆さん、ブース番を交代で引き受けてくださった皆さん、
　どうもありがとうございました。

　立て続けのイベントではありますが（しかも直前の案内になっちゃってご
　めんなさい）、今週末にはオープンソースカンファレンス東京(2010春)に
　も日本MySQLユーザ会(MyNA)として参画します。

  =============================================================
        オープンソースカンファレンス(OSC)2010 東京・春

日程：2010年2月26日(金)・27日(土) 10:00-17:00
     (26日の展示は11:00から、27日の展示は16:00まで）

会場：明星大学 日野キャンパス 26号館
    （多摩モノレール 「中央大学・明星大学駅」から大学まで直結）

詳細URL：  http://www.ospn.jp/osc2010-spring/
  =============================================================

　MySQL 関係のセッションとしては２つあり、
　ひとつめが、
　OSCのスポンサーにもなっているサンマイクロシステムズから、
　(先日大阪のセミナーでもお話をした）梶山さんが、
　「MySQLより愛を込めて」という意味深なタイトルで講演を行います。
　MySQLの【最新情報】を聞かせていただけるようです。
　明日(26日・金)の朝11:15からの枠で、まだ空席はあるようです。

　ふたつめはMyNA枠として木村明治さんが MySQL の GUI ツールである
　MySQL Workbench についての最新情報＋αを聞かせてくれる予定です。
　これも明日、14:00から。お席にまだ余裕あるようです。
　
　また、データベースコミュニティとしては、Firebirdユーザ会のはやし
　さんが、土曜日13:00～の枠で MySQL, postgreSQL と Firebird との
　パフォーマンス比較のお話をされますので、この中でMySQLのパフォー
　マンス測定結果のお話も聞けるかもしれません。
　
　
　セミナー以外でもいつもの通り２日間とも、まったりとブースを開けて
　いますので、MySQL談義や情報交換などをしましょう。
　お時間ある方はぜひブースの「こちら側」にも入ってきてくれると嬉し
　いなぁと思います。経験年齢不問(条件：MySQLを(少しは)好きという
　こと)なので、やってみたいな という方はぜひ現地で声をかけてください。
　ブースにいるだけの簡単なおしごとです！:-)
　いろんな人といっぱいお話できて、何かがきっと詳しくなります。
　いまがチャンス！（なんのチャンスがよく分からないけど(^^;)）


　では現地でお会いできる皆様、よろしくお願いします。



thank you
