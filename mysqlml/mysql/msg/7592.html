>ShowのようなRecordSet以外の回答を持つSQLコマンドの格納領域は
>どこになるのでしょうか、

Showも取得方法としてはRecordSetで扱えば取得できます。

ADOを使用した場合の例なんですが、下記のようになるかと。

Dim adoRd  
Dim s 
Set adoRd = CreateObject("ADODB.Recordset")
adoRd.Open "Show tables", "Provider=MSDASQL.1;Persist Security Info=False;Data Source=myodbc3-test"
Do Until adoRd.EOF
	s = s + vbCrLf + adoRd.Fields("Tables_in_test").Value
	adoRd.MoveNext
Loop
MsgBox s
adoRd.Close
set adoRd = nothing
