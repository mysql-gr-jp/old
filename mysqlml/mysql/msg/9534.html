[Oniguruma のライセンスについて]

Oniguruma のライセンスは BSD ライセンスです。

- Oniguruma
  http://www.geocities.jp/kosako1/oniguruma/

MySQL は GPL なので、
MySQL に Onigurua を含めて再配布する場合のライセンスについては
MySQL に GPL、Oniguruma に BSD が適用されるはずです。(えんどう私見)

[その他]

Oniguruma の作者 K.Kosato 氏には、実験の概要と上記 URL を連絡済みです。
再配布については特に K.Kosato 氏に確認していません。
