Subject: [mysql 09790] mysqldumpの出力結果を取り込もうとするとUnknwon command

オハシと申します。

MySQL4.0.18/RedHatAS2.1の環境を使用しています。

あるデータベース(dbA)を丸ごと違う名前のデータベース(dbB)として使用したい
ので以下を実行しました。
#innodb　なのでmysqlhotcopyは使用できません

1. mysqldump --opt dbA >./dbA.dump
2. createdatabase dbB
3. mysql dbB < ./dbA.dump

すると
Error at line 89: Unknown command '\' .
というエラーが出力します。おそらく文字コードに絡む問題かと思うのですが。
/etc/my.cnfには
default-character-set = sjis
となっており、ダンプのファイルはSJISで問題なく読めてます。
皆さんのお力を貸してください。
