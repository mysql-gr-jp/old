平松＠福岡支部賛同者募集中、です。

今回の趣旨をご説明いたします。

背景としては、そもそもMyNA発足会のときに、東京までいけないな～
と指をくわえてみていた過去がありまして、たまたまPostgresが地方
支部を作っているという話がありましたので、それならいっちょ福岡
でもやってみるべ～、とおもいたったということです。

趣旨としては、きわめてストレートに、
MyNAの紹介ページ(http://www.mysql.gr.jp/のAbout MyNA)にあります、
--------------------------------------
●目的
日本での MySQL の普及を図る。 
ユーザ間のコミュニケーションを図る。 
MySQL の日本語化の検証/開発を行なう。 

●活動
現在の主な活動は ML での意見交換です。時々オフ会もあるかもしれませ ん :-) 

●入会
MySQL に興味がある方はどなたでも入会できます。会費はありませんし、 退会も自由です。 
--------------------------------------
を福岡で盛り上げようということです。

MLでできることはローカルにやる必要ありませんので、

> 日本での MySQL の普及を図る。 →福岡に読み替えてください
> ユーザ間のコミュニケーションを図る。 

> 時々オフ会もあるかもしれませ ん :-) 

あたりがメインの支部活動になるのではと思います。
別に「支部」でなくたっていいと思ってます。
ただ、Postgresが支部なるものを作っているなら、
作ったほうがいいのかなあ、という発想。

そこらへんも含めて、ご賛同される方中心に、
どうしたいか話し合っていければと思います。
