佐々木です。
phpMyAdmin の部分を補足します。

<4.1 未満>
Language :
  phpMyAdmin の表示用 charset です。
  select_lang.lib.php の $available_languages 値のリストです。
  ブラウザ情報や cookie から初期選択値を決定します。
  何れも不明のとき config.inc.php の DefaultCharset 値を採用します。

<4.1 以上>
Language :
  同じ。但し私の環境では DefaultCharset を指定しても utf-8 に戻されます。
MySQLの文字セット :
  SHOW CHARACTER SET コマンドで得られる Description 列値 (Charset 列値) 
  Language に連動しています。
MySQL 接続照会順序 :
  SHOW CHARACTER SET コマンドで得られる Default collation 列値 のリストです。
  cookie から初期選択値を決定します。
  この値は phpMyAdmin が mysql server へ接続するときに、
  SET session collation_connection を発行する際用いられます。
  詳しくは下記を参照してください。
  http://dev.mysql.com/doc/mysql/ja/charset-connection.html


正直、そこまで気になるなら直接プログラムソースを読む努力をされた方が良いと思いますが・・・。
また、phpinfo() で返される mysql モジュールの値も確認した方が良いでしょう。
