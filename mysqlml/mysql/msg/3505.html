山本です。
> +----+----------+
> | ID | keyword  |
> +----+----------+
> | 1  | AAA      |
> | 1  | BBB      |
> | 1  | CCC      |
> | 2  | DDD  　　|
> | 2  | EEE      |
> +----+----------+
このテーブルを適当に作って、
CREATE TABLE test (
	ID INT,
	Keyword CHAR(3)
);

同じテーブルをJOINさせればできますが、こういうことでしょうか？

SELECT a.ID FROM test a, test b 
WHERE a.ID = b.ID 
AND a.Keyword = 'AAA' 
AND b.Keyword  = 'BBB';

検索条件が増えれば、その分JOINさせるテーブルを増やして、
SELECT a.ID FROM test a, test b, test c 
WHERE a.ID = b.ID 
AND   a.ID = c.ID 
AND a.Keyword = 'AAA' 
AND b.Keyword  = 'BBB' 
AND c.Keyword  = 'CCC';
という風にするといいのかな。
