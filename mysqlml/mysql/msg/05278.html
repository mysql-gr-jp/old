> 
> あれ、mysqldumpとかのデータを見ると付いてますよ。。

　　それは SQL 形式だからですね。mysqldump が出力時につけているもので、
　DB の中に入っているわけではないです。mysql フロントエンドで select ... 
　して表示 *されなければ*、DB の中には入っていません。で、

　・PHP 等で SQL として渡すときはエスケープ
　・LOAD DATA INFILE で生データを渡すときはそのまま

　どちらでも DB の中に入るものは同じはずなのです。
