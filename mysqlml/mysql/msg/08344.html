はじめて投稿します
サカイといいます。

OpenOffice.orgに接続して使用すべく、約1ヶ月前から
暇を見つけては、MySQLと格闘しています。

データベースを作成し、テーブルを作り、
mysql> LOAD DATA INFILE "/home/myname/textfile.txt" INTO TABLE tablename;
とすると、
The used command is not allowed with this MySQL version
と出ます。

Web検索して、
サーバーとクライアントの起動時に　--load-infile=1 を加えろとのことでしたので、

# ./mysqld_safe --load-infile=1  &
と
$ ./mysql -u myname -p --load-infile=1
と実行しましたのち、
mysql> LOAD DATA INFILE "/home/myname/textfile.txt" INTO TABLE tablename;
としましたが、再び
The used ccommand is not allowed with this MySQL version
と出てしまいます。

何か根本的に間違っているのでしょうか?
アドバイスやヒントをいただければありがたいのですが。

OS : Vine Linux 2.6 upgraded
MySQL version : Server version  4.0.15a-log

です。
よろしくお願いします。
