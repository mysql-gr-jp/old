1.データの移し方
　MySQL に限らず、普通は次の手順で行います。
　(1) 索引等を含まない空テーブルを作る。
　(2) データをインポートする。(文字エンコードの変換を含む)
　(3) 索引等の作成する。
　PostgreSQL (MySQL含む) なんかだと全 SQL 文を自動生成させて、
　コマンドラインから一括で実行させることも多いですが。

2.レコードの並び
　まずは mysql などのコマンドラインツール上で SQL を発行して
　実際の抽出結果を確認します。
　phpMyAdmin しか使えないなら、同じ SQL を作成して実行させます。

まずはデータ移設と phpMyAdmin の動作を切り離して考えてください。
MySQL は SQL ベースのデータベースですから、彼我のバージョン相違による
影響を考えた上で、移設するための適切な SQL を作成することをまずは
目的にしてください。
