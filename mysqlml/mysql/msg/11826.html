(1) その MySQL Server では web サーバは動いてますか？
→Yes(4)  →No(2)

(2) その MySQL Server ではメール送受信が出来ますか？
→Yes(4)  →No(3)

(3) その MySQL Server ではリモートログイン出来ますか？
→Yes(4)  →No(8)

(4) telnet 192.168.0.100 を実行してみる
→なんか出た(6)  →何も出ない(7)

(5) telnet 192.168.0.100 80 を実行してみる
→なんか出た(6)  →何も出ない(7)

(6) telnet 192.168.0.100 3306 を実行してみる
→なんかでた(9)   →何も出ない(10)

(7) WindowsXP 側でファイアウォールでポートが閉じられて
　　いる可能性があります。とりあえず、セキュリティコン
　　トロールセンターでファイアウォールを無効にするか、
　　ウィルスソフトのセキュリティを無効にして(1)からやり
　　直してください。
(8) まず、サーバとの接続を確認してください。ping 等を使
　　用し、サーバにパケットが届いているか確認してください。
(9) この状態でかねやんが接続出来ないならかねやんの設定
　　に問題があるでしょう。設定等を確認してください。
(10)サーバ側で不必要なポートを ipfw 等で塞いでいませんか？
　　サーバ側のファイアウォール等を確認してください。
