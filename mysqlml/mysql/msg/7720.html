インデックスマージというものがあるのですね。

これを見ると

SELECT DATE_FORMAT(MAX(OUTYMD), '%Y/%m/%d%H:%i:%s') OUTYMD
FROM OUTINF
WHERE PRJNAME    = 'prjname'
  AND SVNAME     = 'svname'
  AND ITEM    LIKE 'item_name%';

に対して作成した

CREATE INDEX IDX_OUTINF ON OUTINF ( prjname, svname, item,
outymd ) ;

は（たぶん）結合インデックスに該当することになるのだと思
います。
他のSELECT文を確認したところ、インデックスを変更しても問
題がないようなのでインデックスの再作成にて対応したいと思
います。
