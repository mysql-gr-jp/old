MySQL 4.1.21でテストしてみました。

create table t1 (
	STR   varchar(100)
)；

insert into t1 values ('１．第１号')；
insert into t1 values ('２．第５２号')；
insert into t1 values ('３．第４５６号')；
insert into t1 values ('２．第４３号')；

とやっておきまして、

mysql> select * from t1 order by length(STR),STR;
+----------------+
| STR            |
+----------------+
| １．第１号     |
| ２．第４３号   |
| ２．第５２号   |
| ３．第４５６号 |
+----------------+
4 rows in set (0.02 sec)

と、よさげに見えます。
