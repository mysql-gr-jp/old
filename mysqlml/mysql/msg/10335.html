MySQL Administratorは強力なビジュアル管理端末でMySQLの環境管理を
容易にし、データベースの運営状況を的確に入手できます。MySQL Administrator
はデータベース管理と保守を１つの途切れのない環境ではっきりと
直観的なグラフィクス・インターフェースで提供します。

MySQL Administrator を使うことで

    * 管理改善で更に高度なデータベースのアベイラビリティを達成
    * ビジュアルなデータベース管理でエラーの削減
    * プロダクティビティ改善でデータベース管理費の削減
    * より容易な特権管理でよりセキュアーな環境を提供

MySQL Administrator は開発者と DBAがサーバーのコンフィギュレーション、
ユーザ管理、ダイナミックにデータベースの状況を監視すること
を含めて、コマンドラインのオペレーションを実行できます。
他の管理はリプリケーション監視、バックアップとリストアー
とログの監視などを含みますが、MySQL Administratorのグラフィックス
の端末で容易に行えます。

「MySQL Administrator は非常に直感的なビジュアルなツールでMySQLの管理

行います。この２つは長期にわたって大きなデータベースの構造の
管理に必須です。性能を上げるためのサーバーの最適化とチューニング

ユーザ管理

ユーザ管理と特権を与えることとユーザの特権情報見ることは
非常に簡単になりました。ユーザ・特権管理の機能を使って
統合グラフィカルユーザ管理端末でMySQLサーバーのホストを守ります。
例えば、フォームベースのインタフェースではユーザの追加、パスワード
の設定、ユーザ・プロファイル情報の簡単な方法を提供します。
グローバル、データベースまたはテーブル・コラムレベルの特権
を割当・撤回するのは入手できるリストから特権の割当・撤回
をするくらい簡単です。最後に特権タイプによって簡単に
階層的にユーザ特権を見ることができます。

一箇所でのダイナミックな状態監視

ダイナミックな状態監視でMySQLの環境の状態を示す情報が提供
されます。システムの作動中に、ダイナミックなグラフで
メモリーの使用状態、接続状況、ヒット率、SQLのクエリの数
などを監視できます。「Disk Health」タブからディスク・スペース
テーブルスペースやログファイルの概略を得ることができます。
ビジュアル状態監視機能で状態や接続の数、クライアントから送受信された
バイト数、バッファーの割り当てサイズなどのサーバ変数の階層的
なビューを提供します。

MySQLを迅速に最適化

MySQL Administratorは重要なパラメーターを見たりセットすること
ができます。これでMySQLの性能を最適化できます。グラフィックス
のインターフェースで、フォーム・ベースの環境を使って、普通の
パラメータと高度なパラメータを細かく調整することができます。
例えば、メモリー使用パラメーターをセットでき、MyISAM とInnoDBの
テーブル・タイプに関するサーバー・スタートアップ・パラメータを
設定できます。

迅速なディザスター・防止と回復

バックアップと復旧を行うには単に画面をみてスキーマとテーブルを
階層的に示されたデータのディレクトリから選択するだけです。グラフィクス
の環境はTabletype, Rows, Data Length, Update time　などを
監視するのが更に容易になります。

一目でサーバー情報が得られる

MySQL Administratorのサーバー情報のセクションはサーバーとクライアント
の基本的な情報を提供します。サーバーの状態、クライアント・サーバの
バージョン番号とIPアドレスや接続情報が一目で分かります。

リプレイケーション状態の監視

簡単にリプリケーションをセットでき、MySQL環境のスピード
を増加します。MySQL Administratorのリプリケーションの機能は
リプリケーションのマスターとスレーブのネットワーク・トポロジ
を表示することです。グラフィックスの環境を使えばIPアドレスやログファイル
の情報も容易に得ることができます。

集中管理環境からサーバログを見る

エラーログ、バイナリーログ、InnoDBログなど
のログファイルを集中管理環境より見ることができます。これで、
データベースサーバーの診断をして問題や変化ををすばやく補足できます。

「MySQL Administratorには感心しました。簡単だし包括的なツールで

サービスのコントロール

サービス生成を完全にコントロールして、サービスのインストレーション
や設定の時間を削減しましょう。MySQLサーバーを開始し、また止めて
必要なログを見ましょう。

カタログの閲覧

カタログを閲覧して、対応するテーブルのコラム、インデクス、そのテーブル
へのユーザの特権を見ましょう。テーブルを監視して最適化できます。

複数のプラットフォームサポート

MySQL AdministratoはMicrosoft Windows と Linux上で作動し、
MySQL version 4.0 またはそれ以後の版に使用できます。
