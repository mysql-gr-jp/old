黒騎士 wrote:
> Apache(又はIIS)をインストールしないとMySQLは使用できないのでしょうか？
使用できます。というか、MySQLはデータベースサーバであって、
Apache(or IIS)はWebサーバですので、連携させることもできますが、
それぞれ独立して使用できます。

> WindowsのDドライブにMySQLをインストールし、
> mysqladminでパスワードを設定しようとしたら、
> mysqladmin: connect to server at 'localhost' failed
> error: 'Can't connect to MySQL server on 'localhost' (10061)'
> Check that mysqld is running on localhost and that the port is 3306.
> You can check this by doing 'telnet localhost 3306'
> と表示されました。
> 接続できないとのエラー内容なのはわかるのですが、

エラーメッセージをもう少し読み直してみましょう。

> Check that mysqld is running on localhost and that the port is 3306.
mysqldがlocalhost(自分のマシン)の3306番のポートで動作しているのか
チェックしてみてください。

> You can check this by doing 'telnet localhost 3306'
コマンドプロンプトから telnet localhost 3306 とすることでチェック
できますよ。

ということです。たぶん、MySQLをインストールはしたものの実行して
いないのではないでしょうか？
