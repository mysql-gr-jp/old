"mysqldump"はSQLコマンド(Selectとか、Insertとか)ではありません。
ので、mysql_queryで投げてもエラーになります。

これは、対話型で
mysql -uなんとか -p
Password:******
で入った後に
SQL> mysqldump …
とやるのと同等です。


うちのサーバではダンプを取るのに
以下のようなやり方をしています。
ご参考になれば幸いです。

<?
exec("/usr/local/bin/mysqldump -h localhost -u なんとか -pパスワード 
--add-drop-table DB名 > テキスト.sql",$ret1);
?>
