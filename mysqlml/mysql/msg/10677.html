MySQL AB は今日2004年はセールス、パートナーシップ
技術とのどの分野でも全てにわたり記録破りの年であったと発表しました。
今月末までにMySQL のデータベース・サーバーは千万件（今年だけで）の
ダウンロードがあり、世界でもっとも人気のあるオープン
ソースのデータベースであるという地位を揺るがないものに
するでしょう。

今年は何千という会社がMySQLを選択してウエブ、重要なビジネスアプリと
その他のソフトウエアの根幹にしました。これは性能、信頼性、
簡単な展開、大規模な経費節減の理由によるものです。

MySQL ABのCEO のMarten Mickosは「MySQLが成功しているのは
世界中で企業やその他の組織でオープンソースが頻繁に使用
されるようになったことを繁栄しています。我々がエンタープライズ用の
ソフトとサービスベンダーとして成長しているのは
我々の開発者、ユーザコミュニティと企業パートナーの貴重な

に過ぎません。2005年にはもっとデータベース市場に貢献
するつもりです。」と語りました。

ために改善しました。MySQLの性能は光のように速いです。」

技術のマイルストーン

MySQLは今年幾つもの重要な製品と技術を達成しました。

    * MySQL 4.1 ? このアップグレードはサーバーにより高度なクエリ
の機能を与えます。これは以下の機能によって支えられています。
サブクエリ、より速くセキュアーなクライアント・サーバー通信、
新しいインストレーション・コンフィギュレーションツール、そして
各国や地域別の文字セットのサポートなどです。
    * MySQL ビジュアルツール ? 先週紹介されましたが、MySQLは
使い勝手のよいビジュアルなデータベースのツールを出荷することで
市場を拡張しました。
    * MySQL クラスター ? 業界初のエンタープライズ用の
データベース・クラスター技術で停止するわけには行かない
アプリケーションに使用します。 MySQL クラスターはMySQLと
クラスター技術を結合し、ミッションクリティカルなデーターベース
に99.999%の信頼性を保障します。
    * MaxDB ? SAP認定のオープンソースデーターベースはMySQLの
製品の１つとして追加されました。SAPお客さまのおよびSAPのお客さま
にライセンス、サポート、トレーニング、およびコンサルティング
サービスを提供します。今年、MySQLはネイティブのPHPとPerl
をMaxDB用にリリースして、アプリケーション開発者に、機能強化、
性能と恩恵を提供しました。
