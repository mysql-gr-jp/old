MySQL クエリ・ブラウザー1.1.0ガンマと MySQL Administrator 1.0.14 が
ウインドウズとLinux版用にリリースされました。
 
これはクエリ・ブラウザー（グラフィックスのクエリ・シェルとSQLスクリプト・エディタ）
の最初のリリースです。機能は:
 
- クエリを行い結果をエディットする。
- 結果を CSV, HTML と XMLにエクスポートする。
- SQL スクリプト・エディタをハイライトするシンタックス;
- クエリ・ブックマークとヒストリで、容易によく使用されるクエリ
を入手
- クエリの結果を並べて比較
- マスター詳細クエリを作成。選択された行からの値を使って、
１つの結果の行を選択しセカンダリのクエリをアップデート
- それ以外にも多数
 
このリリースは多くの改善を含みます。一番注目されるのは
MySQL5.0版の機能です。例えば、 Stored Procedure、most notably
the support for MySQL 5 features, such as Stored Procedures, Function
と Viewsです。
 
- 複数の結果を返還するstored proceduresへのサポート
- Stored Procedures, Functions と Viewsを作成してエディトするサポート
- 多くの UI の改良
 

MySQL AdministratorはMySQL用のGUI管理コンソールで、ユーザ管理やMySQLの
コンフィギュレーション、テーブルのエディットなどができます。
 
詳細は
http://www.mysql.com/products/administrator/
 
この版はバグを修正し幾つかの改善を含みます。
 
どちらのツールもGPLライセンスで無料で入手でき、ウインドウズかLinux用
のバイナリーかソースかのどちらかの形式でも存在します。
ダウンロードは
http://dev.mysql.com/downloads/administrator/
 
と
 
http://dev.mysql.com/downloads/query-browser/
 

全てのミラーサイトがアップデートされているとは限りません。
