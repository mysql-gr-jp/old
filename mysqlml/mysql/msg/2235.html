　いいえ、直接 bad_query.hh はインクルードしていません。

#include <sqlplus.hh>
　のところからでエラーが検出されます。

　ソースを追いかけてみたら、sqlplus.hh -> result1.hh -> bad_query.hh でインク
ルードしていました。

　次のことをやってみました。

# cp sqlplus.hh sqlplus.cc
# g++ -M sqlplus.cc -I. -I../mysql
sqlplus.o: sqlplus.cc defs ../mysql/mysql.h /usr/include/sys/types.h \
 /usr/include/features.h /usr/include/sys/cdefs.h \
（省略）
 /usr/include/g++-2/map.h /usr/include/g++-2/multiset.h bad_query.hh \
（省略）

　exsample にあるソースで、BadQueryなる例外クラスを使用している（bad_query.hh
は直接インクルードしていない）のを見ると、必要なファイルなのかなと思うのです
が実際のところどうなのだろう・・・
