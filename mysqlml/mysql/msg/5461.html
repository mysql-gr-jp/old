>WinXpにmysqlサーバをインストールした、ということですよね。
>そして、そこで何をしたらどこでどう文字化けが起きたのですか？
>
>例えば、そのWinXpのコマンドプロンプトでmysqlクライアントを
>実行し、create database → create table → insert → selectと

http://www.mysql.com/　からリンクをたどり、
１）MySQL 3.23 -- stable release (recommended) 
 ・・・MySQL 3.23.49 Windows 95/98/NT/2000/XP (12M) 
２）MySQL Max 4.0 -- alpha release
 ・・・MySQL 4.0.1-alpha Windows 95/98/NT/2000/XP (17.2M) 
を落とし、SETUPを実行しました。

DOS窓から、mysqlで、
　mysql>create table test
　(
　a1 int,
　a2 text
　);
　mysql>insert into test
　(a1,a2)
　values('1','表示')
　;
　mysql>select * from test
　+------+------+
　| a1   | a2   |
　+------+------+
　|    1 | 侮ヲ  |
　+------+------+
　1 row in set (0.00 sec)

となります。１）、２）とも同じ現象。（「侮ヲ」はSJIS関連の不具合では
毎度おなじみ）

なお、
 　「My.ini」を修正し「sjis.conf」の登録
 　　　http://homepage1.nifty.com/yito/anhttpd/faq/php/sjis.conf.txt
 　　　をc:\mysql\share\charsets に置く。
でこの現象は解消します！
