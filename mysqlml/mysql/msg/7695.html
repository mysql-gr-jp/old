上岡です。rootもどりました！

停止がわからなかったので、
サーバー自体を再起動して、

# ./bin/safe_mysqld --skip-grant-tables --skip-networking --user=mysql &
でMySQLを起動して、
# ./bin/mysql --user=ユーザー名 mysql
mysql> grant all on *.* to root@localhost identified by "パスワード";
だと、
mysql> ERROR 1047: Unknown command
でしたので、
mysql> grant all on *.* to root@localhost;
Ok, 0 rows affected (0.05sec)
mysql> flush privileges;
Ok, 0 rows affected (0.01sec)

で、rootが復活しました。以前のパスワードのまま使えるようです。

とみたさん、織田さん。ありがとうございました！
