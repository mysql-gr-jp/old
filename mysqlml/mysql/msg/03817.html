実は、特許のデータを実験データとしていれました。
全部で390万件です。

以下のSQLは50秒ぐらい掛かります。

select count(*) from 公報書誌 where
(
 ((公報書誌.公開日又は公表日 >= '1998/05/01') and 
　(公報書誌.公開日又は公表日 <= '1999/12/31')) 
 or 
 ((公報書誌.登録日 >= '1999/01/01') and 
　(公報書誌.登録日 <= '1999/12/31'))
)

もちろん、それぞれのフィールドにIndexをつけています。
公開日又は公表日のみなら62万件で、2．2秒。
登録日のみなら14万件0.43秒。
OR条件で50秒はちょっとどうにかしたいです。
