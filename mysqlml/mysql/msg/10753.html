【原因】
MySQL3.28.58-1は、LONG PASSWORDを(たぶん)使用しているため、
SHORT PASSWORDに変換する。

【対処方法】
mysql> UPDATE user SET Password = OLD_PASSWORD('mypass')
    -> WHERE Host = 'some_host' AND User = 'some_user';
mysql> FLUSH PRIVILEGES;
