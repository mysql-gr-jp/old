こんにちは。Mroongaプロジェクトの林です。

Mroonga 9.05をリリースしました！

リリースアナウンス:
   http://mroonga.org/ja/blog/2019/07/30/mroonga-9.05.html

MroongaはMySQLで日本語全文検索を実現するストレージエンジンです。高速で
あることや位置情報検索をサポートしていることなどが特徴です。詳細につい
はドキュメントをご覧ください。

  * Mroongaの特徴 ― Mroonga v9.05 documentation
    http://mroonga.org/ja/docs/characteristic.html#what-is-mroonga

変更点:
   http://mroonga.org/ja/docs/news.html#release-9.05

さて、今回の主な変更点は次のとおりです。

  * MySQL 5.6.45, 5.7.27 and 8.0.17(制限あり)対応パッケージの提供
  * ユニークインデックスを更新する際の不具合を修正

すでにユニークインデックスを作っている人はテーブルを作り直す必要があります。
テーブルを作り直すには、ダンプ・リストアするか、以下の手順を実行してください。

  ALTER TABLE (ユニークインデックスを作ったテーブル) DISABLE KEYS;
  ALTER TABLE (ユニークインデックスを作ったテーブル) FORCE;
  ALTER TABLE (ユニークインデックスを作ったテーブル) ENABLE KEYS;

それでは、Mroongaでガンガン検索してください！
