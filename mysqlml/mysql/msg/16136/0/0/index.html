-----------------------------------------------------------------------

「オープンソースカンファレンス2014 .Enterprise＠Osaka」開催にあたり

開催にあたり
オープンソースカンファレンスは、企業システムにおける急速なOSS利用の拡がり
を受けて、データベースをテーマにした「OSC.DB」、クラウドをテーマにした
「OSC.Cloud」を開催してきました。
これら2つを統合した「OSC.Enterprise」は、企業システム全体でのOSS活用を
テーマにしており、2013年12月に東京で行いました。

「OSC.Enterprise」第二弾として、2014年9月に大阪での開催が決定しました。
企業所属の方にお仕事としてご参加いただきやすい、平日金曜日、JR大阪駅から
直結「グランフロント大阪」内で開催です。

出展者と直接対話できるエンタープライズOSSの情報収集の場として、ぜひご活用ください。
