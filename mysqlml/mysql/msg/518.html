Subject: [mysql 518] Japanese charset


データベース及びMySQL超初心者です。

MySQLでは日本語が使えると聞いております。そこでFreeBSDのPortsを
利用してインストールしてみました。もちろんインストール時には
--with-charset=ujis を指定しております。しかしインストール後、
日本語の文字列が使用できません。具体的には以下の通りです。

(1) 例えば住所録を作ろうと AddressTable なる下記のような
テーブルを作ります。
mysql> show columns from AddressTable;
+-------------+----------+------+-----+---------+-------+
| Field       | Type     | Null | Key | Default | Extra |
+-------------+----------+------+-----+---------+-------+
| Name        | char(40) | YES  |     | NULL    |       |
| Post_Number | char(10) | YES  |     | NULL    |       |
+-------------+----------+------+-----+---------+-------+
2 rows in set (0.02 sec)

(2) そして以下のように文字列を入力します。

mysql> insert into AddressTable values ('長潟隆治', '123-0873');

しかしリターンキーを押してもカーソルが外れるだけです。。
そこで '長潟隆治'のかわりに 'Nagagata Takaharu' とするときちんと
テーブルに登録されます。

マニュアルやその他調べられるものはいろいろ調べたつもりですが解決
できません。やはりインストールをもう一度やり直した方がよいので
しょうか。なにぶんデータベースについては初心者なので、とんでも
ない勘違いをしているかも知れませんが、ご教示くださいますよう
お願いします。	

使用しているのは FreeBSD 2.2.6 です。
