> １．MySQLへの接続（？）は下のもので大丈夫なのでしょうか。 
> これ以外に接続の仕方がわかりません。
それで接続と考えて良いと思います。MySQLのコマンドモードになっています。
ただし、テーブル等の操作にはそれなりの権限が必要です。色々と勉強するので
あれば、とりあえず[root]権限で接続してみると色々と試すことができてよろし
いかと思います。
mysql -u root
で[root]ユーザーで接続できます。パスワードを設定しているのであれば
mysql -u root -p
と入力するとパスワードを聞いてきますので、先に入力したパスワードを入力し
てみてください。その後 CREATE DATABASE を試してみてください。
