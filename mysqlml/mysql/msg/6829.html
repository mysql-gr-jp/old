mysqladminでrootのパスワードを設定することができました．
ご指摘の通り，mysqlを再起動してあげると，上手くいきました．

今回，mysqladminでパスワードを設定するために行った作業を以下に
まとめます．

方法としては以下のようなことを行いました．

TARA wrote:
> まず権限のない状態でMysqlを起動させてから
> 新しいパスワードを設定する方法です。
> rootのパスワードを忘れたときに有効です。

$ su
$ killall mysqld
$ /usr/local/mysql/bin/safe_mysqld --user=root --skip-grant-tables &
$ mysql mysql
mysql> select * from user;
mysql> update user set Password=null where Host='localhost' and User='root';
mysql> exit

ここでmysqlの再起動してあげる．

$ mysqladmin --user=root password your_new_password

これでパスワードを設定することができました．パスワードを設定する時は
一般ユーザでもrootでも同様に設定できました．

なぜインストール直後に，mysqladminでパスワードを設定することが
できなかったかの原因は不明ですが，上記の方法により解決できました．
