PerlからMySQLにアクセスするには、
以下のPerlのモジュールを利用することで可能です。

・DBI module
・Data-Dumper module  　←　Perl5以上では、すでに組み込まれている様です
・MySQL module 

詳しくは
・MySQL の perl のモジュールを使用するには
http://www.softagency.co.jp/mysql/TODO/perl/module.html
・4.10 Perl のインストールについて（MySQLマニュアル日本語版）
http://www.database.ne.jp/jpdoc/mysql-3.23.7/Docs/manual.ja_Installing.html#Perl_support
・20.5 MySQL Perl API（MySQLマニュアル日本語版）
http://www.database.ne.jp/jpdoc/mysql-3.23.7/Docs/manual.ja_Clients.html#Perl

などを御覧になってみてはいかがでしょう。
