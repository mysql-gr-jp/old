私もそんなに詳しくないのですが、「日本語EUC」と「UJIS」は同じもの
と考えていいと思います。

EUC は Extended UNIX Code の略で、エンコーディング方法を規定した
もので、言語は特定していません。日本語 EUC とか eucJP とかいう場合は、
JIS コードを EUC に従ってエンコーディングしたコードのことです。

UJIS は UNIXized JIS のことで、JIS コードを UNIX で扱えるように
(つまり EUC に従って)エンコーディングしたコードのことです。

…ということで同じものだと思います。あやふやな記憶を元に書いている
ので嘘いってるかもしれませんが…(^^;

ちなみに、MySQL に実装するときに UJIS としたのは、

・EUC だと日本語とは限らない。EUCJP だと長い。
・UJIS だと SJIS と語呂が合う。

…という理由からです。
