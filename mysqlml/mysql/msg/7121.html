Auth_MySQL_Info, Auth_MySQL_General_DB のことだと
思いますけど...


mod_auth_mysql のサイトにある README 
http://modauthmysql.sourceforge.net/README には、
Auth_MySQL_Info, Auth_MySQL_General_DB という
キーワードは、記載されておりません.


Auth_MySQL_Info, Auth_MySQL_General_DB の記述が
必要とは、どこに書かれていたのでしょうか？

おそらく、インストールした mod_auth_mysql のバージョンと
参考にした文献のバージョンが　異なっていて、必要だと思って
記述したキーワードが、インストールしたバージョンの
mod_auth_mysql では、使用できないってことでしょう.

バージョンが異なれば　設定するキーワードが　変更されるのは、
比較的良くあることなので、付属するドキュメントを読んでイン
ストールしたバージョンに　あった設定をしましょう.
