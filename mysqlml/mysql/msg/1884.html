とみたです。

MySQL 3.23.14 で、あるテーブルに load data するとサーバがダウンするよう
です。この一週間ほど調べてたんですが、発生条件や原因は良くわかりませんで
した。ソースもおっかけてみたんですが…。

mysql> create table t (aaa int, bbbb char(32), ccccc char(32), dddddddd char(32), eeee date, fffffff text, gggggg char(32), hhhhh char(32), iiiiiii text, jjjjjjjjj text);

でテーブルを作って、load data を実行すると返ってこなくなります。

mysql> load data infile "/tmp/data" into table t;

/tmp/data は次の内容です。

1	xxxxxxxxxxx	xxxxxx	xxxxxx	1999-01-01	xxxxxxxxxx	xxxxxxxx	xxxxxxxxxxx	xxxxxxxxxx	xxxxxxxxxxxxx

↓このように、項目名の長さを変えるとうまくいったりするんですよねぇ。謎…。

mysql> create table t (aaa int, bbb char(32), ccc char(32), dddd char(32), eeee date, fffffff text, gggggg char(32), hhhhh char(32), iiiiiii text, jjjjjjjjj text);
