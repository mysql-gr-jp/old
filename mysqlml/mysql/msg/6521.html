FAQに書いてある通り、ezmlm-mktab コマンドで CREATE TABLE のSQLを作成でき
ますよ。
ezmlm-idx パッケージ内の INSTALL.idx には
>  % ./ezmlm-mktab -d list | mysql -hhost -uuser -ppasswd -f ezmlm
と、一続きで書いてありますが、この命令を一個ずつ動作させてみれば判ると思
います。

DB名は、ezmlm 以外でも動くのか確認はしておりませんが、こちらでは問題ない
のでこのまま使用しています。
