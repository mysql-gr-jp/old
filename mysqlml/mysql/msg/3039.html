森下です。こんにちは。

MacOSXServer1.2へのインストールですが、ソースからコンパイルできました。

本家のサイトのドキュメント（コメント付き）で、pthreadパッケージなしで
コンパイルできたとの報告があります。

で、ふと思い返して、MySQLのソースを解凍た中にあるconfig.guessとconfig.subを
システムの/usr/libexec/以下にあるもので上書きコピーして、
（これでconfigureが通るようになる事がけっこうある。）

そんでもって、
--prefix-/usr/local/mysql --with-extra-charsets=sjis  --disable-shared
のオプション付きで./configureして、その後makeも成功、suしてからmake install
したら、あっさりインストールできました。で、あとは、

alias mysql '/usr/local/mysql/bin/mysql'
alias mysqladmin '/usr/local/mysql/bin/mysqladmin'

だけしましたけど?ysql_install_dbして　safe-mysqldしてからmysqladminで確か
めてみると、ちゃんと起動しています。

使ってる途中で不具合に出会ったりもしてません。

と、ここまで書いて、良く考えたら、pthreadパッケージをずーっと前にインストー
ルしてるんだから、なしでインストールできた事にはならないなあ・・・(^。^;)

でも、Makefileを編集したりなどなしで、インストールできてます。

ではでは。
