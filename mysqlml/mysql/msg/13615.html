こんにちは、広津です。

Mysql4.025-nt と書いてあるので、Windows プラットフォームと仮定します。

Windows MySQL 4.025 バイナリは、

> http://downloads.mysql.com/archives.php

の　MySQL Database Server 4.0　→　4.0.25 (160 files)　とリンクを辿ったページにある、

Microsoft Windows 32. (ZIP format) (1 Jul 2005, 25.1M)
Microsoft Windows 32. (ZIP format) (1 Jul 2005, 21.8M)

のいづれかです。
（上がインストーラー付き、下がインストーラーなし）

Windows バイナリディストリビューションには、
mysqld.exe
mysqld-opt.exe
mysqld-nt.exe
mysqld-max.exe
mysqld-max-nt.exe
の５つのサーバーが最初から含まれていますので、
インストール後、どのサーバーを実行するかで、InnoDB や BDB、名前付きパイプを
使えるかどうかが決まります。

インストーラー付きのバイナリでセットアップすると、セットアップ時にどのサーバーを
選択するかを聞いてきたように思います。
