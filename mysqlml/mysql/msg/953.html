 小林 栄です。

今なら、perl5.005_03でしょうか。
オマケでperlccが付いてきて御得です。

DBI -> DBD の順にインストールしていけば大丈夫です。

perl Makefile.PL
make 
make test

を、DBI、DBDに付いて実行します。DBDを入れる時、static linkを
しないと make test でエラーの嵐になりますので注意が必要です。
(その節は、いろいろと助けて頂きました)

あっと、もちろん、DBDをテストする時は、mysqlが起動してる必要が
あります。

Solaris2.4,2.5.1でも、linuxと同じ手順でいけます。
