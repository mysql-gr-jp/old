テーブルデータのダンプ＆テーブル再作成＆データのインポートが一番確実だ
と思います。

 * mysqldump --default-character-set=binary dbname tblname > tblname.sql

 * tblname.sql を編集して、DEFAULT CHARSET=latin1 を DEFAULT CHARSET=cp932 に変更

 * mysql --default-character-set=binary dbname < tblname.sql

データが失われるかもしれないので、バックアップしてから行なってください。
