PHPが文字コードがEUC-JPで、MySQLのデータがUTF-8でしょうか？
データが正しく保存されている前提ですが

set names ujis

上記のSQLを接続直後に必ず実行することによって、MySQLから
戻ってくる文字コードをEUC-JPにすることができます。
