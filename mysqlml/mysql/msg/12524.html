以下のSQL文の実行で1064のエラーになってしまいます。

INSERT INTO xps_whatsnew
VALUES(0,'manual','(desc_show)','','883840200','\'95から\'97の大会一覧 ');

1064 : You have an error in your SQL syntax. Check the manual that
corresponds to your MySQL server version for the right syntax to use
near '97の大会一覧 ')' at line 1

\'97から\'をはずして、ただの97にしても、97\'にしてもエラーがなくなりま
す。\'が97の直前の時だけ発生するようです。

このエラーが発生するのは以下の環境１で、環境２では発生しません。

環境１：WinXP_pro_sp1＋AnHTTPD1.42n＋MySQL4.0.26-nt(3.23.49)＋PHP4.4.1
　　MySQLのバージョンですが、winMySQLadminでは4.0.26-ntと表示されるの
　　ですが、phpinfoで見ると3.23.49と表示されます

環境２：linux(詳細不明)＋MySQL4.0.26＋Apache1.0.33＋PHP4.4.2-dev

なにか変えればよくなるのか全くわかりません。
