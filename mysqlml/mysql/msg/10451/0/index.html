MaxDB PHP 拡張のリリース
2004年11月8日
 
MaxDB PHP の新しい拡張機能がリリースされました。phpのodbc
拡張版の前の版に比較して、新しいドライバーは種々の恩恵
を提供します。最初にこの拡張機能はMaxDBのネイティブなインターフェース
をサポートします。そのおかげで、おおいに改善された良い性能を
提供できます。第二に初めのころの限られたドライバーのODBC
機能に比較して幅の広い機能を提供します。この機能は
PHPのmysqlと mysqliにコンパチです。
 
新しいドライバーおかげで既存のPHPのアプリやソリューションを簡単
にMaxDBに移植できます。このためPHPの解でMaxDBの幅広い機能、
スケーラビリティ、性能と使いやすさを利用できます。
 
この最初のリリースはPHP4とPHP5の双方をサポートします。
PHP5の新しいオブジェクト指向の機能をサポートする版の準備は
進んでおり、数ヶ月でリリースされます。PHP の拡張は完全な
標準のドキュメンテｘ－ションが用意されます。
 
現在新しいPHPの拡張はMySQLのダウンロード サイトから入手できます。
数週間のうちにPHPに接続するMaxDB のクライアント・インタターフェース
はMySQL FLOSSライセンス例外に適用されます。そのため、MaxDB とPHP
は他の多くのオープンソースのライセンスと使用することができる様に
なります。FLOSS 例外が適用されると新しいPHP拡張機能はPHPの Pecl
リポジトリに移管されてPHP ライセンス3.0の元自由に配布されます。
またドキュメンテーションとソースはPHPのCVSにチェックインされます。
