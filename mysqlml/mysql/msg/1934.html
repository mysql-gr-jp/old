1ヶ月前からMySQLを使い始めた初心者です。
どうぞよろしくお願いいたします。

passwordを設定しているユーザのDB接続について、
教えてください。

mysqlユーティリティでは、接続できるのですが、
DBIのconnetを使用すると、下記のエラーが発生して接続できません。
    DBI->connect failed: Access denied for user: 'foo@myhost'
         (Using password: NO)

mysqlは、次のように実行しています。
    % mysql -ufoo -phogehoge

DBIのconnectは、次のように実行しています。
    $USER = 'foo';
    $PASSWD = 'hogehoge';
    $Dbh = DBI->connect("DBI:mysql:$db:$HOSTNAME:$PORT", $USER,
           $PASSWORD);

.my.cnfに user,passwordを設定しても同じ現象です。
passwordの設定をはずすと、うまく接続できます。

以上、よろしくお願いします。
