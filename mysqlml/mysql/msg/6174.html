> CREATE DATABASEを行おうとすると、以下のようにlocalhostから
> のアクセスが拒絶されてしまいます。

> mysql> CREATE DATABASE sample;
> ERROR 1044: Access denied for user: '@localhost' to database 'sample'

mysqlサーバの設定方法に問題があり、場合によってはサーバが正常に
起動していない可能性があります。
もし"WinMySQLadmin"が常駐しているのでしたら、
タスクトレイ上の"WinMySQLadmin"アイコンに描かれている信号機の色を
チェックしてください。
「赤」もしくは「黄」の場合、サーバは停止しています。
また、"Show me"（アイコンをタスクトレイに格納しない）の設定でしたら、
"WinMySQLadmin"のウィンドウをアクティブにし、
ウィンドウの右上に描かれている信号機の色をご確認下さい。

サーバを正しく動作させる方法については、
改めて新規投稿なさった方がよろしいかと思います。
