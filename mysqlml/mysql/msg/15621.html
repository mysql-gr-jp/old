> mysql>  show variables like 'character%';

ここの設定がLOAD DATA INFILEに影響を与えるそうです。
試しに、

ALTER DATABASE strus CHARACTER SET sjis;

として、もう一度ロードできるか試してみてください。

うまくいったら、全体的にUTF-8でいくか、Shift_JISでいくか
方針を決めて、いったん作業を最初からやり直すのがよいと思います。

> mysql  Ver 14.14 Distrib 5.4.3-beta, for Win32 (ia32)

書籍に付属のバージョンを使っているのではないかと思いますが、
MySQLのバージョン5.4はベータ版のまま欠番になっており、
正式リリースはありません。

最新版の5.5か、一つ前の5.1を使うことをおすすめします。
