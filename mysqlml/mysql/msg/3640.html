Subject: [mysql 03640] OS のユーザを MySQL で管理

とみたです。

Postfix でメールを受け取るユーザを、/etc/passwd でなくて MySQL で
管理したくて、つまり村上さんの qmail+MySQL パッチのようなことをしたくて、
色々と試してみました。

最初は Postfix のソースをいじろうかとしてたんですが、結局 getpwnam と
getpwuid をフックすればいいんじゃん！

ということで、LD_PRELOAD を使って getpwnam, getpwuid を横取りするように
して、Postfix に限らず、動的リンクしているプログラムすべてに有効な
汎用的なものができあがりました。

こんなの使ってみたい人います？ (ドキュメント無し (^^;)

	http://www.tmtm.org/ja/mysql/mysql_pw-0.1.tar.gz

で、肝心の Postfix については、標準の機能で MySQL を使えることが
わかったので、こいつは使わなかったんですが… (^^;
