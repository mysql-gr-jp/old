Subject: [mysql 02263] トランザクション処理について

　豊島＠クロスネットです。

　最新の MySQL 3.23.21 を使ってトランザクション処理の速度などの調査をしたいの
ですが、トランザクション処理が動かなくて参っています。
# 他のバージョンで試したことも無く・・・
# このバージョンでトランザクションがちゃんと動いてるよ！という方も
# よかったらそのバージョンを教えてください。

　Berkeley DBを入れたり/etc/my.cnfを弄っているのですがさっぱり動いてくれませ
ん。

BDB_HOME=/home/BDB

# コンパイル設定
./configure
 --prefix=/home/mysql
 --with-charset=euc_kr
 --with-berkeley-db=/home/BDB
 --with-berkeley-db-includes=/home/BDB/include
 --with-berkeley-db-libs=/home/BDB/lib

# 以下、/etc/my.cnf の mysqld部分
[mysqld]
port        = 3306
socket      = /tmp/mysql.sock
language    = /home/mysql/share/mysql/japanese/
bdb-home    = /home/BDB
set-variable    = key_buffer=16M
set-variable    = max_allowed_packet=1M
set-variable    = table_cache=64
set-variable    = sort_buffer=512K
set-variable    = net_buffer_length=8K
set-variable    = myisam_sort_buffer_size=8M
# これを設定するとエラーが出る。
#set-variable	= autocommit=0
log-update
skip-locking

　なにか設定がたりないのでしょうか？
