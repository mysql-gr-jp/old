【エラーメッセージ】
Can't connect to local MySQL server throught socket 
  '/var/lib/mysql/mysql.sock' (13) in line nn.

【エラーが発生するPHP関数】
mysql_connect('localhost','username','password')
mysql_pconnect('localhost','username','password')
この関数のみの数行でテストプログラムを作っています。
ホスト名など各引数は、定数を与えています。

【動作環境】
テスト環境のため、localhostでの環境です。
Redhat Enterprise Ver.3
 mysql-3.23.58-1
 mysql-devel-3.23.58-1
 php-4.3.2-8.ent
 php-mysql-4.3.2-8.ent
 apache 2.0.46

【確認したこと】
・mysqladmin pingでMySQLのalive
・/var/lib/mysql/mysql.sockの存在とパーミッションが777
・/etc/my.cnfで[mysqld],[client]の各セクションで
　　socket=/var/lib/mysql/mysql.sock であること
・mysql --user=username -p databasename　を実行し、
　パスワード入力すると接続できること
・MySQLデーモンは以下のコマンドで手動起動
　/usr/local/mysql/bin/safe_mysqld &
・MySQL起動時のログに[Warning]が一個存在し、[Error]がない
　[Warning] Asked for 199668 Thread stack,but got 126976
・telnet localhost 3306 でポート番号の確認
