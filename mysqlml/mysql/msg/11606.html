mysql>FLUSH PRIVILEGES;

-----------------------------------------
<?php
 $DBSERVER="www.xxx.com";
 $DBUSER="aaa";
 $DBPASSWORD="bbb";

 $con=mysql_connect($DBSERVER,$DBUSER,$DBPASSWORD);
 if($con){
  print "接続成功　接続IDは".$con."です";
 } else {
  print "接続失敗";
 }
?>

------------------------------------------
Warning: Host 'www.xxx.com' is not allowed to connect to this MySQL server
in /home/.sites/114/site88/web/test1.php on line 6

connect to this MySQL server in /home/.sites/114/site88/web/test1.php on
line 6

ゴリ押しで入れてもらったためか、あまりよくわかってないようで
