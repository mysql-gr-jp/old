あー、また multipart だぁ。忌むべし＞Gates

一括定義とは、
-------------
create table test (
  name char(32),
  addr char(64)
);
-------------
とか言う table.def を一発で、ということなら
shell で

> mysql -B データベース名 <table.def

とかやれば出来ます。これは単にバッチ実行なので、一度に複数とか、
ついでにテキストデータのインポートとかも書いてしまえます。
