こいしです。

返値を count の大きな方から小さな方へとソートしたいのですが、どうも
上手く行きません。以下の様に　WHERE　無しの場合ではソートされて
正しい返値があるのですが、WHERE を設定すると何も返って来ません。
もともと、ORDER　と　WHERE　の複合処理は出来ないのでしょうか？

*上手く行く
$result = mysql_query("SELECT FROM $table ORDER BY count DESC");
*上手く行かない
$result = mysql_query("SELECT FROM $table ORDER BY count DESC WHERE
count='0'");

count　は BIGINT UNSIGNED。
