小野です。

SELECT max(count) c,name FROM TABLE ～

は、「c を max(count) のエイリアスとして用います」という意味で、
正確には、

SELECT max(count) as c,name FROM TABLE ～

と記述します。

>   max(count) c の部分で max(count)を  c にマッピング？していると
>  考えていいのでしょうか？

ですから、およそ仰る通りです。
