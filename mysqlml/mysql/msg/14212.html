ムハマドです。

データベースかテーブルのGRANTだけは不十分です。/etc/hosts.allowのように
MySQLにアクセス制限もあります。
MySQLへ接続して、アクセス権限があるかどうかを確認してみてはどうでしょう
か。
次のように確認して、ユーザに対してHostの列にWindowsの環境からの接続（IP
かホスト）の情報があるかを確認ください。

mysql> use mysql
mysql> select Host, User from user;

権限はなければ、追加してみてください。
