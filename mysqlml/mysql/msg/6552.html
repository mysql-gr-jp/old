On Fri, 08 Nov 2002 23:50:28 +0900
HOLA <hola@xxxxxxxxxx> wrote:

> あるdatテーブルが
> 
> id : smallint(3)
> dat1 : varchar(64)
> regist_date : datetime ( "Y-m-d H:i:s" )
> 
> とすると
> SELECT count(dat1) FROM dat GROUP BY regist_date
> でSELECTかけると　date("s")で,くくると思いますが例えば
> "d"でGROUP BYをかけたり　若しくは週単位（月～日曜日）でといったことをし
> ようと思うとGROUP BY以降の文法はどのように書いたらよい
> のでしょうか？

「date("s")でくくる」というのが何を意図しているのかよくわかりませんが、
やりたいのはこういうことでしょうか。

	SELECT count(dat1) FROM dat GROUP BY DAYOFMONTH(regist_date);
	SELECT count(dat1) FROM dat GROUP BY WEEKDAY(regist_date);
