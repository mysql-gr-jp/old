竹山と申します。

MySQL 5.1 において、ユーザ単位で DROP DATABASE を禁止し、
かつ DROP TABLE は許可する方法について調べております。

マニュアルから、DROP 権限を外せば、DROP DATABASE できなくなることはわかりました。

＞DROP DATABASE を利用する為には、データベース上の DROP 権限が必要です。
http://dev.mysql.com/doc/refman/5.1/ja/drop-database.html
http://dev.mysql.com/doc/refman/5.1/ja/grant.html

ですが、これを読む限り、DROP DATABASE と DROP TABLE に関する権限は一緒になっており、
DROP DATABASE のみを禁止することはできないように思われます。

何かいい方法がありましたら、ご教示いただけないでしょうか。
