こちらはデフォルトでインストールした状態ですか？

> MySQL Error Number 1130
> Host '192.168.1.2' is not allowed to connect to this MySQL server
まず、こちらですが、CentOS に限らず MySQL をインストールした直後ではユー
ザは root@localhost しか存在せず、localhost からしか接続できないようにな
っていると思います。そのため、リモートから接続できないかと思われます。
このエラーメッセージ番号は、ローカルからしか接続できない場合に表示される
ものと記憶しております。

したがいまして、ユーザを作成し権限を付与する必要があります。

GRANT ALL PRIVILEGES ON *.* TO ユーザ名@ホスト名 
IDENTIFIED BY 'パスワード文字列' WITH GRANT OPTION;

> MySQL Error Number 2003
> Can't connect to MySQL server on'localhost'(10061)
また、こちらは XP の方にも MySQL をインストールしているということでしょう
か？もし、そうならこのエラーメッセージ番号はインスタンスが起動していない
か、データベースがクラッシュしている場合に表示されるものと記憶しておりま
す。サービスが起動しているかどうか確認し、もし起動しているようならログフ
ァイルのエラー内容確認してみるといいです。
