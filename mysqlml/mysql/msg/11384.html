MySQLのエンタープライズ生態系は拡張続ける
04/20/05

Vance McCarthy著
MySQL は今週のMySQLユーザコンファレンスでエンタープライズ
でのオープンソースの主な解のコアになる立場を固めました。 多くの
コマーシャルとオープンソースのソフト会社はMySQLの2005年
にOracle, Sybase, MicrosoftやIBMなどのコマーシャルデータベース
に取って代わる戦略をサポートするべくその機能をMySQLに
追加します。

MySQLのNicolas Pujolは「MySQLの採用が増加しており採用が増加すれば
その生態系は拡張します。 今ある一定以上の地点に達したので
エンタープライズ・ソフトのベンダーが注目を始め
アプリケーションにデータベースを使用する際にカスタマーの
コストを削減しようとします。」と語りました。

オープンなエンタープライズの傾向は新しい製品・サービス・サポート
を生み出します。

# Arkeia社はデータ防御とホットバックアップのソフトの
提供社ですが、Arkeia NetworkバックアップとArkeia サーバー
バックアップ製品群にMySQL Network サーティフィケーション
を取りました。

# EBM ソフト社は中小企業用のビジネス・インテリジェンスの
ソフトウエアのOLAP4Allのアップグレードを発表しました。OLAP4All 2.0 は
分かりやすい画面とレポートのデザインで、ユーザがカスタム
のレポートや解析の画面を設計する際、数多くのコンポーネント（
チャート、テーブル、テキストやイメージなど）を
選択することで容易に生成することができます。

# Embarcadero Technologies はMySQLのデータのライフサイクルを
管理する製品を発表しました。これらは開発者とDBA
がデータのモデル、データベースやスキーマの生成、複雑な
 ETL (extract, transform, and load)のコーディネート、マイグレーション
やテストの管理や性能テストを行うことを容易にします。

# Emic Networksはオープンソース用にアプリケーション
のクラスターの解を提供しますが、MySQL Networkに参加して
MySQLベースのアプリケーションのハイアベイラビリティを
増長します。Emic CEO のEero Teerikorpiは「Emicの m/cluster
はミッションクリティカルなMySQLのデータベースを常時(24X7)
アクセス可能にします。サービスの停止なしでオンラインで
保守も可能にします。MySQL Networkで双方のカスタマは
両社のメリットを生かし、最高のアップタイムと信頼性と
スケーらビリティを達成します。」と述べた。

# HiT Software はDBMoto 5 ツールの紹介をしました。これはMySQL
のユーザが他の商業データベースからマイグレートするのと
リプリケートとXMLと MySQLの結合を助けます。

# Metilinxは一連のデータベース・ツールを発表して、DBAがMySQL
やその他のコマーシャル版のデータベースのミックスでリプリケート
やリカバリーを可能にします。これはLANやWANを介しても可能です。

# OpenLogic は最新のBlueGlue オープンソース・インフラ管理
スイートを発表。これを使用すればMySQLユーザのために
開発と展開を促進。BlueGlueはMySQLベースのオープンソースの
スタックを数分でインストール、コンフィギュア、結合します。
また予め用意されたインストール、インテグレーションの
スクリプトやコードを通じてMySQLベースのエンタプライズアプリケーション
の開発を促進するアプリケーションを含みます。

# Quest ソフトはエンタプライズのカスタマの間でToad for
MySQL ソフトが急速に人気が上がっていると報告。Toad for MySQL
はデータベースの開発を 促進し、開発者とアドミのために
データベースのオブジェクトの管理を自動化します。これでユーザ
の生産性を向上します。更にToad for MySQLはDBAと開発者のほかの
データベースからの経験をMySQLへ適用することができます。

# Safe ソフトウエア社はコアの部分でSpatial ETL
(extract, transform, and load)をMySQLをサポートします。この解で
MySQLのユーザは130のフォーマットやシステムからspatial またはspatial
でないデータをインポートしたらエクスポートすることができます。

# SugarCRMは Sugar Suite 3.0 ベータで CRM Open Source アプリケーション
を発表。3.0 版はMySQL上で実行し企業やユーザに毎日のビジネス活動
へのより良い管理能力を提供します。これにはSugar Collaborateを
使用しマーケティング、セールスやサポート・モジュールからの改善
分を含みます。オープン・エンタプライズ傾向の記事はここから。
http://www.oetrends.com/news.php?action=view_record&idnum=420

# Tableau ソフトはビジネス・インテリジェンス産業の
ビジュアルな解析やレポートシステムを提供します。Tableauは
今日MySQLのエディションを発表しました。この製品は技術能力
のある人々にMySQLのデータベースに格納されている情報を
ドラッグ・ドロップのインターフェースで見て
理解できるようにします。

# WebyogはMySQLのGUIであるSQLyogの開発社ですが、HTTP/SSHトンネル
機能を発表しこれを使えばユーザが効率的にMySQLを管理できます。
これはMySQL のポートがブロックされていてもまたリモートアクセス
が許可されていなくても可能です。SQLyog があればユーザはユーザは
ウエブベースの管理ツールの制限から開放され、MySQLの生産性が
あがります。

# Zend TechnologiesはPHPの開発者ですがMySQL Network サーティファイド
パートナー・プログラムに参加しました。このプログラムに参加することで
ZendのPHPの解はMySQL Network用にサーティファイされます。これでエンタープライズ 

用のPHP のアプリもMySQLと問題なく結合されます。
