MySQL 5.0のエンタープライズ・データベースの機能

新しいMySQL 5.0 はエンタープライズ・レベルの信頼性
と高性能を持ち、高度なSQL 2003 標準を実装し、ビジネス
クリティカルなアプリに対する一番コスト・パーフォーマンス
の良い解を提供します。製品の新しい機能はプロプリエタリー
データベース開発者やDBAにはなじみの深いものです。この
ためITの部門はデータ、アプリやスキルセットを統合できます。

MySQL 5.0はたくさんのエンタープライズの機能を提供します。

* Stored ProceduresとSQL 関数 -- to ビジネス・ロジックを
データベースに組み込み、性能をアップ
* Triggers -- データベース・レベルで複雑なビジネスロジックを
実施
* Views -- 重要な情報を守秘
* Cursors -- 容易にデータベース開発と大きなデータの参照
を可能に。
* Information Schema -- メタデータに簡単なアクセスを
* XA 分散・トランザクション -- ヘテロな環境での複数データベース
を含む複数のトランザクションをサポート
* SQL Mode -- サーバーが保障する新旧のデータの全一性
* 新しいフェデレートでアーカイブストレッジエンジン --
MySQLにユニークなプラグインできるストレッジエンジンのアーキテクチャ
を持ち、多大な柔軟性、機能と性能を維持、データベースの
エンジンをスワップインやアウトが容易。これはユーザの
アプリの要求による。
* 新しいマイグレーション・キット -- 新しいグラフィカルな
ツール。全てのデータやオブジェクトをOracle, Microsoft SQL Server
や Microsoft Access やその他のデータベースからMySQLへマイグレート
* Instance Manager -- リモートでMySQLのサーバーをスタートと
ストップ。同時にリモートでコンフィギュレーションファイルの編集、
エラー、クエリログの読み。
* アップデートした Connectors and Visual ツール -- 新しい
高性能版のMySQL's ODBC, Java と .NET データベース・ドライバーは現在
存在。その他アップデートした MySQL Query BrowserとMySQL Administrator.

サポートしているプラットフォーム

製品版のMySQL 5.0は以下のプラットフォーム上で利用可能。 Linux,
Windows, Solaris, Mac OS X, FreeBSD, HP-UX, IBM AIX 5Lとその他。
全てのMySQL データベースのソフトは２重ライセンスで提供。開発者
と企業はオープンソースライセンスとコマーシャル版を選ぶことが
できます。エンタープライズのユーザは MySQL Networkのサブスクリプション
を得ることもできます。これには認定済みのソフトとサポートが
含まれます。ソフトのダウンロードは http://www.mysql.com/products/.
