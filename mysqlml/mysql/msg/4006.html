> 	$result = mysql_query("select * from TABLE_A 
> 		where TABLE = $ABC");
> 
> 	このSelect文の結果の $resultを使用して
> 	さらに 絞り込み検索（Select分の実行）を行いたいのですが
> 	何かいい方法をご存知の方いましたらご教授ねがいます。

MySQL には sub-select の機能がないので、１回の SQL ではできないですね。
結果を一旦別のテーブルに保存しておいて、それからテーブルを join して、
再度 select をすれば良いと思います。

	create temporary table TMP select * from TABLE_A where ～;
	select * from TABLE_A, TMP where TABLE_A.COLUMN1 = TMP.COLUMN2 ～;

とかって感じで。
