天野です。

OS 環境は Linux でしょうか。だとすれば、Mysql や PhpAdmin の問題ではなく、Linux 
の文字コードの割り当てが Windows と一部異なることによる文字化けです。

Linux 上で「～」など一部の記号文字や機種依存文字を[JIS系コード]←→[unicode系
コード]変換すると化けてしまいます。

これを回避するには、windows 側で utf-8 に変換してから mysqlで扱うか、Linux 側で
の文字変換プロセスにこの問題の対策を施すかということになります。

詳しくは http://legacy-encoding.sourceforge.jp/wiki/ をご覧ください。
