	これなんですけど、私がいつもやってる手順は

	1. 本家から SRPMSつまりソースパッケージをとってくる
		wget http://www.mysql.com/Downloads/MySQL-3.22/MySQL-3.22.32-1.src.rpm
	2. ソースをインストール
		rpm -ivh MySQL-3.22.32-1.src.rpm
	3. SPEC を変更
		cd /usr/src/redhat/SPECS
		vi mysql-3.22.32.spec
	   configure 行を探して以下を追加
		--ith-charset=ujis \
	4. binary package を作成
		rpm -bb mysql-3.22.32.spec

	実際には ujis とか sjis オプションつけてコンパイルしたよ
	ってことで release をいじったりしてますが、最低限これだけで
