LinuxサーバにMySQLを入れて、ODBCを経由してAccessで利用しています。
今困っているのが、Time というタイプの入力方法です。
 
AccessのテーブルにMySQLの「AAA」というテーブルをリンクさせて、
Time型の「BBB」というフィールドに 8時30分 というデータを登録させます。
 
最初に行ったのが、AAA テーブルに BBB フィールドを追加しました。
．．．ALTER TABLE AAA ADD BBB Time;
 
次に ODBC を使って、AccessからAAAテーブルをリンクさせて、
Access の AAA テーブルを開いて、BBB フィールドに08:30:00 と入力して
テーブルを閉じました。
 
再度、AAAテーブルのBBBフィールドを確認すると、2010/05/30 23:52:00
と表示されます。
 
どの部分を訂正すると、正しく 08:30:00 と登録できるのでしょぅか。
