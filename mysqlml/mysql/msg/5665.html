その後、PostgreSQL の方でも色々調べて見ました。
以下、PostgreSQLの日本語ドキュメント から・・・

----------------------------------------------------------------------

Name

CREATE DATABASE  --  新規にデータベースを作成する 

Synopsis

CREATE DATABASE name [ WITH LOCATION = 'dbpath' ]

入力 

name
作成するデータベースの名前 

dbpath
データベースを所定の場所以外に作成したいときは、
バックエンド サーバが解釈できる環境変数(例えば 'PGDATA2') 
あるいは絶対パス(例えば'/usr/local/pgsql/data') で指定することができます。 
どちらの場合でも、データベースを作成する場所をinitlocation コマンドで
事前に設定しておく必要があります。 

----------------------------------------------------------------------

うぅ・・・MySQLでも同じオプションがあったら文句なしなんですが。
