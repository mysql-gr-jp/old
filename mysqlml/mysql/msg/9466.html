mysql の話題から外れますが、私なりの見解を述べたいと思います。

> 中国の事情は知りませんが、韓国人は Unix とオープンソースに疎いです。

日本もさして変わらない、というのが私の感覚です。

Linux も FreeBSD も「タダで使える UNIX」くらいにしか感じて
いない方が殆どでしょう。

ただ、FreeBSD/NetBSD にあっては、IBM-PC/AT 互換機ではなく
NEC の PC-9801(9821ではない) の時代から移植が試みられ、実際に
稼働している例も多いようです。が、PC-9821 の事実上の終焉によ
って PC-9801/9821 は業務アプリを Windows に移植するお金が無い
ところか、ゲームの為に持っているというコンシューマが存在する
くらいで、多くの方は手放してしまっています(私は仕事上あります
けど)。

全く無視する訳ではないですが、UNIX の普及に貢献した、とは
あまり言えないのではないか、と思っています。

> Unix の流行った 1980年代に米国製の高価なワークステーションを大量に購入
> するだけの経済力がなかったせいではないと思っています。

これは同感です。実のところ、当時バブルで経済力のあった日本で
これらをどれくらい購入したかは知りませんが、当時理系の学生
だった私の通学した大学には１台もありませんでした。あったのは
NEC の PC-8801 or PC-9801、若しくは Fujitsu の FM-7 などです。

もっと立派？な大学や研究所にはあったのかもしれませんが、私
などは PC-9801(無印)に MS-DOS 1.27 に Fortran77 を勉強した、
なんて感じです。UNIX という名前は知っていても実際に動作する
環境を見たことがなかった訳です。さすがに就職してからはみる
事もありましたが、まだまだ各社独自アーキテクチャのビジネス
用 PC、或いはオフコンの時代で、身近なものとはとても言えなか
ったのです。
