遠藤さんは書かれました
>CREATE TABLE AA (
>  a1 int,
>  a2 int,
>  INDEX i( a1, a2 )
>)
>
>と
>
>CREATE TABLE BB (
>  b1 int,
>  b2 int,
>  INDEX i1( b1 ),
>  INDEX i2( b2 )
>)
>
>って違うんですよね？

上の場合は、a2 にのみ関する条件が与えられた場合はインデックスは
使われません(というか使えませんが)、下の場合は、b2 にのみ関する
条件が与えられた場合は、インデックスが使われます。
