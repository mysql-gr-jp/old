OS error code  28:  No space left on device

ディスクが溢れたのではないかというところを疑ってみてはと思います。

  実行開始〜実行中〜実行完了までに
  C:ドライブの空き容量がどのように変化するか調べる。
  (正常に完了する場合でも一時的に空きがギリギリまで減っていないか)

・Windowsで該当ディレクトリ・ユーザに
  クォータが仕掛けられていないか調べる。

エラーが起きたSQL、テーブル定義、それぞれのレコード件数を
見せてもらえると、何か起こりそうなSQLなら分かるかもしれません。
