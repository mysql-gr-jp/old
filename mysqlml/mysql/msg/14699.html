ひらつかです

> Server characterset:    utf8
> Db     characterset:    latin1

Server charactersetは、設定【後】に作ったデータベースに対して
デフォルトのキャラクタセットを指定するものです。

Db charactersetも、設定【後】に作ったテーブルに対して
デフォルトのキャラクタセットを指定するものです。

character-set-serverを設定してmysqld再起動しただけでは、
既存のデータベース、テーブルとも、何も変わらないです。

一旦データを引き抜いて、テーブルを作り直し、
データを入れなおすことをおすすめします。
