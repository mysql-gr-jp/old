> どのようにうまくいかなかったのかを書いた方が解決は早いと思います。

mysql_query("CREATE TABLE ".$table."(id INTEGER NOT NULL AUTO_INCREMENT,chat
VARCHAR(200) NOT NULL,PRIMARY KEY(id),INDEX (id,chat))");
を
mysql_query("CREATE TABLE ".$table."(id INTEGER NOT NULL AUTO_INCREMENT,chat
TEXT NOT NULL,PRIMARY KEY(id),INDEX (id,chat))");

と　VARCHAR(200) 　を　TEXTに　書き換えただけで、テーブル（データファイル）
自体が作成されなくなってしまいます。しかも、ＰＨＰでも、MySQLの方でも、
エラーログが表示されていない様です。正しくテーブルが作成されているのか
確認する為に、テーブルも毎回破棄させています。
