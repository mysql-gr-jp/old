C:\Documents and Settings\saori>
と記載されております。この直後に
mysqladmin -u root create webdb
と記述しますと、
'mysqladmin'は内部コマンドまたは外部コマンド、
操作可能なプログラムまたはバッチファイルとして認識されて
いません。

と帰ってきてしまいました。

また、ＳＱＬモニターを起動させた直後の行で試しました。
C:\Documents and Settings\saori>CD C:\mysql\bin
C:\musql\bin>mysqladmin -u root create webdb

すると、

C:\mysql\bin>

と出てきただけでした。
ですので、mysqlモニターを起動させた直後に入れました。
C:\mysql\bin>mysql

mysql>mysqladmin -u root create webdb
