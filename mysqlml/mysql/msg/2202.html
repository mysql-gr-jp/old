お蔭様で、例の問題解決しました！

ローカルへのファイル出力でのコード変換は以下のコードで解決しましたので、報告
いたします：
--------------------------------------------------
Writer out = new OutputStreamWriter(System.out,"EUC_JP");
PrintWriter outWriter = new PrintWriter(out);
outWrite("A string to System.out");
outWrite("Another string to System.out");
// Some access to database
outWrite("Other strings to System.out ^^;;");
outWrite.close();
// destroy the database connection
--------------------------------------------------
その後EUC_JPをSJISに変換しても、問題なくShift_JISファイルを生成しました。
