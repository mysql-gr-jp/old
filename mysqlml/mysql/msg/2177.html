おはようございます、佐々木です。

つい最近ODBC経由でmysql接続を試したところです。
やっぱりGUIは便利ですね。


1. mysql(Linux)側の設定

   GRANTなどでテーブルへの参照のパーミッションを設定する。

      DB名       : hoge
      WinのIP    : 192.168.1.2
      接続user名 : user
      接続pass   : pass

   の場合、具体的に言うと

      GRANT ALL on hoge to user@xxxxxxxxxx IDENTIFIED BY 'pass';

   でいけます。ALL を SELECT にすれば DELETE などは許可しませんし、
   user@'%' とすればどのマシンからも接続を許可します。
   詳しくはマニュアルの「6 MySQL のユーザー権限はどのように動くか？ 」
   をご参照ください。

2. Access(Win)側の設定

   どっかから MySQL ODBC ドライバをダウンロードしてインストール。
   http://www.softagency.co.jp/mysql/Win/myodbc.html とか。

   コントロールパネルの ODBC Data Sources を開きます。
   User DSN で目的のDBのマシン名、テーブル名などを追加。

   Accessを立ち上げて [ファイル] - [外部データの取り込み] - [インポート]。
   ファイルの種類を ODBC に変更すると Machine Data Source にさっき設定し
   た DB があるのでセット。
