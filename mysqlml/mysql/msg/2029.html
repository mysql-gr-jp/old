manual.ja_Reference.html#BLOB から

MySQL 3.23.2 以上では、BLOB と TEXT 項にインデックスを持つことが出来ます。
それ以前のバージョンではインデックスをもつことが出来ません。 

moeruさんの
題名「[mysql 02023] TINYTEXT は INDEXにできない？」のメールからの引用です
m> 
m> 
m> こんにちは、モエルです。
m> お世話になっています。
m> 
m> 質問なんですが
m> 
m> TINYTEXTは、INDEX (title)にできないのでしょうか？
m> 
m> 
m> CREATE TABLE item (
m> 	itemid INT(20) NOT NULL AUTO_INCREMENT,
m> 	userid INT NOT NULL,
m> 	date timestamp(14),
m> 	title tinytext NOT NULL,
m> 	PRIMARY KEY (itemid),
m> 	INDEX (userid),
m> 	INDEX (title),
m> 	INDEX (date),
m> );
