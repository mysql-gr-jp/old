かじやまです。

7.0のGAリリースを発表したMySQL Cluster関連ですが、
注意点がいくつか出てきていますので下記に列挙します。

- MySQL Cluster 7.0 Windows版
　既にダウンロード可能になっていますが、
　Windows版のみGAリリースではなく、"pre-GA"リリースだそうです。。。
　他のOS用のコードと比べると、実装している機能は同じだが、
　本番運用向けと言うにはもう少しテストが必要 by 開発チーム

- MySQL 6.3からMySQL7.0へのアップグレード
　現在ダウンロード可能な7.0.5には、
　6.3からの無停止でのアップグレードが出来ません。
　無停止でのアップグレードを検討される場合は、
　月末までにリリースを予定している7.0.6をお待ち下さい。

- NDB APIのJava実装について
　これまでNDB/Jという名前で出ていたNDB APIのJava実装ですが、
　NDB/Jでググったとき（英語で）に出てくる下記などは古い情報です。
　- Johan Anderssonのブログ（注意書き有り）
　- SourceForge.net: NDB/J: NDB/J

　最新の情報はNDB/Bindingsとして他の開発言語の実装とともに、
　launchpadにプロジェクトページが作られています。
　https://launchpad.net/ndb-bindings
