クラスター

CL 1. MySQL 5.0はMySQLクラスターに対応していますか？
はい。

CL 2. MySQL 5.0はMySQLクラスターに新しい機能/性能を提供していますか？
5.0でのMySQLクラスターは使用メモリや様々なスピードの最適化を導入していま
す。詳細はこちらをご覧下さい。
http://dev.mysql.com/doc/mysql/en/mysql-5-0-cluster-changes.html

CL 3. MySQL 5.0はディスクベースのMySQL Clusterに対応していますか？
いいえ、まだです。この機能に必要不可欠な基盤は現在開発中ですが、全機能に
ついては"rolling feature"としてMySQLのロードマップでは位置づけられていま
す。つまり、主要な機能ではないが将来的に時間あるときに実装されるといった
ものです。特定のお客様からの要望により、このスケジュールが変わることもあ
ります。詳細はこちらをご覧下さい。
http://dev.mysql.com/doc/mysql/en/mysql-5-1-cluster-roadmap.html
