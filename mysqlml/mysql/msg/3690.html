　こんにちは。加藤です。

　　何やらFAQ化してきたようなので、前回書いたものをちょっとだけまとめて
　みました。よろしければMyNAのFAQでお使いください。


○現象

　　スクリプトを実行すると、以下の警告が出て実行できない。
--
Warning: MySQL Connection Failed: Can't initialize character set n (path: default)
in hogehoge.php on line nn
--

○原因

　　PHP4に同梱のMySQLクライアントライブラリの不具合。

○対処

　　PHP4に同梱のものではなく、MySQLに含まれるライブラリを明示的に指定して
　PHPをmakeしなおす（configureから）。

（例）
　# configure --with-mysql=/usr/local
　# make
　# make install

　（※）/usr/localの部分はMySQLインストール時に--prefix=で指定したpath
