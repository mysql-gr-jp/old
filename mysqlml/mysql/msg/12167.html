2006年MySQL ユーザーコンファレンス

2005年９月２１日

MySQLで発見、コネクト、成功し、あなたのビジネスをスケールアウトしましょう。

2006年４月24-27日 ― カリフォルニア州、サンタクララ
プロポーザルの締め切り2005年11月7日

実社会へのMySQLの応用の知識と経験を語りましょう。
来年のコンファレンスの準備を始めました。コンファレンスで
の発表のプロポーザルを求めています。
プロポーザルを提出するにはここを押してください。
http://www.mysqluc.com/cs/mysqluc2006/create/e_sess

2005年のユーザーコンファレンスの成功を元に、MySQLは O'Reilly
Mediaと共同で来年のコンファレンスも開催します。開発者、パートナー
や他のユーザを集めて2006年の MySQL ユーザーコンファレンスは多くの
チュートリアルやセッションを網羅します。またMySQL　Universe
を紹介します。最新のビジネスと製品のオーバービューから
スケーリング、オプティマイゼーションとMySQLの内部まで。

来年は展示ホールを更に充実し拡張します。また、出席者と
参加者に特別なイベントを企画しています。

コンファレンス詳細

MySQL ユーザーコンファレンスは1000のオープンソース
の熱狂的な支持者とMySQLを中心とする生態系の関連会社
を集めます。2006年のコンファレンスのテーマは
「 MySQLで発見、コネクト、成功し、あなたのビジネスをスケールアウトしま
しょう。」
です。コンファレンスでは製品の広い支持、機能をアピールして
コミュニティのメンバーに最良のデータベース・ソフトウエアを
開発できるように、知識を吸収できるように、ツールを開発
できるように、ビジネスを展開できるように、そして個人のスキル
を伸ばすことがえきるようにします。

提案は出席者の役に立つものでなければなりません。

* ベストプラクティスを含む、MySQLの全部の力を際だたせるもの。
* MySQL5.0とそれ以降の機能を学び、使う。
* MySQLのアドミ
* 実社会でのアプリケーションのアーキテクチャーを開発する専門家
から学ぶ
* MySQLへのマイグレーション
* リプリケーション、クラスタリングや他のツールや技術
を使用したスケールアウト
* 多くの素晴らしいMySQLコミュニティのプロジェクトについて学ぶ

トピックとしては

* DBAとセキュリティ
* アーキテクチャーと技術
* ビジネスと事例
* データウエアハウスとビジネス・インテリジェンス
* LAMP (Linux, Apache, MySQL, PHP/Perl/Python)
* Java
* .NET/Mono
* MaxDB
* マイグレーション
* 性能チューニングとベンチマーク
* クラスター、リプリケーションと他のスケールアウトと
ハイアベイラビリティの解
* 組み込み MySQL: libmysqldと組み込みOS 環境
* コミュニティ・プロジェクト

そのほかの一般のトピックとセッション。プロポーザルを
用意するときは、対象の聴衆を考慮してください。開発者
DBA, ビジネス・マネージャーか。。。レベルも。概要
から詳細な内部情報まで。MySQL Developer Zoneで
どのトピックに一番興味があるかのアンケートをとって
みました。

プロポーザルは２つのタイプに分かれます。
１は４５分のセッションでのプリゼン。２は３時間
のチュートリアルです。

重要な日は

* 提出の期限は2005年11月7日の午前１２時です。
* 2005年11月３０日までにプロポーザルの採用についてお知らせします。
* 最終のセッションの詳細、スピーカーの経歴、チュートリアルの
資料の締め切りは2006年3月13日までに。
* コンファレンスの登録は2005年12月にオープンします。

コンファレンスの詳細は
http://www.mysqluc.com/
