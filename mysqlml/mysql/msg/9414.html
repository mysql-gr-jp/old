On Mon, 24 May 2004 04:29:47 +0900
Hideki Hayashi <loneliness@xxxxxxxxxx> wrote:

> 例えば、以下の様なデータで、直前のデーターと直後のデータを取得したいの
> ですが。
> 何か良いsql文はないでしょうか？
> 
> ------------------------
> id	data
> ------------------------
> 1	aaaaa
> 3	bbbbb	
> 4	ccccc  現在の位置(SELECT * FROM table WHERE id = 4)
> 9	ddddd
> ------------------------

何のひねりもないですが (^^; こんな感じでどうでしょうか。

mysql> select * from t where id < 4 order by id desc limit 1;
+------+------+
| id   | data |
+------+------+
|    3 | bbbb |
+------+------+

mysql> select * from t where id > 4 order by id limit 1;
+------+------+
| id   | data |
+------+------+
|    9 | dddd |
+------+------+
