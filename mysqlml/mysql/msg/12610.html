procedures

移行元：Windows2003server、Oracle9i
移行先：WindowsXP Professional、MySQL5.0.17
バージョン：Migration Toolkit1.0.21
現象：Manual Editing画面で下記エラーメッセージが出て、移
行できません。
  - Views
      `ryu`.`kn0430`
      --------------
          The generated SQL has to be checked manually.

Oracle側の上記Viewの中身は下記のとおりです。
select
a.db_chanel,a.db_tiku,a.db_jigy,b.db_soshiki3,a.db_line,a.db_class,a.db_kensu
from ryu.kn0410 a, ryu.md3mast b
where a.DB_LINE = b.DB_LINE(+)
