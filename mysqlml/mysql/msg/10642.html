MySQLが使いやすいビジュアルなデータベースのツールをリリース

2004年12月8日

新しいGUIツールでもっとも人気のあるオープンソースのデータベースの市場を更に開拓

カリフォルニア州、クプティノ市発 ?

もっとも人気のあるオープンソースのデータベースを開発会社の
MySQL ABはデータベースの開発者と管理者を手助けする
グラフィカルなクエリと管理ツールを発表しました。新しい
MySQL クエリブラウザーとMySQL アドミニストレータは
MySQL Developer Zone（http://dev.mysql.com/）からダウンロード
できます。

MySQLのマーケティング担当の副社長のZack Urlocker
は「MySQLは常にリレーショナル・データベースサーバーを
使いやすくするという評判を得てきました。このグラフィカル
なクエリ・ブラウザーとアドミニストレータをリリースして、
更に広範囲の企業とコミュニティーのユーザーの皆様
にオープンソースの技術を提供することができます。
こういったツールを使えば、クエリと管理をする際に
ユーザはよく知られているルックアンドフィールで
容易に作業ができます。

MySQLクエリ・ブラウザー

MySQL クエリ・ブラウザーはグラフィカルな環境でクエリを
生成、実行、そして最適化するのに使用できます。これが
あればMySQLのデータベース・サーバーに格納されている情報
により簡単にアクセスし解析することができます。

MySQLクエリブラウザーは 統合したツールでドラッグ・アンド・ドロップ
の機能を持ち良く知られたウエブ・ブラウザーのような環境です。

    * クエリ・ツールバーとデータベース・エクスプロアーがあれば
クエリのために簡単にテーブルやフィールドを選択できます。
    * テーブル・エディターはビジュアルにデータベースのテーブルを生成・
変更・削除します。
    * 結果ウインドウではユーザは複数のクエリを比較・管理できます。
    * オブジェクト・ブラウザーでユーザはデータベースとブックマーク
の管理ができます。
    * スクリプト・エディターは大きなSQLのスクリプトを生成・編集・デバッグ
するインタフェース提供します。

MySQL アドミニストレーター

MySQL アドミニストレータはデータベース管理者が簡単に
どのようにデータベースが作動しているかの管理や監視できます。
初めてビジュアルでデータベースのオペレーションを扱うこと
ができます。これにはサーバーのコンフィギュレーション、ユーザ管理、
リプリケーションの状態監視、バックアップ・リストアーとログビュー
を含みます。

カナダのロックポートの独立系のソフトベンダー(ISV) の社長のAdam Majer
は「アドミニストレータはデータベースの管理を今までにない透明さ
で可能にしました。こういったツールがあれば、もはやデータベース
の管理や監視は職人技ではなくなります。以前はエキスパートのみ
が知っていた性能や信頼性の機能をこれで得ることができます。」

プラットフォームと価格

MySQLクエリブラウザーと MySQL アドミニストレータはLinuxとウインドウ用
があります。

MySQL データベース・ソフトウエアーは２重ライセンスのモデルで
提供されます。オープンソースのライセンス（GPL)かコマーシャル
ライセンスで使用できます。 MySQLアドミニストレータのコマーシャル
ライセンスのスイートは以下を含み開発者・ユーザに付き$295です。
(MySQL クエリブラウザーとMySQLアドミニストレター Administrator)

ソフトのダウンロードやその他の情報は以下へどうぞ。
http://www.mysql.com/products/query-browser/ と
http://www.mysql.com/products/administrator/.
