MySQL が動作している Linux マシンの /etc/hosts ファイルに
Windows マシンの IP アドレスは書き込まれてるでしょうか？

僕もはたなかさんと同じように、user テーブルに新しいユーザを
追加したのに、MySQL にアクセスできない状況になりました。

/etc/hosts に
192.168.1.xxx win-client
(xxx は Windows マシンの IP アドレス)と追記してみてください。

僕の場合はこれを書き込むことによって、Windows マシンから TCP/IP 経由で
MySQL に接続できました。
