もりです。

  非常に簡単な文字化けチェックプログラムを以下に置いておきました。
  http://gadgetsoft.web.fc2.com/soft/chkjcode.zip

  Windows用のソフトで、自分用に作成したものですが、JIS X 0201と
JIS X 0208の中で、どの文字が文字化けするのかをチェックするために作りま
した。
  クライアント文字コードとしてcp932とSJISがテストできればよかったので
特にUTF8などを表示やテストできるようにはしていません。サーバー側のコー
ドは何でもかまいません。
  とりあえず、ADO接続できるDBならどれでも使えるとは思いますが、今の所
MySQL, PostgreSQL, SQL Server, ORACLEしかテストしていません。
  詳細はreadme.txtを読んで頂くと分かると思いますが、簡単なテスト用テー
ブルを作成して、JIS X 0201、JIS X 0208をINSERTし、それをSELECTして文字
化けが発生するかどうかを見ているだけのものです。
  以前、ODBC経由で文字化けが発生するという投稿がありましたが、そういう
時のチェックに少しでも役にたつようでしたら使って下さい。
  チェック用に作っただけのものですので、テスト用のデータベースで使って
下さい。本番用DBでは使わないようお願いします。
