> MySQL-client-community-5.1.31-0.rhel4.i386.rpm
> MySQLサーバーを再インストールしたりしていますが、解決方法が

SQLノード側にインストールされているパッケージはMySQL Clusterのものではありません。
おそらくndbclusterなどのオプションを認識できないので起動できないのではないかと。

クラスタのところにリストアップされているのが、まさにSQLノードに必要なパッケージです。

データノードには
Cluster storage engine

管理ノードには
Cluster storage engine management
Cluster storage engine basic tools
Cluster storage engine extra tools

などが必要です。
