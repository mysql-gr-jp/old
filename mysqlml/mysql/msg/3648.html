とみたです。

MySQL と Ruby で POP サーバを実装してみました。

ユーザ情報は MySQL で管理します。つまり OS のユーザ情報には依存しません。
また、メールデータもすべて MySQL 内に保持します。

	http://www.tmtm.org/ja/mysql/tpops-0.1.tar.gz

これは一応ドキュメントつきです。
