On Thu, 5 Feb 2004 21:56:13 +0900
Hidesuke Shimada <hide@xxxxxxxxxx> wrote:

> CMライセンスを購入して適用したらGPLが消えるのかというと、私は消え
> ないが正しいと思います。なぜならGPLはMySQLの使用条件なので使用する
> のを止めない限り有効と考えられます。

CMライセンスの MySQL と GPL の MySQL は別物だと考えた方が自然じゃない
でしょうか。

CMライセンスで MySQL を購入した場合は、その MySQL は GPL ではない。そ
のため、改変してソース非公開で配布しても問題ない …と思うのですが。

> CMライセンス購入者は著作権者
> ないしそれを代表または代理するものから購入しているので優先適用の
> 特権を与えられているに過ぎないでしょう。従って、CMライセンス購入者
> であっても新しいコネクションツールやMySQL自体に新しい機能などを開発
> してGPLで公開することは差し支えないと考えます。個々のケースでどちら
> を選択するかはユーザーの側に任されていると考えて良いのではないで
> しょうか。

「CMライセンス購入者」と特別に見るのがおかしいような…。CMライセンス 
MySQL を購入しても、それは MySQL という一つの製品を購入しただけなので、 
GPL MySQL の利用/改変/配布になんら問題はないですよね。
