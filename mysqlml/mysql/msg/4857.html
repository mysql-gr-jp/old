> ほとんどのPerlモジュールに共通することですが、perl Makefile.PLを実
行するときにオプションを指定しないと、Perlの標準のライブラリのディレ
クトリにモジュールをインストールします。ですので、どの資料を見ても
rootでの設定方法しか載っていないんだと思います。
>
> root以外のユーザでインストールする場合にはそのユーザで書き込みでき
るディレクトリにインストールし、あとはuse libなどで、インストールし
たディレクトリからモジュールを読み込むようにすればOKです。具体的には
>
> % perl Makefile.PL PREFIX=~/perl
> % make
> % make test
> % make install
>
> 等としてあげればいいと思います。
