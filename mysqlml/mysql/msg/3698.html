うちの会社の環境では問題なくアクセスできてますけど・・・

172.17.88.72のマシンにあるデータベース(DB1)に
テーブル(table1)を作成し172.17.88.71側から参照する例

172.17.88.72側の設定
test% mysql -uroot DB1
mysql> create table table1(field1 char(10), field2 int);
mysql> insert into table1(field1,field2) values('斉藤',38);
mysql> insert into table1(field1,field2) values('藤本',32);
mysql> insert into table1(field1,field2) values('大久保',21);
mysql> grant select on DB1.table1 to em@xxxxxxxxxx identified by 'pass';
		(172.17.88.71のemユーザーに対してselectの許可を与える)

172.17.88.71側の操作
[em@eztest em]$ mysql --host=172.17.88.72 -uem -ppass DB1←172.17.88.72側へ接続
mysql> select * from table1;
+--------+--------+
| field1 | field2 |
+--------+--------+
| 斉藤   |     38 |
| 藤本   |     32 |
| 大久保 |     21 |
+--------+--------+
3 rows in set (0.00 sec)
mysql>
