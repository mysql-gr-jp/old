根本的な解決策でなくて申し訳ないですが，参考までに・・・

MySQL+PHP3 の組み合わせでとりあえず動かしたい場合， Caldera 
OpenLinux eServer2.3 が非常に楽でした。雑誌付録の FTP 版しか試し
てないですが，インストールタイプを「 Web サーバ」に指定してインス
トールすると， MySQL も PHP3 もインストールされちゃいます。 (^^;)

そのままインターネットへ常時接続するのはちょっと不安ですが，イン
トラネットで素早く MySQL サーバを立ち上げたい時はとても楽でした。
Webmin という Web ベースの管理ツールもついています。

ちなみに，入っているバージョンは，

	MySQL 3.22.25
	PHP 3.0.12-i18n-beta4
	Apache 1.3.9

です。しかし個人的には，実運用機では LASER5 Linux6.0 Rel2 を使い
PHP3 や MySQL はソースからコンパイルしていますが・・・
