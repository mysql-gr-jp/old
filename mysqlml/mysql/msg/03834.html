西田と申します．
初めて投稿させていただきます．
（こちらへの投稿が場違いでしたらお許しください）

現在，VC++からMyODBC経由でMySQLサーバに接続するアプリを
作成しています．

しかし，レコードの参照はできたのですが，更新や削除を行うと，
「レコードセットは読み取り専用です」
というメッセージが出て更新や削除ができません．
（新規追加はできます）

どうかご教授よろしくお願いいたします．

--- VC++コードの抜粋 ----------------------------

// データベースのオープン（更新可，カーソルライブラリは使用しない）
m_database.Open(NULL, FALSE, FALSE, m_pSet->GetDefaultConnect(), FALSE);
// レコードセットの構築
m_pSet = new CXXXRecordSet(&m_database);
// レコードセットのオープン
m_pSet->Open(CRecordset::snapshot, NULL, CRecordset::none);
// 先頭レコードをカレントレコードにセット
m_pSet->MoveFirst();
（ここで，レコードの参照は可能）
// レコードセットの削除
m_pSet->Delete();        <--- 「レコードセットは読み取り専用です」

[OS]     WindowsNT 4.0
