最終的には

select field1, field2, field3, field4 as f4
  into outfile 'temp' from table1 group by field1 order by f4 desc;

delete from table2;
load data infile 'temp' into table table2;

select field1, field2, field3, field4
  from table2
    group by field1;

というやり方にしました。グループ化された場合は先頭のレコードが表示される
という前提です。特にどこかに書いてあったわけではないのですが、なんとなく
そんな感じでしたので。
