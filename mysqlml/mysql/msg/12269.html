なので、どこででも実行出来るはずです(^^;

> 「’ｎｅｔ’は、内部コマンドまたは外部コマンド、操作可能なプログラムまたは
> バッチファイルとして認識されていません。」

って事は、パスが通ってないのかな？
%systemroot%\system32\net.exe start mysql
ではどうでしょうか？

> マニュアルを読んだのですが、そのあたりが書いていなかったものですから。

マニュアルですかぁ。。。Windows のヘルプとかぁ・・・(^^;
help net ってすれば出ます。
あ、これももしかしたら
%systemroot%\system32\help.exe net
かも。

とりあえず、マイコンピュータを右クリック、プロパティから「詳細設定」
タブ、「環境変数」ボタン「システム環境変数」か、「ユーザ環境変数」の
PATH を調べてみて、%systemroot%\system32 が入っていなかったらいれて
おきましょう。

とりあえず、net start コマンドは「スタート」－「設定」－「管理ツール」
－「サービス」の中の当該サービスの実行で、 net stop は停止を実行する
コマンドです。ですので、画面から行うなら、サービスの起動をすれば良い
と思います。

ちなみに、リスタートは
net stop mysql && net start mysql
で出来ます。tips ですね(^^)
