Subject: [mysql 11416] MaxDBの7.6.00.09 (beta)がリリースされました。


MaxDBの7.6.00.09 (beta)がリリースされました。
以下よりダウンロードできます。
http://dev.mysql.com/downloads/maxdb/7.6.00.html.
バグ修正と新しい機能が含まれます。

その一部です。

* DB procedure はLONGデータタイプのパラメータをサポート
* CREATE USERとALTER USERステートメントでDISABLE/ENABLE CONNECT
パラメータをセットすることで、それぞれのユーザとユーザー
のグループがデータベースにアクセスすることを防げます。
* JDBCのSELECT/FETCH オプティマイゼーションが改善
* SQLDBC APIメソッドを実行してSQLDBCトレースを活性化します。
* デフォルトでForwardのみのカーサーやSELECT/FETCHオプテイマイゼーション
をセット

変更の詳細は
http://dev.mysql.com/doc/maxdb/changes/changes_7.6.00.09.html

技術的な問題でMicrosoft Windows (32 bit)用のphp4-と
php5-dlls はクライアントのダウンロードのページには一時的に表示されません。
以下のftpサーバーからmaxdb-php-v4-win-32bit-7_6_00_09.tgzと
maxdb-php-v5-win-32bit-7_6_00_09.tgzをダウンロードしてください。
ftp://ftp.sap.com/pub/maxdb/current/7.6.00/

Microsoft Windows (64 bit) 用のPython のパッケージ
はまだ7.6.00.09版にアップデートされていません。
