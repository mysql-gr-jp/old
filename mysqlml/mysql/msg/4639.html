Satoshi Iwanaga <iwanaga@xxxxxxxxxx> wrote:
 |
 | 現在MySQLを/usr/local/mysqlに置いているのですが、
 | /usr の容量が100%になってしまいました。
 | ここで質問なんですけれど、MySQLを/usr 以外に移動させることはできるのですか？
 | 例えば/usr2 とかに・・

できるとは思いますが、いろいろ設定ファイルや起動ファイルを変える必要が
あって面倒だとか自分自身がどのファイルを書き換えればよいか把握できない
なんてことがあるなら、単純にシンボリックリンクを張ってしまえば？

# cp -p /usr/local/mysql /usr2/hogehoge/boeboe/mysql
# mv /usr/local/mysql /usr/local/mysql.old
# ln -s /usr2/hogehoge/boeboe/mysql /usr/local/mysql

試したわけじゃないので無保証です :-)
＃ cp -p じゃなくって tar とかした方がいいのかもしれないですし。

もしディスクを増設することが可能なら、大きなディスクを買ってきて、
/usr/local 以下を全部そっちに移すとかの方が根本的かもしれないですね。
