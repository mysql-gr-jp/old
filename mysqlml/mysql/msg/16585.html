お世話になります。

 > 
WorkbenchにServerが実際に使っているものと同じものを指定する方法がわかりません。
と言ってましたが、自己解決しました。

WorkbenchのDatabase/Manage Connections...をクリックし、Manage Server 
Connections画面を開く。
次に、アカウントを選択し、System ProfileタブのConfiguration 
File欄に使用するiniファイルをパス付で指定する。
ここで指定されているiniファイル名が下記の表示に反映されました。

MySQL WorkbenchのAdministrationタブのServer StatusでConfiguration FileMySQL 
WorkbenchのAdministrationタブのOptions File画面最下部に表示されるConfiguration 
File
Option FileのGeneral/Directriesの表示

ありがとうございました。
