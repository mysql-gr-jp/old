Subject: [mysql 04434] Access denied for user: '@localhost'

TurboLinux Server 6.5 に MySQL-4.0.6 をソースからコンパイルしてインストール
しまし。
configureの指定は、次の通りです。
#!/bin/sh
./configure \
--with-pgsql \
--with-mysql \
--enable-track-vars \
--with-apxs=/usr/sbin/apxs \
--enable-versioning \
--without-gd \
--enable-trans-sed

一般ユーザーで mysqladmin create foo としてデータベースを作成しようとすると
次のエラーメッセージが表示されます。
"mysqladmin: Create failed; error: 'Access denied for user: '@localhost' to
database 'user"

スーパーユーザになって同じことをするとちゃんと作成できます。

スーパーユーザーで作成したデータベースに、一般ユーザーでアクセスしようとして
"mysql foo"とすると次のエラーメッセージが出力されます。
”ERROR 1044: Access denied for user: '@localhost' to database 'foo'”

一般ユーザーでデータベースを作成／アクセスすることは出来ないのでしょうか？
