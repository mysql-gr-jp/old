こんばんは。Mroongaプロジェクトの堀本です。

Mroonga 9.12をリリースしました！

リリースアナウンス:
   http://mroonga.org/ja/blog/2020/01/29/mroonga-9.12.html

MroongaはMySQLで日本語全文検索を実現するストレージエンジンです。高速で
あることや位置情報検索をサポートしていることなどが特徴です。詳細につい
はドキュメントをご覧ください。

  * Mroongaの特徴 ― Mroonga v9.12 documentation
    http://mroonga.org/ja/docs/characteristic.html#what-is-mroonga

変更点:
   http://mroonga.org/ja/docs/news.html#release-9.12

さて、今回の主な変更点は次のとおりです。

  * [CentOS] MySQL 5.6.47, 5.7.29, 8.0.19をサポートしました。
  * [CentOS] MariaDB 10.1.44、10.2.31、10.3.22、10.4.12をサポートしました。
  * MySQL 8.0.19のソースを使ったビルドがエラーになる問題を修正しました。
  * [CentOS] Percona Server 5.6.47をサポートしました。
                                                                           
それでは、Mroongaでガンガン検索してください！
