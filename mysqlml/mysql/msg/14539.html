> ここで、 cat_1st をキーとしてソートしたいのですが、
> 単純に ASC や DESC でソートするのではなく、
> 特定の順番で並べたいと考えています。
> 
> 例: 'P' → 'A' → 'N' → 'C' → ... → 'NO'
> 
> このようなソートを、MySQLだけで実現できるのでしょうか？

5.0でしか試してませんが、
あまり数が多くなければCASE文でできると思います。

mysql>
select * from example order by
case cat_1st
 when 'P' then 1
 when 'A' then 2
 when 'N' then 3
 when 'C' then 4
 else 100
end;

+---------+
| cat_1st |
+---------+
| P       | 
| A       | 
| N       | 
| C       | 
| NO      | 
+---------+
5 rows in set (0.00 sec)
