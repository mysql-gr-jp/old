こんにちは。
Mroongaプロジェクトの林です。

Mroonga 9.03をリリースしました！

リリースアナウンス:
   http://mroonga.org/ja/blog/2019/05/29/mroonga-9.03.html

MroongaはMySQLで日本語全文検索を実現するストレージエンジンです。高速で
あることや位置情報検索をサポートしていることなどが特徴です。詳細につい
はドキュメントをご覧ください。

  * Mroongaの特徴 ― Mroonga v9.03 documentation
    http://mroonga.org/ja/docs/characteristic.html#what-is-mroonga

変更点:
   http://mroonga.org/ja/docs/news.html#release-9.03

さて、今回の主な変更点は次のとおりです。

  * より大きなデータに対してインデックスを作成できるように INDEX_LARGE フラグを
    サポートしました。COMMENT に 'flags INDEX_LARGE'を指定すると有効にできます。
  * 最新のMySQLやMariaDB、Percona Serverをサポートしました。
    * MySQL 5.6.44, 5.7.26.
    * MariaDB 10.1.40.
    * MariaDB 10.2.24.
    * MariaDB 10.3.14, 10.3.15.
    * Percona Server 5.6.44. 
  * MariaDB 10.1/10.2をバンドルしたWindows版パッケージの提供もはじめました。
    (これまでは10.3の1系統のみの提供をしていた)

それでは、Mroongaでガンガン検索してください！
