"ochiai" <s.ochiai@xxxxxxxxxx> wrote:

> 既にデータが入っているフィールドのキャラクタセットは
> 変更できないのかもしれません。

MySQLのちょっとした罠ですが、キャラクタセット変更は
テーブルではなくてカラムに対してかけます。

× ALTER TABLE ta DEFAULT CHARACTER SET = cp932;
○ ALTER TABLE ta MODIFY c1 VARCHAR(100) CHARACTER SET cp932;

テーブルに変更をかけても、すでにあるカラムには変更がかかりません。
いろいろ面倒ですので作り直すのが無難だとは思います。
