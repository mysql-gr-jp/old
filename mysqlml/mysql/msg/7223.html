> 式にWebサーバ上のMySQLに登録しようと思うのですが、ローカル上のMySQL内のデー
> タベースをWebサーバ上のMySQLへコピー（と言う表現があっているかは判らないんで
> すが）する方法はあるのでしょうか？

私は

$ mysqldump -h 127.0.0.1 -A | mysql -h web-server

とか

$ mysqldump -h 127.0.0.1 -B database | mysql -h web-server

とかを使うことが多いです。
＃本当は
＃1)DBごとにダンプ
＃2)tgzに固める
＃3)コピー先に持っていく
＃4)展開する
＃5)DBごとにmysqlに突っ込む
＃とすることが多いですが。
