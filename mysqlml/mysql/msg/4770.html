> > 項目を増やすとうまくいきません。
> > 例えば、第一ソートを「年齢」とし、同じ年齢なら
> > 「身長順」とかしたい場合どうしたらいいのでしょう。
> > order by name order by height　ではだめなのでしょうか。
> > それともmysqlでが、複数の項目によるソートは出来ない

>
> order by name,height
>
> です　カンマで区切ります
>
> 名前逆順　身長順なら
> order by name desc,height
> とかって書きます　
>
> MySQL-J DOC より引用
>
> 出力用に選択されたフィールドは、ORDER BY節やGROUP BY節において、フィール
> ド名、フィールドのエイリアス名、又はフィールド番号にて指し示すことができ
> ます。フィールド番号は1から始まります。
>
> mysql> select college, region, seed from tournament
>            ORDER BY region, seed;
> mysql> select college, region AS r, seed AS s from tournament
>            ORDER BY r, s;
> mysql> select college, region, seed from tournament
>            ORDER BY 2, 3;
>
> 逆順で並べたい場合には、 ORDER BY 節の中で、あなたが並べたいと思っている
> フィールドの名前の後ろに、 DESC (descending) キーワードを追加します。デ
> フォルトは昇順です； これは ASC キーワードを指定したことになります。
