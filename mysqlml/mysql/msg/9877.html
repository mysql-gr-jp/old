MySQLでは、
　LOAD DATA でリモートPCから取り込めますが
　SELECT ... INTO OUTFILE はローカルに書き出すものです。

私の場合、Windowsのコマンドラインから
mysql -h XX -u UU -pPP DNAME < SCR.sql > d:\temp\dfile.csv
などとしています。

mysql -e "..." > ...

とかも使えます。
