ぢょ〜さん、ご返信有難う御座います。

>       configure のオプションにもよると思いますが，通常 PHP 4 では PHP
>     に同梱の libmysqlclient を，PHP 5 ではシステムでインストールされ
>     ている libmysqlclient を使うようになるでしょう．
> 
> #PHP 5 では libmysqlclient は付属してません

今となっては OS 再インストールをしてしまったので configure
オプションを正確には思い出せませんが、MySQL に関しては確か
以下を指定したと思います。

--with-mysql ではなくて --with-mysql=path/to/mysql

確認で、再度 PHP ソースを落としてきて ./configure --help
を確認してみると、

--with-mysql[=DIR]
Include MySQL support. DIR is the MySQL base directory.
If unspecified, the bundled MySQL library will be used.

という事なので、
なるほど、[=DIR]が余分だった！
と自分なりに納得してみました。
