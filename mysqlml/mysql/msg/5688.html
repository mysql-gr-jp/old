> Warning: Host 'localhost is not allowed to connect to this MySQL server
> Warning: MySQL Connection Failed: Host 'localhost' is not allowed to
> connect to this MySQL server
> Warning: MySQL: A link to the server could not be established
と、いわれているようなので、ネットワークが接続禁止になっているようなきがし
ます。my.cnf
に（Windowsの場合はmy.ini）

skip-networking

が設定されていないでしょうか。これがある場合はネットワーク接続を認めないよ
うです。これを
#skip-networking
とコメントアウトしてみてください。

どうでしょうか。
