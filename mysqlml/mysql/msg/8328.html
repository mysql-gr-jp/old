Subject: [mysql 08328] カタカナの曖昧検索

はじめまして。黒田と申します。

mysql 4.0.16-ntでの動作についてなのですが、

create table test2(name varchar(8) binary not null);
insert into test2 set name='クロダ';
insert into test2 set name='クロカワ';

のようにしたあと、以下のように曖昧検索をかけると

mysql> select * from test2 WHERE name LIKE binary '%クロダ%';
+----------+
| name     |
+----------+
| クロダ   |
| クロカワ |
+----------+
2 rows in set (0.00 sec)

のように、'%クロ%'と同じ結果が返されてしまい、「ダ」が無視されています。
文字コードは
default-character-set=ujis
のようにしています。

過去ログを拝見しましたところ、binaryにしたり文字コードを変えることで対応できる、
と書いてあったのですが、このように試してみてもダメでした。
何か間違っているのでしょうか？
