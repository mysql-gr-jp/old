# たまには代表っぽいことを…

ご存じの方もいると思いますが、OSDN 主催で 2/14-15 に Open Source DB
Conference が開催されます。

	http://osdn.jp/event/db2002/

MySQL AB もスポンサーになっていて、MySQL AB の David Axmark の講演もあり
ます。このチュートリアル(有料)の枠で MyNA として講師をやらないかという話
が来ています。チュートリアルは 90分間 x ２枠です。

そこで、チュートリアルのテーマ および チュートリアルの講師 を募集します。

「このようなテーマのチュートリアルなら聴講したい」とか「このような場で発
表したい」とか、ご意見くださいませ。ML で表明するのに抵抗があれば、個人
宛にメールください。

講師として 90分間がキツければ、一つの枠を複数人でやるというのもOKみたい
ですので、「30分間だったらやっても良い」というのも歓迎です。

また、チュートリアルの後に MyNA としての枠(90分 無料)もあります。BOF が
良いかと思うのですが、他に何か良いネタがあればご意見ください。

たとえば、MySQL を使った小ネタの発表会とか、MySQL でこんな失敗しましたの
暴露会とか…。

なにぶん、MyNA としてのイベントは初めてのことなので、「BOF とはどういう
ものか」も良くわかってないです (^^; フリートークのこと？

今のところ、MyNA にはイベント担当のスタッフというのもいません。BOF の進
行役とか仕切ってくれる人がいればありがたいです。

ちなみに MySQL のチュートリアルは 2/15 です。その前後に MyNA オフ会をや
るのも良いですね。これも誰かが幹事をやってくれれば…(^^;
