>>> クラスター全体が落ちてしまっているので、
>>> --nowait-nodesで起動しようとしましたが、
>>> 以下のようなエラーが出てしまいやはり起動できませんでした。
>>
>> エラーはNode3から発生しているので、--nowait-nodesで指定するのは3です。
>> Node 3以外を--nowait-nodes=3で起動して下さい。
>> Node 3はひとまず起動しないでください。
>>
>>> この場合、--initialオプションを実行するしかないと思うのですが、
>>> DBのデータも初期化されてしまうのでしょうか？
>>
>> はい。
>> --initialオプションをつけるとデータが初期化されます。
>>
>> この手のエラーは設定を小さな値に変更した場合にも出ることがあります。
>> MySQL Clusterでは、MaxNoOfTablesなどの各種設定値を動的に縮小できるように
>> なっていませんのでご注意下さい。（値を増加することは可能です。）
