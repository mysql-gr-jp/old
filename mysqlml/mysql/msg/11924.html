新しいディユアル コアテクノロジー付きのDellのサーバーは
PowerEdge 850に関しては79%の性能増加、PowerEdge 830に
関しては60%の増加を見込めます。 MySQLとJBoss を提供
することで、Linuxのサポートを増加

テキサス州、Round Rock市　2005年８月８日

Dell は今日の発表でインテルのディユアル コアテクノロジー付き
の新しい２つのエンタープライズ用の製品を紹介しました。
PowerEdge 830とPowerEdge 850 サーバーは２つのMPUの性能と
コンピュータ力を１つのソケットに擁し、より多くのコンピュータ力を
必要とするユーザに手ごろな解を与えます。どちらのシステム
も前のモデルの２倍のメモリーを提供し、更に３つの異なった
MPUの選択、投資に対して見合う管理機能を提供します。

PowerEdge 830 は１ソケットのタワー型でワークグループやリモート
オフィースや小企業に向きます。ウエブ、インターネット・アクセス、
メッセージング、バックオフィース、やプロダクティビティ
のアプリなどです。更に、ディユアル コアテクノロジーで
PowerEdge 8001の62%増しの性能を提供できます。

PowerEdge 850 は１ソケットの 1U (1.75 インチ) ラックマウント
のサーバーで、小企業やコストを低く押さえたいデータセンタなど
に適します。ウエブのインフラ・サーバーに適し、スケーラビリティ
と将来の性能増加を見込むユーザーのために次世代の
技術を使用します。ディユアル コアテクノロジを使い PowerEdge 850は
79%以上の性能増で、ウエブ・サーバーでは前の版のPowerEdge 7502
よりも26％増加になっております。

DELLの副社長のNeil Handは「Dellは成長し続ける会社を
支援するために投資を守りサポートします。ディユアル コアテクノロジ
やPowerEdge 830 や850にあるその他の高度な機能でカスタマーの
ＩＴへの投資を現在も将来も守ります。」と語った。

こういったシステム上の高度な機能はカスタマーの幅広いデマンド
に答えることができます。PowerEdge 830 はオプションでホットプラグ
SCSI RAID アレー、機能強化されたPCI-Express と
サイズを小さくして保守やインストレーションを軽減。PowerEdge 850
は新しい高性能のPCI-Express アーキテクチャでディユアル
エンベッデドのギガビットの NICを含む。

そのほかのPowerEdge 830 と 850 特徴は:

* 2.53GHz インテル Celeron か2.8 か 3.6 GHz Intel
P4 （800MHz ）フロント・サイド・バスまたは 3.0/3.2GHz インテル Pentium
D ディユアル コアMPU;
* インテル E7230 チップで将来のプロセッサーをサポートできる
1067MHzまでスケールアップできるフロントサイド・バス
* 8GB までの DDR-2 SDRAM
* オプションの SATA ドライブまたはU320 SCSI ドライブ

MySQL/JBoss 搭載の完全なLAMP スタック

同時にDellのカスタマーには LAMP (Linux, Apache, MySQL and PHP/Perl)
のキーの部品を提供できます。LAMP スタックはバンドルした
Linuxのソフトのパッケージを提供します。MySQL Network サブスクリプション
とJBoss Network サポート・サブスクリプションはDELLから
直接提供されます。カスタマーは今やDELLからエンドーツーエンド
のLAMPのソリューション・スタックを購買できます。このソリューション
は PowerEdge サーバー、Red Hat Enterprise Linuxまたは
Novell SUSE Linux Enterprise Server 、 MySQL データベース
アプリ, MySQL Network, JBoss アプリケーション・サーバーと
JBoss Networkです。

Dell PowerEdge上でMySQL を使用したアプリの情報は
www.dell.com/mysql. JBoss Networkの更なるサポートと価格情報は
www.dell.com/jboss.

簡単な管理、簡単に所有

PowerEdge 830と850の 特別な柔軟性と性能、更にDELLが
提供する管理ユーティリティ、ソフトウエア、とプロフェッシオナル
サービスでTCOは小さくなります。どちらのサーバーもDELLの
OpenManage スィートが搭載されています。これを使えば
使い勝手の良い、セキュリティ、変更管理、簡素化された
モニター機能を入手でき、サーバーを簡単にインストールできます。
OpenManageのツールはウエブ・ベースのインタフェースで
簡単にＩＴのリソースにアクセスできます。詳細はwww.dell.com/openmanage

Dellはまたリモートサイトへの展開とサポートを容易にする
包括的なサポートを提供します。カスタマーはＯＳをプリインストール
して購入できます。またはカスタムでソフトやハードをインストール
して出荷前にテストすることもできます。Dell はサーバーインストレーション
サービスを提供します、またシステムのアップタイムを増加する
ためのエンタープライズ・サービスを提供します。詳細は www.dell.com/services.

価格情報等

Dell PowerEdge 830と PowerEdge 850 は今月よりそれぞれ$699と $749から
お求めいただけます。どちらもWindows Server 2003, Red Hat and SuSE Linux
をサポートします。Dellのサーバー、サービス、ツールの情報は
www.dell.com/servers.
