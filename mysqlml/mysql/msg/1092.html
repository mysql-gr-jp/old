3.23 の機能は結構よさそうなものがおおいです。(まだバグありますが)

このほかに嬉しくなる点は

 ファイルを読み込み関数 load_file()
   (バイナリを項目にinsertする時使えて便利)

 INSERT 文で SELECT が使用できる
   INSERT ..... SELECT ...

 INSERT 文で、values をコンマ区切りで羅列
   INSERT INTO ... VALUES (....) , (.....)

 あとは MyISAM が 4GBytes 以上をサポートしている事。

 使えそうでしょう(^^)
