とみたです。

ん～ SELECT は「存在しない行」は選択できませんからねぇ…。

次のように、調べる範囲の数をすべて含んでいるテーブルを作っておいて、

	mysql> select id from fuga;
	+------+
	| id   |
	+------+
	|    1 |
	|    2 |
	|    3 |
	|    4 |
	|    5 |
	|    6 |
	|    7 |
	+------+

結合すればできますね。

	mysql> select fuga.id from fuga left join hoge using(id) where hoge.id is null;
	+------+
	| id   |
	+------+
	|    4 |
	|    6 |
	+------+
