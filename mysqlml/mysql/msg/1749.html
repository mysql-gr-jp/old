MySQLを ODBC接続しようとしてうまくいかず、MySQLに mysqlツール

削除したりしているうちに mysql ツールが止まってしまいました。

システム全体をリブートしたところ、 mysqld が立ち上がりません。

>Found old style password for user 'php'. Restart using --old-protocol

とあったので、 safe_mysqld --old-pr & で起動すると立ち上がります。
今までは safe_mysqld -l & で起動していたのに変です。

ODBC接続に関しては、自分の間違いに気が付いて問題なく接続できる
のですが、上の --old-protocol の問題は解決しません。
マニュアルを見て isamchk を走らせましたが、--old-pr でないと
立ち上がりません。

問題なく稼動はしていますが、気持ち悪いので元に戻したいのですが
ぜひ力を貸してください。

UNIX socket             /tmp/mysql.sock
