こんにちは。池田です。

Tritonnプロジェクトという名前でMySQLに日本語全文検索機能を
追加する開発をしているのですが、たまには宣伝をさせて下さい。

公式サイト　http://qwik.jp/tritonn/

# MySQL UC 2007 Tokyoにて講演も行ったプロダクトです。

この度、Tritonn 1.0.7（with MySQL 5.0.45)をリリースしたのですが、
今回からLinux用のバイナリファイルもリリースするようになりました。

以下のプラットフォーム向けのバイナリがあります。

* Linux 32bit(x86)向けのRPMバイナリパッケージ
* Linux 32bit(x86)向けのtarballバイナリパッケージ
* Linux 64bit(Intel64/AMD64)向けのRPMバイナリパッケージ
* Linux 64bit(Intel64/AMD64)向けのtarballバイナリパッケージ

コンセプトは「MySQLの利便性を損ねることなく全文検索機能のみAddOnする」
というところにありまして、MySQL本家のバイナリと同様、tarball版は
展開するだけですぐに使えます。RPM版もrpm -iするだけです。

内部的にはSennaとMeCabを使っているのですが、SQLインタフェースへの
影響は微小なので気にならないレベルになっています。

ぜひお試し下さい。

尚、なんでわざわざMySQLの改造を行っているかというと、MySQLで
日本語全文検索ができないのが理由なのですが、詳しくは以下を
ご覧下さい。

http://qwik.jp/tritonn/about.html

もちろん、MySQLがOSSだからできることでもあります。
OSS万歳。
