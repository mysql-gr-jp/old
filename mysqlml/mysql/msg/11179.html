MySQL Cluster はまだまだ発展途上の技術で、上記の記事が出てからも
ずいぶん変わっています。今のところ 4.1.8 以降は同様の config
スタイルになっていますが、まだ Fix はしていないようです。

そういうわけで情報も不足していますが、本家の
http://dev.mysql.com/doc/mysql/en/ndbcluster.html
は英語ですが、そう難しくはありません。

とりあえず試す、ということなら
http://dev.mysql.com/doc/mysql/en/mysql-cluster-quick.html
だけでも十分だと思います。
試すだけなら MyNA のページにブータブル CD である
Knoppix があります。（トップにも載っています）
焼いてブートするだけで
MySQL Cluster が起ち上がります。

MySQL Cluster は、いまだ開発途上とはいえ、障害発生時、確かにシームレスに
切り替わります。ロードバランシングされているので、１台こけても
あいているマシンに振られ、復旧時に自動キャッチアップされ、
その面でもお手軽です。
