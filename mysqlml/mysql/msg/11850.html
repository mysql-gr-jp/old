oceanblue17> がでてしまいます。
<中略>
oceanblue17> # java -classpath
oceanblue17> mysql-connector-java-3.0.6-stable-bin.jar
oceanblue17> MyJdbcTester Exception in thread "main"
oceanblue17> java.lang.NoClassDefFoundError: MyJdbcTester

参照しているのが jar ファイルなので、-jar オプションが必要な
はずです。

java -jar -classpath mysql-connector-java-3.0.6-stable-bin.jar MyJdbcTester

と入力してみてください。
