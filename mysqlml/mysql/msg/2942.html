> SELECT の LIKE で、関係の無いレコードが検索されてしまう不具合が発生
> しました。

> 「value2 like '%ダ%'」に対して何故か「ミ」という文字が選択されてし
> まいます。もしかして LIKE は全角文字に対応してないのでしょうか？

たぶん charset が latin1 とかになってるんでしょう。
mysqladmin variables の character_set で確認できます。

mysqld 起動時に --default-character-set=ujis オプションをつけて
起動すれば、日本語EUCを正しく扱うことができます。
