製品
MySQL (4.1.2 Alpha)がリリースされました。
このリリースは開発リリースで新しい機能とバグフィックスを含みます：
* 文字セット変換とprepared statementプロトコルでの
MYSQL_TYPE_BLOB typecodeをサポート
* EXPLAIN SELECT ステートメントの出力に hidden SELECT of UNIONの
説明を加えました。
* mysql コマンドライン　クライアントが複数の 「-e]オプションをサポート
* 新しいmyisam_data_pointer_sizeシステム変数

詳細は
http://lists.mysql.com/announce/201
ダウンロードは
http://dev.mysql.com/downloads/mysql/4.1.html

MySQL Connector/J (3.1.2)がリリースされました。
このリリースはALPHAブランチ上のType-IV all Java JDBC の新しいリリースで
MySQL-4.1およびMySQL-5.0に適します。（MySQL-4.1.2とそれ以降の
版でサーバー・サイドのprepared statementを使用する
際には必要です。）

このリリースは：
* ストアード・プロシージャの引数パージングを引数のサイズ
（char(), varchar()）が指定された際の問題をフィックス
* 'cacheCallableStmts'を通じてcallable statementキャッシュをオンに
* ストアード・プロシージャに対して出力引数が指定されない
時、適当でないクエリが発生し出力引数を取り出そうとして、そのため
サーバーにシンタックスのエラーが生じる場合をフィックス

詳細は
http://lists.mysql.com/announce/202
ダウンロードは
http://dev.mysql.com/downloads/connector/j/3.1.html

MySQL　MaxDB 7.5.00.14がリリースされました。
この版はMaxDBプロダクション版の保守リリースで、以下を
含みます。

* SQL Studioはトレース・ウインドウに挿入・アップデート・除去で
影響を受けた行の数と、ステタスバーに最後に実行された
ステートメントのアップデートと除去を示します。
* inv_directを使うジョインは並行して実行できます。
* "check table catalog"コマンドは255以上の長さのコラムでスタック
の内容が正しいかもチェックします。

詳細は
http://dev.mysql.com/doc/maxdb/changes/changes_7.5.00.14.html
ダウンロードは
http://dev.mysql.com/downloads/maxdb/7.5.00.html
