はい、仰る場所にありました。

しばらくLinuxのコマンドを使っていなかったのですっかり忘れて
いたのですが、locateで探せたんですね．．．

で、うまくrootユーザで入れるようになりました。手順は、

1  su して 
2  /etc/rc.d/init.d/mysql に行って ./mysql stop で mysqld を停止
3  /var/lib/mysql 以下の *.frm を rm *.frm で削除
4  mysql_install_db を実行
5  サーバリブート

ここで何故か mysqld が自動起動されなかったので
6  ./mysql start で mysqld を start

この後サーバにログインし su して mysql を使ってみると

# mysql
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 1 to server version: 3.22.32

Type 'help' for help.

mysql> use mysql
Reading table information for completion of table and column names
You can turn off this feature to get a quicker startup with -A

Database changed
mysql>

といった感じで、この後うまくユーザを追加できました。

マニュアルの 6.11 MySQL権限許可の初期設定 の部分が参考に
なりました。以前読んではいたのですが、その時になると以外
と思い出せないものですね．．．
