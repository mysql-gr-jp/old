以下のphpスクリプトを動かしたら、「接続に成功しました」になりました。
phpinfo();ではインストールしたMySqlが表示されていないようですが、
これでPHPから使用できるということですね。
もう少し色々と勉強してみます。ありがとうございました。

<?php
$link = mysql_connect('localhost', 'mysql_user', 'mysql_password');
if (!$link) {
    die('接続できませんでした: ' . mysql_error());
}
echo '接続に成功しました';
mysql_close($link);
?>
