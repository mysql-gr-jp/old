私は、SCO OpenServer 5.0.2 + gcc ( 2.7 ) + SCO Development で
Mysql-3.21.21a-beta を、今年に入ってから、使用しています。
あまり、聞きなれない OS だと思いますが、AT 互換機の Un*xです。

Hardware は、Inte* 4*6 の 66 Mhz です。

Mysql の compile は、Documents どおりスムーズに行なえました。
( 民斗 <tommy@xxxxxxxxxx>の、日本語 patch を使用させていただきました。)

Benchmarks を実行するには、 perl が必要なので、perl5.004_04を install.
ついでに、apache_1.2.4にphp-2.0b12を追加して apache を install.

とりあえず、apache に access している log が mysql の DB
に取れているので
、動いているみたいです。
