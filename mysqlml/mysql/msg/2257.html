# MySQL3.23.15以降しか使ったことが無いので外しているかもしれません。

　RegHatでRPM版をインストールしたのであれば、
 /etc/rc.d/init.d/mysql
　が出来ていると思います。それを、

・起動
# /etc/rc.d/init.d/mysql start

・停止
# /etc/rc.d/init.d/mysql stop

　で可能だと思います。
（safe_mysqldだけでの起動はだめみたい？）

　もし、コンパイル(configure;make;make install)してインストールした場合は、
その場所（prefix）から
 shared/mysql/mysql.server
# デフォルトではprefixは、/usr/local になってます。
　のファイルが/etc/rc.d/init.d/mysqlと同様のスクリプトに成っています。
　これに実行権限を付けて、/etc/rc.d/init.d/mysql にシンボリックリンクを張れば
あとは同じように起動/停止が出来るはずです。
# ついでに、mysql_install_db を実行しないとだめかもしれません。

# コンパイルについての詳しい内容は、READMEファイルに載ってると思う・・・
