timestampは，レコードが変更された時間を記録するためのものな
ので，こういう用途にはマッチしません。

　DATETIME を指定して明示的に日付と時間を与えておけば永続的
な抽出条件として扱えます。

INSRT INTO tabele_name set DATE = NOW() , **** とか。

ALTER TABLE table_name modify DATE datetime ;
で変換できるんじゃないでしょうか？

　あと，カラム名にDATEは使わない方がいいと思います。
