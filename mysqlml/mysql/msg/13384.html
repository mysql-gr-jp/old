3.23て副問い合わせ使えないんじゃ?
副問い合わせ使えるなら
select count(*) from A
where Name not in (select Name from B);

副問い合わせ使えない場合は
select count(*) from A
left join B on A.Name = B.Name
where B.Name is null;

試してない。
