遠藤　俊裕 wrote:

> で、質問なのですが、データベースにはどうやってデータを入れ
> ればいいのでしょうか？^^;
> データの切り出し等は Perl を使用するつもりです。

perl から直接 mysql  をいじれるモジュールがあるんですが、
それは詳しい人に。perl じゃなく、awk とかでもいいですが
切り出されたテキストがタブ区切りで、テーブルのフィールドと順序が
あってる場合、

mysql データベース名

と言うコマンドを打ち込んで

load data infile "/tmp/table.txt" into table mytable;

とすれば入ります。って私のコメントは最近こればっかだな。

http://www.mysql.gr.jp/jpdoc/3.23/manual.ja_toc.html

に非常に良くできたマニュアルがあります。見てみてください。

> PHP とかいうのを使うんですか？

Web から使うならphp は非常に便利ですが、
遠藤さんの場合はバッチでやりたそうなので、先の例をload.sqlとかにいれて、

mysql -B データベース名 <load.sql

とかいうのをshell script  にいれておけばいいでしょう。

> よく分かってません。うー。本が出るのを待った方がいいの？^^;

とりあえずは先の日本語マニュアルでしょう。
