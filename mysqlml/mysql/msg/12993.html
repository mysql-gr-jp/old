オオグシです。

mysql-standard-4.0.27-pc-linux-gnu-i686-glibc23.tar.gz で試してみました。

結果は以下の通りです。

1.rpmをアンインストール後、バイナリをインストールすると以下のようになり
ます。

mysql> use
ERROR 1049: Unknown database '

まだ、おかしいようです。

2.OSをインストールし直して、rpmをインストールしていない状態でバイナリを
インストールすると正常にメッセージが表示されました。

mysql> use
ERROR:
USE must be followed by a database name

3.mysql（バイナリ）を停止させ、rpmをインストール後、mysqlを起動し実行

mysql> use
ERROR 1049: Unknown database XXX　←文字化け

4.mysql(rpm)を停止させ、mysql(バイナリ)を起動すると、正常に動作します。

mysql> use
ERROR:
USE must be followed by a database name

余計わからなくなってきました。
インストール時になにか起こってそうなので、gccあたりを探ってみようと思っ
ています。
