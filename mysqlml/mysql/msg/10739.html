森下＠ちょっと休憩中（ちゅうかもうすぐ朝やで(T_T)）です。

>> sudo /Library/StartupItems/MySQL/MySQL start
>> と実行すると
>> Password:
>> Starting MySQL database server
> になっているので、MySQLは動き始めている？のに．．．．．

「Starting MySQL database server」
は、MySQL自体が出すメッセージではなく、
StartupScriptItemsに登録されているスクリプトが出すメッセージです。
たぶん起動スクリプトが、エラー処理を考慮せずに書かれていて、
起動失敗しても単純に出しているだけでしょう。

本当に起動しているかどうかは、
Applicationのなかのユーティリティフォルダにある、
アクティビティモニタで、該当プロセスがあるかどうか確認してください。
