Subject: [mysql 04779] rpm 版 MySQL が Upgrade できません。

どうも、山本@ふりふりです。
rpm版MySQLをUpgradeしたときに変な状態になったので教えてください。

サーバのメンテナンスを兼ねてrpm版MySQLを3.23.44-1からMySQL-devel-3.23.46-1に
Upgradeしたときです。
ftp.softagency.co.jpからrpm版MySQLをダウンロードしました。
・MySQL-3.23.46-1.i386.rpm
・MySQL-client-3.23.46-1.i386.rpm
・MySQL-devel-3.23.46-1.i386.rpm
・MySQL-shared-3.23.46-1.i386.rpm

そして
# rpm -Uvh MySQL-3.23.46-1.i386.rpm
と以下のエラーメッセージが出てUpgradeできません。
============================================================================
===
[root@www admin]# rpm -Uvh MySQL-3.23.46-1.i386.rpm
Preparing...                ###########################################
[100%]
file /usr/lib/mysql/mysqld.sym from install of MySQL-3.23.46-1 conflicts
with file from package MySQL-devel-3.23.44-1
============================================================================
===
また
# rpm -Uvh MySQL-devel-3.23.46-1.i386.rpm
でやってみても同じ結果でしたが
# rpm -Uvh MySQL-client-3.23.46-1.i386.rpm
# rpm -Uvh MySQL-shared-3.23.46-1.i386.rpm
では正常にUpgradeできております。
さらに"/usr/lib/mysql/mysqld.sym"のファイル名を
変更してやってみてもだめでした。
ちなみにrpmのバージョンはrpm-4.0.2-6xで
OSはredhat6.2です。

恐れ入りますが、この解決方法がございましたら
ご指導、ご鞭撻のほどお願いいたします。
