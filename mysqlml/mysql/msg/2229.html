　MySQL C++ API (MySQL++ 1.7) を使って開発を行っているのですが、インストール
後、include/mysql++ の下にある<sqlplus.hh>をインクルードしたソースファイルをコ
ンパイルすると、bad_query.hh が見つからないとエラーが出てしまいました。

include/mysql++にコピーしておこなったところ問題なくコンパイルもとおりうまく動
いているようです。
　このインストール時に bad_query.hh がコピーされていないのは、インストールのバ
グ？
