片寄です。

> C:\mysql\bin>mysql -h serveraddress -u root -p
> Enter password: ****
> ERROR 2003: Can't connect to MySQL server on '172.16.3.5' (10054)
>
> C:\mysql\bin>mysql -h serveraddress:3306 -u root -p
> Enter password: ****
> ERROR 2005: Unknown MySQL Server Host '172.16.3.5:3306' (11001)
>
> パスワードを要求しているのはローカルのMySQLなのでしょうか？

パスワードを要求しているのは、クライアント側の"MySQL Client"です。
しかし、ここで入力されたユーザ名とパスワードはサーバにログインする時に
サーバ側に送られ、サーバ側で認証されます。


リモートのMySQLにログインするには、サーバにMySQLが導入されていることと
クライアントPCにも"MySQL Client"の導入が必要です。

"MySQL Client"を単独で導入するか、MySQL Serverを導入することが必要です。
"MySQL Server"を導入すると同時に"MySQL Client"も導入されます。
Windows用MySQLをインストールした時に"MySQL Client"が導入されています。


リモートにログインする場合  mysql -h SERVER
ローカルにログインする場合  mysql -h localhost
