"list" <list@xxxxxxxxxx>さんwrites:
> きっとこれらが
> # ./mysqladmin -p create xyz_db<LF>
> とした時、空っぽのディレクトリのみが作られてしまうのでしょうか？

MySQLマニュアル 7.4 CREATE DATABASE の項より引用です
MySQLにおけるデータベースは、データベース内のテーブルに相当するファイルを
含むディレクトリとして実装されます。初期作成後はデータベース内にテーブルは
存在せず、CREATE DATABASEステートメントは、MySQLデータディレクトリ下にディ
レクトリを作成するだけです。

まずはあわてずマニュアルを読んでみるのが近道のように思います。
