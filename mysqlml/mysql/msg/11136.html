MySQL Connector/MXJ 1.1.0 がリリース（ダウンロードできます。）

MySQL Connector/MXJ 1.1.0はMySQLのRDBを展開・管理
するためのJavaのユーティリティのパッケージです。

MySQL データベースをJavaのアプリに追加するのは以下の
ように簡単です。

1) CLASSPATHにConnector/JとConnector/MXJ .jar　ファイルを追加。
2) DBC connection stringに１つパラメータを追加:

String url = "jdbc:mysql://localhost:3306/?"
      + "socketFactory="
      + ServerLauncherSocketFactory.class.getName();

このパラメータを追加することで、Connector/J JDBC ドライバーは自動的
に最初の接続で自動的にMySQL　データベースを展開してスタート
します。

更に、MySQL Connector/MXJは既存のJavaのアプリにバンドルできJavaの
オブジェクトとして管理でき、JMX MBeanとして展開も可能です。これで
Javaの開発者はエンドユーザに対して、インストレーションの手間を
削減することでJavaのアプリを容易に展開できます。

MySQL Connector/MXJがMySQLデータベースをJavaベースのコンポーネント
のように扱うには、まづシステムが実行しているプラットフォームを決定して、
適切なバイナリーを選択して、エクスキュータブルを実行します。オプションと
して最初のデータベースは特別なパラメータで展開します。

JMX MBeanとして MySQL Connector/MXJ はJMX v1.2 準拠のMBean コンテーナ
が必要です。例えば、JBoss の第４版などです。MBeanは標準のJMX 管理
APIを使用してプラットフォームに相応しい
パラメターを提示（そして設定）も行います。

JBOSSに展開すると同時にJDBCへの拡張を使う際のインストラクションが
含まれています。必ず"README"を読んでください。

1.1.0 版での新規なことは MySQL 4.1.10 用の下記のプラットフォーム
上のリソースです。
* Linux x86
* Mac OS X 10.3
* Windows XP/2K/NT x86
* SunOS sparc

ソースとバイナリーのダウンロードは:
 http://dev.mysql.com/downloads/connector/mxj/1.0.html
