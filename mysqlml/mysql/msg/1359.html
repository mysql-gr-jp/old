windows には、mysql_install_db は不要です。
(これは UNIX の shell script です。)
windows はインストール時に、強制的に作られます。


> >２．mysqldの起動の際、--datadir=G:\というオプションをつけると
> >常駐してくれないのですが、「評価版」ならではの制限なのでしょうか？
> 
> ２．に関してですが、c:\mysql\data以下のディレクトリをデータ専用の
> ドライブにコピーして

コピーが大事。
MySQL は、データベースディレクトリ(この場合、標準が C:\mysql\data\)
のしたに、database を directory として作成しますが、mysql データベース
(この場合 C:\mysql\data\mysql\ ) は、必須です。
G:\ とやると、G:\mysql\ を探しに行く事になります。
