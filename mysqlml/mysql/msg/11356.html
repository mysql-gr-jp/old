パッケージはどこからどういうものをとってきましたか？
ネット上には、MySQLのサイトからたどれるもの以外にも、
Mac OS X用のパッケージがあって、デフォルトの設定が違っていたりします。
また、アンインストールはどのように行いましたか？

起動スクリプトの出所と、本体の出所が違っていて、
違うところを指しているような気がします。

また、サーバ版のMac OS X（最初からMySQLがインストールされている）と
クライアント版のMac OS X（自分でMySQLのインストールが必要）とでも、
取り扱いは違いますよ。

とりあえず、
sudo find /usr -name mysqld
とでもして、MySQLデーモンが潜んでいる場所を探し出す。
次に、起動スクリプトが指すMySQLデーモンがそこを指しているか確認する。
のふたつをしてみてください。
