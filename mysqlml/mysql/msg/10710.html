Subject: [mysql 10710] MySQL Connector/MXJ 1.0.9がリリースされました。

MySQL Connector/MXJ 1.0.9がリリースされました。
（ミラーサイトによっては設置まで２－３時間掛かるかもしれません）

MySQL Connector/MXJはMySQLを展開・管理するためのJMX MBean です。
これを使えばJavaの開発者はエンドユーザのインストレーションの
問題を軽減してデータベースを必要とするアプリを展開できます。

Connector/MXJ にはJMX v1.2準拠のJBoss4版のようなMBeanコンテーナー
が必要です。

JMX MBeanとして、Connector/MXJはMySQL データベースをJavaベース
のコンポーネントとして扱います。システムが実行している
プラットフォームを決定し、正しいバイナリを選び、エクスキュータブル
を実行することで達成します。オプションとして初期のデータベース
を展開して、JMX管理APIを通じてそのプラットフォームに適する
パラメーター（とその設定）も提供します。

JBossに展開するインストラクションも含まれています。
"README"と"GettingStarted.txt"のファイルを読んでください。

1.0.9 版では次のプラットフォームのMySQL 4.1.8 リソースを含みます。

* Linux x86
* Mac OS X 10.3
* Windows XP/2K/NT x86
* SunOS sparc

JBossをお使いなら、JBossのライフサイクルの拡張方法を採用
しましたので、JBossがシャットダウンするとデータベースも
問題なく停止します。

ソースとバイナリーはここから

http://dev.mysql.com/downloads/connector/mxj/1.0.html
