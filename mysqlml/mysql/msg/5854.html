UPDATE で LIMIT 節を使うと指定した個数のみ更新ができます．
update test set b_1 = 111 where a_1 = 'aaa' limit 1;
http://www.mysql.gr.jp/jpdoc/3.23.x/manual.ja_Reference.html#UPDATE

しかし，あいまいな指定なのでどのレコードが更新されるかはわかりません．
