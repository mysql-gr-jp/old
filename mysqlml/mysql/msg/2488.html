中嶋です。

OpenZOLAR 6.0.0をリリースします。OpenZOLARは、商用版ZOLAR
(http://www.isr.co.jp/)と同等な機能を、フリーのデータベース
向けにリリースしたものです。バイナリリリースとなります。
今回のリリースからMySQLにも対応しました。

今回提供可能になったプラットフォームは、以下のとおりです。
それぞれ、 PostgreSQL 7.0版とMySQL 2.23.21版をリリースしま
す。さらに、PPCLinux にも対応しました。

    Linux glibc2: Red Hat Linux 5.2Jビルド 
    Linux glibc2.1.3: Red Hat Linux 6.2Jビルド 
    Linux SPARC: Red Hat Linux 6.2ビルド 
    Linux PPC: OpenBlockSビルド(kernel: 2.2.13, glibc2.1.2) 
    FreeBSD 3.x: FreeBSD 3.2ビルド 
    NetBSD m68k: NetBSD m68k 1.4.2ビルド 

(おねがい)
OpenZOLARをご利用の皆様におねがいがあります。LinuxPPC版の
開発には OpenBlockSを使用しており、Macintosh上のLinuxPPC
では、動作を確認して おりません。OpenBlockSは、特殊な
LinuxPPCですので、Macintosh版LinuxPPCをお使いの方は、ぜひ
動作結果をレポートして下さい。

また、NetBSD/m68k版MySQL対応の動作確認はできていません。
OpenZOLARの ビルドはできましたが、私の開発環境上にMySQLの
クライアントライブラリ を正しく作成できていません。NetBSD
/m68kをお使いの方は、ぜひこの問題 に関するアドバイスをお
願いします。
