> mysql -u root
でパスワードなしおよび
> mysql -u root -p
でパスワード無しでenterを押すとmysqlのプロンプトになります。

パスワードなしのrootはアクセス可で、パスワードありのrootはアクセス不可とい
う状況でしょうか？

が設定されているか
