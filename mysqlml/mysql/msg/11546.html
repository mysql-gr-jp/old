Subject: [mysql 11546] MySQLがMacintosh OS X用のオープンデータベースのツールをリリース

MySQLがMacintosh OS X用のオープンデータベースのツールをリリース
http://www.mysql.com/news-and-events/news/article_896.html

2005年6月7日

ビジュアル・クエリとアドミのツールのベータ版は
http://dev.mysql.comからダウンロード可能

サンフランシスコのApple Worldwide Developer Conference,発

オープンソースのデータベースの開発社のMySQL ABは今日
２つのビジュアルのツールをMacintosh OS X用を発表しました。開発者、
DBAやユーザーを助けるための Macintosh OS X用の
ビジュアル・クエリとアドミのツールで、GPLライセンスで
以下からダウンロードできます。 http://dev.mysql.com.

MySQLと新しいMacintosh のツールは
サンフランシスコのApple Worldwide Developer Conference
でデモされています。

開発担当の副社長のMaurizio Gianolaは「Apple Macintosh
はMySQLの開発でよく使われるプラットフォームの１つです。
ビジュアル・クエリとアドミのツールを提供出来て嬉しいです。
MySQLは高負荷が掛かっているApple Xserveサーバー上
で高速で高信頼性を示しています。」と語り、eBay, AOL Time Warnerや Expedia
を顧客に持つLiveWorldの副社長のBruce Dembeckiは「一番マニアックなMacの
ユーザーでも賛成するくらいセットアップが簡単です。クライアント
のディスカッション・ボードで毎日たくさんのページに
アクセスがありますか、MySQLとMacはスケールアップして問題
はありません。」と語りました。

MySQL クエリ・ブラウザー

MySQLクエリブラウザーはビジュアルのツールで、MySQLのデータベース
のクエリを生成、実行、最適化するのに使います。ウエブブラウザー
のようなインターフェースで統合されたドラッグ・アンド・ドロップ
のツールで MySQLデータベース・サーバー内に格納される情報
にアクセスして、解析するのに容易で生産性があがります。

MySQL Administrator

MySQL Administratorはパワフルなビジュアルなコンソールで
DBAがMySQLのアプリを容易に管理しモニターすることができます。
直観的なグラフィカルなユーザー・インターフェースで、
DBAがMySQLのオペレーションを熟知できます。これには
サーバー・コンフィグレーション、ユーザー管理、リプレイケーション
の状況、バックアップ・リストーやログの閲覧などです。

その他のプラットフォーム

Macintoshの他にMySQLクエリ・ブラウザーとMySQL Administrator
はLinuxとWindows用があります。

ダウンロードと他の情報は
http://www.mysql.com/products/query-browser/
http://www.mysql.com/products/administrator/
