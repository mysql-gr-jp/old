MySQLと Red Hatはカスタマー・サポートの改善を誓約
2005年4月21日
http://informationweek.com/story/showArticle.jhtml?articleID=161500351

MySQL CEOはビジネスはグリッド・コンピューティング用
にMySQLデータベースとRed Hat Linuxの使用を増加させていると語る

Charles Babcock著、InformationWeek

MySQL ABはRed Hatとパートナー関係を築き、企業が大きなサーバー
クラスターやグリッドを建設する際にLinux上でオープンソースの
データベースを実行するのをサポートすることを保証します。

MySQL CEO のMarten Mickosは「大きなサーバーを買うよりも
安価なサーバーでクラスターまたはグリッドを形成して
水平にスケールをすることをサポートします。」と
MySQLのユーザーコンファレンスのキーノートの
スピーチのすぐ後のインタビューで述べました。 Mickosは
このアプローチは 「scaling up」でなく「scaling out」と呼んでいます。

１時期MySQL とRed Hatは競合関係にありました。３年前Red Hat
は PostgreSQLの版をオープンソースのデータベースとして
Red Hat　Databaseのブランドで提供しました。しかしながら
これは成功しませんでした。２年前Red HatはMySQLをRed Hat Linuxの
ディストリビューションの一部として配布し始めました。今日
２つの会社はジョイントのサポートを正式化しました。

両社はグリッド・コンピューティングを提供する企業と協業したいと
思っています。グリッド・コンピューティングはMySQLとLinux
の双方のデマンドを増加する傾向があります。サンフランシスコ
の２年目のActiveGrid Inc.はLinuxと MySQLをどちらも利用し
スケールリングをサポートするグリッドを形成するソフトを今月発表しました。

MickosはGoogle Inc. をITのインフラを「scale up]ではなくて
「scaling out」で形成する会社の例だと指摘しています。 またCox
Communications Inc. は14 Linuxのサーバー・クラスターで
データを収集し、25 LinuxのサーバーでMySQLを実行してデータを
解析すると指摘しました。

Travelocityの旅行のウエブサイトを運営しているSabre Holdings Corp.
はGoldenGate Software Inc.を利用してデータ同期と管理を行い、
LinuxとMySQLのグリッド上で旅行情報を提供しています。オンラインの
ソーシャルネットワークのサイトのFriendster.comは95のMySQL と Linux
のサーバーで運営していますとMickosは続けました。

MySQLに はスケールできるデータベースのクラスターやグリッド
を建設する際にハイアベイラビリティや高性能のクラスタを提供できる
ソフトのベンダーのパートナーもいます。Embarcadero Technologies Inc.や
Emic Networksなどです。
