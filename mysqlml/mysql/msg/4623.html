みなさん、ありがとうございます。

スレ元の朝倉です。

方向転換してDSO版でのインストール作業に変更しました。

# cd /usr/local/src/apache_1.3.20
# ./configure --enable-module=so
# gmake
# gmake install

# cd ../php-3.0.18-i18n-ja-2
# ./configure --with-mysql=/usr/local/mysql --with-apxs=/usr/local/apachedso/bin/apxs --enable-i18n --enable-mbregex --enable-track-vars
# gmake
# gmake install

以上のコマンドでインストール完了です。

がしかし、新たな問題に直面しました。
# apachectl start
の結果が以下になります。
---------------------------------------------------------------------
# /usr/local/apachedso/bin/apachectl configtest
Syntax error on line 206 of /usr/local/apachedso/conf/httpd.conf:
Cannot load /usr/local/apachedso/libexec/libphp3.so into server: ld.so.1: /usr/local/apachedso/bin/h
ttpd: 重大なエラー: 再配置エラー: ファイル /usr/local/apachedso/libexec/libphp3.so: シンボル ap_user
_id: 参照シンボルが見つかりません。
---------------------------------------------------------------------
その後いろいろ変更して試したのですが、相変わらずです。
変更履歴が以下になります。
---------------------------------------------------------------------
→　httpd.confを以下のように変更
# Example:
# LoadModule foo_module libexec/mod_foo.so
LoadFile /usr/local/mysql/lib/mysql/libmysqlclient.so
LoadModule php3_module        /usr/local/apachedso/libexec/libphp3.so
LoadModule php3_module        libexec/libphp3.so
---------------------------------------------------------------------
→　パスの追加
# ldd /usr/local/apachedso/libexec/libphp3.so
でリンクしている各モジュールのディレクトリを
LD_LIBRARY_PATHに追加。
---------------------------------------------------------------------
