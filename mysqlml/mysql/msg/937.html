[Date: Thu, 3 Jun 1999 14:35:23 +0900  From:"Akinobu Yamamoto"]

> rootで入ってmysqladminコマンド(mysqladmin -u username -p create database)
> で行っているのですが、なぜか'Access denied for user'ということで
> 蹴られてしまいます。

これは何をやっているかというと、

	MySQL のユーザ username がデータベースを作成しようとしている

ということです。当然 username にデータベース作成の権限が与えられて
いなければエラーになります。

たぶん、これは本当にやりたいことではないと思います。勝手にやりたいことを
推測すると、

	MySQL のユーザ username でアクセス可能なデータベースを作成する

ということではないかと思います。これが本当にやりたいことであれば、

 1. MySQL のユーザ username を作成する。
 2. データベース database を作成する。
 3. username に database のアクセス権を与える。

という作業が必要になります。

この辺のことは、マニュアルだけではちょっとわかりずらいので、

	http://www.softagency.co.jp/mysql/TODO/permission.html

なんかをみるのが良いと思います。
