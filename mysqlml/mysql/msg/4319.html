Win98/ME/NTではどうかわかりませんがWin2Kで無理矢理Batファイルで
かこうとするとこんな感じですかね。


--foo.bat-----------------------------
@echo off
rem SQLファイル名の保存
if "%1" == "" goto :EOF
set infile=%1

:TOP
rem 引数を一つずらす
shift /1

rem もし次が無ければ終了
if "%1" == "" goto :END

rem infileから読み込んで表示する
for /F  "tokens=* delims=;" %%A in (%infile%) do call :print %%A

rem 頭に戻る
goto :TOP

:print
@echo %*
goto :EOF

:END
rem 変数の後始末
set infile=
--------------------------------------
--create_tbl.sql----------------------
CREATE TABLE u_%1
( id int,
  name char(20)
);
--------------------------------------
で
c:\>foo create_tbl.sql 10 20 30 40 | mysql test

二つ以上引数が渡せないとか、行頭のスペースやタブが削除されるとかの
問題があるのですべてのケースに対応できる訳じゃ無いけど^^;
