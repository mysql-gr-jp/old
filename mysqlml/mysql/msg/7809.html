はじめまして、

/Library/StartupItems/MySQL/

の中のMySQLというファイルが起動スクリプトになります。
それを確認されてはどうでしょうか？

ちなみに僕の場合は、

/Library/StartupItems/MySQL/
	MySQL
	Resources/
		English.lproj/
			Localizable.strings
		Japanese.lproj/
			Localizable.strings
	StartupParameters.plist

という構成で、

MySQLの中身は

#!/bin/sh
. /etc/rc.common
##
# Start MySQL Server
##

if [  "${MYSQLSERVER:=-NO-}" = "-YES-" ]; then

	ConsoleMessage "Starting MySQL Server."

	sudo sh -c "nohup /Library/MySQL/bin/safe_mysqld --user=mysql &"
	
fi

という感じで、/Library/MySQL/以下にインストールしたものを起動しています。

StartupItems関係の設定については、Appleからもドキュメントが出ていたと思います。
検索すればすぐ出てくると思います。
