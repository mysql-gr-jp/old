SQL文の中でIF文を使った事がなかった為、こう言う事も出来るのかと感心しました。
勉強不足でお手数お掛け致しました。

SELECT 
  BASEDATE,
  PAGENAME,
  SUM( COUNT ) AS E_COUNT,
  if(PAGENAME <> @chk, @r1 := COUNT,@r1 := (@r1 + COUNT)) AS E_ALLCOUNT, 
  (@chk := PAGENAME),
FROM カウントテーブル名 ...

このような形で実現する事ができました。
