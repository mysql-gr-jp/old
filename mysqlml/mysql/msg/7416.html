以下、参考までに手順です。

+------ tar解凍・インストール ------+
	# tar xvf mysql-4.0.12.tar
	# cd mysql-4.0.12
	# ./configure -prefix=/usr/local/mysql
	# make
	# make install
	# su - mysql
	# /usr/local/mysql/bin/mysql_install_db 
	# chown -R root.mysql /usr/local/mysql
	# chown -R mysql.mysql /usr/local/mysql/var
	# cp /path/to/mysql-3.23.54a/support-files/my-medium.cnf /etc/my.cnf

+------ MySQL起動 ------+
	#/etc/rc.d/init.d/mysql start

参考：http://www.roomc.jp/mysql/inst_mysql0323.html
