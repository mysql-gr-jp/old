MySQL Network 詳細とMySQL 5.0のプレビュー

MySQL ウエブ・セミナー - 2005年7月27日

ビジネスに重要なアプリケーションを成功裏に
実行するための要求はデータベースの信頼性と
セキュリティとアップタイムを向上させることです。
MySQL Networkはエンタープライズのデータベース
のアプリを効率よく開発し展開できるため
の環境です。このセミナーでは以下を
カバーします。

* アドバイザーの利用で問題が起こる前に解決
* 認定ソフトで信頼性とアップタイムの改善
* 知識ベースにアクセスすることで、よくある
質問に対する回答
* MySQLサポート・グループへ直接アクセスすることで
特定のアプリの問題を解決
* 自動ソフトのアップデートとカスタマイズされた
アラート

MySQL Networkを使えば開発者とDBAが時間と経費を節約して
さらに生産性を向上させます。

MySQL NetworkがMySQL　5.0を知る一番速い方法なので、それの
プレビューもお届けします。機能の一部は

* Stored Procedure
* Trigger
* View
* Cursor
* XA
* Precision Math
* Information Schema

発表者:

Andy Bang, MySQL Networkのディレクタ
Alex Roedling , シニア・プロダクト・マーケーティング・マネージャ

内容:

MySQL Network詳細: MySQL Networkがいかにデータベース
のアプリの信頼性、セキュリティとアップタイムを
改善するかを学びます。アドバイザーがいかにして、問題
が起こる前に解決するのかを学びます。 いかに、 MySQL Network
が開発者とDBAの生産性をあげ、時間・工数・経費を 節約
できるかを学びます。MySQL Networkが一番速く
MySQL 5.0を学ぶ方法なので、5.0のプリビューを
提供します。
