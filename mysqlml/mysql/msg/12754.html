Subject: [mysql 12754] コネクション時のレスポンスについて

病院の電算担当をやっております中山と申します。

コネクションのオープンに毎回15秒前後かかります。
レスポンスを向上させたいのですが、解決策があれば教えて頂けないでしょうか。


【私が行ったこと】

・サーバーの再起動

・他の端末に同じ環境を作成して動作検証

・メモリ容量の見直し
　　以下のように値を変更しました。
　　---------------------------------------
　　set-variable = key_buffer_size=256M
　　set-variable = max_allowed_packet=1M
　　set-variable = sort_buffer=1M
　　set-variable = record_buffer=1M
　　---------------------------------------


【Server環境】
・メモリ：1G
・CPU：Pen4 3GHz
・MySQL Server Ver4.0.25
・ODBC Ver3.51.11.00
・開発環境 VB6.0 SP5


【コネクション部分のソース】
最終行のopen部分が遅いです
--------------------------------------------
Public gconADOConnection_MySQL      As ADODB.Connection

strConnect="{MySQL ODBC 3.51 Driver};SERVER=**.**.**.**;DATABASE=**;UID
=**;PWD=**;"

Set gconADOConnection_MySQL = New ADODB.Connection
gconADOConnection_MySQL.CommandTimeout = 60
gconADOConnection_MySQL.Open strConnect
