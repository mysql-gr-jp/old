MySQL Shellはまだまだ知名度がいまいちな気がしてなりませんが、
X DevAPI以外にも、バージョンアップ時に便利なUpgrade Checkerや
1テーブルに対して複数スレッドでデータのロードが可能なImportユーティリティー、
MongoDB(!)のBSONをインポート可能なユーティリティーなど
便利機能も多数実装済み、かつ機能追加もどんどん行われています。
  https://dev.mysql.com/doc/mysql-shell/8.0/en/mysql-shell-utilities.html

MySQL Shellにはプラグイン機能があり機能拡張も可能です。
ぜひMySQL Shell本体を使ってみた方、プラグインを作ってみたい方、
プラグインをもう作ってる方などのご登壇もお待ちしております。

ぜひご参加ください。
