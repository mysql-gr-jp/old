>Microsoft Accessで作成したテーブルをインポートしたいのですが、どうすれば良

MySQL は起動していますか？
ユーザはありますか？

ここまで準備が出来ているなら、まず MyODBC をダウンロードして
インストールして下さい。

その後、[コントロールパネル]-[32ビットODBC]で[ユーザDSN]を作
成して下さい。
# 始めからある sample-MySQL でもいいです。

その後 Access 側から ODBC 経由でテーブルをエクスポートして下
さい。

これでいわゆる MySQL 側から見たインポートが実現できるはずです。
