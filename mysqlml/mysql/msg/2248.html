場合によっては、そういう状況もあり得ます。

GPL は、GPL のソフトを含む・改変したものについても　GPL を要求します。
以下の場合に、製品のライセンスを GPL にしなくてはならず、
全ソースを公開することになります。(UNIX, Windows 共に)

そのため、もしソースを公開したくない・GPL にしたくないのであれば、
MySQL のライセンスを購入し、従来の MySQL FREE PUBLIC LICENSE
を受けることが可能です。

想定される状況：
1. MySQL サーバーに変更を加えたものを、売る場合・ソースを公開しない場合
2. MySQL サーバーを組み込んだソフトを売る場合・ソースを公開しない場合
3. MySQL だけでしか動作しないソフトを、
   MySQL と共に出荷して売る場合・ソースを公開しない場合
   これは MySQL に変更を加えていなくても、その商品が MySQL を組み込んで
   いる(上記2項と同じ状態) とみなされます。

現在、2,3 に関しては、GPL version2 の 第2項、

    b) You must cause any work that you distribute or publish, that in
    whole or in part contains or is derived from the Program or any
    part thereof, to be licensed as a whole at no charge to all third
    parties under the terms of this License.

の解釈として、現在、TcX , MySQL AB が取っているスタンスです。


もちろん、MySQL を含まない状態でソフトを売ることに関しては、
GPL の影響外になりますし、
GPL にそってその商品のソースを無償で公開するのであれば、
問題はありません。
