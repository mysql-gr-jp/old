3.23.21 -> 3.23.40に先日ヴァージョンアップしたばかりです。

3.23.21 では、以下のＳＱＬでtable1.userid <> table2.useridができていましたが

SELECT DISTINCT user.userid FROM table1 LEFT JOIN table2 ON table1.userid=table2.userid,user 
WHERE user.userid = table1.userid AND table1.userid <> table2.userid 
AND table1.itemid = '$itemid' GROUP BY table1.userid ORDER BY RAND() limit $quantity;

3.23.40では、table2.userid is null と書かないと結果が０になります。

SELECT DISTINCT user.userid FROM table1 LEFT JOIN table2 ON table1.userid=table2.userid,user 
WHERE user.userid = table1.userid AND table2.userid is null 
AND table1.itemid = '$itemid' GROUP BY table1.userid ORDER BY RAND() limit $quantity;

3.23.40ではライブラリーも変更されているようなので
ヴァージョンを戻すのが大変そうなので
全部ＳＱＬを書き換えました(;_:)
