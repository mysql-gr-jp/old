おおあさです。

この現象、ハイパーバイザーに VMware Player を使ってる環境の事象として
(たいてい ssh が切れるという題目で)よく聞くんですが、自分では実際に
当たったことがないので以下のようなサジェッションを挙げてみます。

・ホストマシンの仮想アダプタを vmnetcfg 等で hostonly か bridge に変更
　する(NAT=VMnet8 を無効化する)
・ホストマシンの VMware NAT Service を停止する
・ゲストマシンのNICを準仮想化(vmxnet)に変更する
・ゲストマシンのNICの TSO をオフにする


mysql クライアント(CLI)のコネクションが切れる事象へのワークアラウンド
としては、ServerAliveInterval をオンにした ssh クライアントでゲストOS
にログインして mysql クライアントを使う、ポートフォワーディングを使う
とか・・・


ご参考まで。

よろしくお願いします。
