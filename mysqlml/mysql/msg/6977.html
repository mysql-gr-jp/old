>警告: perl-DBD-MySQL-2.1017-3.i386.rpm: V3 DSA signature:
>NOKEY, key ID db42a60e
>エラー: Failed dependencies:
>        libmysqlclient.so.10 is needed by
>perl-DBD-MySQL-2.1017-3
>
>と怒られます。ちょっと調べると、libmysqlclient.so.10
>はmysql-share...うんたらに含まれているとか。


rpm -ql MySQL
でlibmysqlclient.so.10が入ってるか見てみて
無かったら
libmysqlclient.so.10は、MySQL-sharedに入ってます。

www.rpmfind.netでlibmysqlclient.so.10で検索すると
libmysqlclient.so.10を使ったMySQLのヴァージョンが出ます。

お勧めは
http://www.mysql.com/downloads/mysql-3.23.html
から最新版を全部インストールするのがいいかと思います。

perl-DBD-MySQLを一度削除してから入れるのが早いかと。。
