いつもお世話になっております、
バインシステムの平井と申します。

MySQLをソースからコンパイルしてインストールしたのですが
キャラクターセットのconfファイルが入ったフォルダが見つかりません
でした。

これはどこにあるのでしょう？通常は
/usr/local/share/mysql のだかだと思うのですが

申し訳有りませんが教えてください。

-------仕様
OS:Red hat 7.0J
MySQLバージョン:3.22.32
コンパイルしたファイルの取得先:MySQL本部のFTPより

インストールコマンド:
gzip -dc mysql-3.22.32.tar.gz | tar -xvf -
cd  mysql-3.22.32.tar.gz
./configure --with-charset=ujis --with-extra-charsets=all
--with-mysqld-user=mysql
make
make install

です。
