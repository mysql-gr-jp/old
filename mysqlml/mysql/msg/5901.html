EUCでの統一も検討し、実際に試して見たりもしたのですが
Win上でAccess等の既存のツールが使えないなど不便な点もありました。

Cold Winterさんの以下の方法が上手くいきましたので、
こちらで対応することにしました。

-----------------------------------------------------------------
strMojiretsu1 = "ファイル一覧を表示する";
intKazu2 = 0;
intKazu3 = 0;
try
{
    Connection cont = DriverManager.getConnection(…);
    PreparedStatement prpStmt = cont.prepareStatement("insert into table1
values(?, ?, ?)");
    prpStmt.setString(1, strMojiretsu1);
    prpStmt.setInt(2, intKazu2);
    prpStmt.setInt(3, intKazu3);
    prpStmt.execute();
}
catch (Exception ee)
{
    /* エラー処理 */
}
-----------------------------------------------------------------

Java＋MySQL＋SJISの場合は、PreparedStatementを用いる
この方法が最適のように感じました。

引き続き動作確認を行いたいと思います。
ありがとうございました。
