   ■■  Open Source Database Conference 開催のお知らせ  ■■
   ■■　    今すぐhttp://osdn.jp/event/db2002/ へ       ■■

近年、オープンソースのSQLデータベース・システムが商用データベース
と同等の機能、性能、信頼性を備えつつあります。また、MySQL, PostgreSQL,
Interbase に代表されるオープンソースのデータベース・システムは、
Apache Webサーバーや Perl, PHP といったスクリプト言語との相性も良く、
Webアプリケーションの開発においてはそれを支えるバックエンド・データベース
として欠かせなくなってきております。

　今回、OSDNジャパンはそれを踏まえて、オープンソース・データベース・
システムにフォーカスした「Open Source Database Conference」を開催すること
に致しました。これにより、国内におけるオープンソースのデータベース・
システムが普及、促進されることを期待します。

　なお、今回のカンファレンスには、MySQL AB, CTO  David Axmark氏や
IBPhoenix, President Ann Harrison氏のが海外から特別講演者として
来日する予定です。
