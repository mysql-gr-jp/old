鶴長です。@ITの私の記事をご参照頂きありがとうございます。

"mysql"コマンドはローカルな Unix ソケット経由を使っているのに対し、
javaアプリケーションではTCP/IPソケット経由のため、挙動が違ってく
ると思います。（←だと思うのですが、違っていたらご指摘下さい）
参照権を適切に設定すれば解決するのですが、現在はどうなってますで
しょうか。

# mysql mysql
>select Host,User from user where User = "www";

Host			| User
----------------------------------
%			| www

しかない場合は、localhostを明示的に加えてみて下さい。

> GRANT Select,Update,Insert,Delete ON test.* TO www@localhost IDENTIFIED BY 'XXXX';
> select Host,User from user where User = "www";

Host			| User
----------------------------------
%			| www
localhost		| www
