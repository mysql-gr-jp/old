MYSQLを立ち上げましたか？
それとも、C:\直下にmy.cnfというファイルがありますか？そのファイル中に、

# Uncomment the following rows if you move the MySQL distribution to another
# location
#basedir = c:/mysql/
#datadir = c:/mysql/data/

があったら、下記のように変更してください

# Uncomment the following rows if you move the MySQL distribution to another
# location
basedir = d:/mysql/
datadir = d:/mysql/data/

他のC:\xxxxはD:\xxxxに変更されなければならないと思います

日本語は下手ですから，すみませんでした。
