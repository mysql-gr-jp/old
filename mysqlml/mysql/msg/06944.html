Subject: [mysql 06944] variable の設定について

初めてメールをいたします。佐藤と申します。

現在、MySQLのオプションの設定で悩んでおります。
fulltext searchのオプション(variable)のひとつ、ft_min_word_lenの変更を行いたいのですが、
うまくいきません。
環境はRedHat Linux7.2で、ソースコードからインストールしたMySQL4.0.9gammaを使用しています。

現在までに以下の方法を試しました。
１）SET構文による設定
　set variable ft_min_word_len
　を実行すると、エラー（unknown system variable'ft_min_word_len'）が出てしまい、
　変更できません。

２）my.cnfによる設定
　/etcにmy.cnfを作成し、
  　[mysqld]
　　ft_min_word_len=1
　と書き込んで保存し、MySQLを起動しようとすると、起動に失敗します。
　記述を
　　ft_min_word   1
　や
　　set-variable ft_min_word_len=1
　としても同様に起動できませんでした。

３）ft_static.cによる設定
　ソースコードをコンパイルする前に、/myisam/ft_static.cの
　　ulong ft_min_word_len=4;
　という記述を
　　ulong ft_min_word_len=1;
　に変更し、インストールを行いましたが、インストール後にshow variablesで見てみると
　デフォルトの４のままで、変更は反映されませんでした。

サイトなどで情報を見ると変更はできるようなのですが・・・

お手数とは思いますが、ご助力よろしくお願いいたします。
