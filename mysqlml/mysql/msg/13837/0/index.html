>同じくDos窓からmysql mysqlと入力しても、管理用のmysqlDBにはアクセスできません。

これって、"mysql"ユーザでログインしてるということですか？
MySQLの管理ユーザは"root"ユーザですので「Dos窓」から

mysql -uroot -p

と実行して、"root"ユーザのパスワードを入力してログインし、

use mysql;

とすれば、管理用のmysqlDBにアクセスできます。
