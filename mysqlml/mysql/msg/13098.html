「終了」ってmysqlクライアントの終了?
「エラー」ってエラーコードかなんか吐いて終了しろって意味?
プロシージャの書き方聞いてる?

select文の延長線じゃできないんじゃないかな。
シェル上でmysqlクライアントにファイルに書いたSQL文を喰わせるって話なら
分岐SQLと分岐後SQLに2分割して単純にシェルスクリプト書くだけなんだが。

#!/bin/bash
a=`mysql test < 1.sql | tail -1`
if [ $a = 0 ]
then
  mysql test < 2.sql
else
  exit 3939
fi
exit
