はじめまして、松本と申します。

今Apache+PHP+MySQLでデータベースの勉強をしています。「MySQL＆PHPでつくるWeb
データベース」をみながら簡単な掲示板をつくったのですが、PHPからMySQLへの接続
ができません。
ソースは
MyBBS.incは

<?
$MySQLHost = 'localhost';

$MySQLUserName = 'spencer';

$MySQLDataBase = 'mydb';

?>

とし、MyBBS.phpでは

<?

include("MyBBS.inc");

$MyLink = mysql_connect($MySQLHost, $MySQLUserName, $MySQLPassword);

mysql_select_db($MySQLDatabase);

MySQLでもuserとしてspencerがあり、データベースにもmydbというのが存在します。
これで実行すると
Warning: Host 'localhost is not allowed to connect to this MySQL server
Warning: MySQL Connection Failed: Host 'localhost' is not allowed to connect
to this MySQL server
Warning: MySQL: A link to the server could not be established
と出てきます。
環境は
RedHutLinux7.2
PHP 4.2.0
MySQL 3.23.49
Apache 1.3.24
です。
