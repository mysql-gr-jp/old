mysql> create table test2
    -> ( suryo numeric(14,4) null,
    ->   tanka numeric(15,3) null,
    ->   kingaku numeric(15,3) null);

というテーブルを作るなら、

mysql> create table test2
    -> ( suryo numeric(14,4) null,
    ->   tanka numeric(15,3) null);

にしてしまって、kingakuはSELECT時に

mysql> select suryo, tanka, suryo * tanka as kingaku;

にしたほうが良いと思いますけど。
