Subject: [mysql 04981] SELECT コマンド内で変数を使用したい

宏です。

MySQLデータベースをPerl使い作成しているのですが、SELECTコマンドを使った
時、変数を使った場合はエラーが表示され、普通に数字を指定した場合はちゃんと
結果が表示されるのですがこれを変数を使った場合も結果を表示するようにしたい
のですがどのようにすれば良いのでしょうか？

《数字指定》
$sth2 = $dbh->prepare( qq{ SELECT bangou, DATE_FORMAT(hizuke, "%d日%H:%i"),
komento FROM bbs1 ORDER BY bangou DESC LIMIT 0, 5 });
・上記の場合は結果が表示される。

《変数指定》
$sth2 = $dbh->prepare( qq{ SELECT bangou, DATE_FORMAT(hizuke, "%d日%H:%i"),
komento FROM bbs1 ORDER BY bangou DESC LIMIT 0, "$goukei" });

↑
・矢印で示した部分を変数に変えるとエラーが表示されてしまいます。
ちなみに$goukeiには’5’が代入されています。

何方か良い方法をお教え下さい。
