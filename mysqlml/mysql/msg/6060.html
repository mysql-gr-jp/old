確か福嶋さんの環境はWin98+Apache+PHP+Mysqlでしたよね？
この場合PHPマニュアル(http://www.php.net/manual/ja/function.fread.php)にある
ように
RCV.php内の
>$fp=fopen($userfile,"r");
を
>$fp=fopen($userfile,"rb");
に変更してみてください。

RCV.phpでファイルを登録した時のUploaded file SizeとMysqlクライアントで
select ID,length(Dbinary) from stddoc;
した時のサイズが違っているようならこれが原因かと思います。
