Subject: [mysql 10733] Mac OS XでMySQLが使えない。

こんにちは。きさらと申します。

最近、Webアプリ作成の仕事をする事になりまして、今まで使っていた
Mac（Power Book G3 MacOS X 
10.3.7)にmysql-max-4.0.22-apple-darwin7.5.0-powerpc
を入れて色々とテストしようと思い、書籍（九天社のWebデザイナーのためのフリーDB
構築法）を参考にインストールしてみました。一度は、MySQLが実行できていて
phpMyAdminが動いていたのですが、Macを再起動してから、以下の様なエラーになるよ
うになりました。

----------------------------------
Welcome to phpMyAdmin 2.5.7-pl1

phpMyAdmin tried to connect to the MySQL server, and the server 
rejected the connection. You should check the host, username and 
password in config.inc.php and make sure that they correspond to the 
information given by the administrator of the MySQL server.

Error

  MySQL said:

#2002 - Can't connect to local MySQL server through socket 
'/tmp/mysql.sock' (2)

  [Documentation]
----------------------------------
と表示されます。

また、ターミナルで、
sudo /Library/StartupItems/MySQL/MySQL start
と実行すると
Password:
Starting MySQL database server
となるので、たぶんインストールは成功していると思うのですが、

/usr/local/mysql/bin/mysql
を実行すると

ERROR 2002: Can't connect to local MySQL server through socket 
'/tmp/mysql.sock' (61)
とエラーになります。

また、
/usr/local/mysql/bin/mysqladmin -u root password abcde
とパスワードを設定しようとすると

/usr/local/mysql/bin/mysqladmin: connect to server at 'localhost' 
failed error:
  'Can't connect to local MySQL server through socket '/tmp/mysql.sock' 
(61)'
Check that mysqld is running and that the socket: '/tmp/mysql.sock' 
exists!
とエラーが表示されます。

/tmp/mysql.sockは、ちゃんと存在していますが、ファイルサイズはゼロでした。

どうすれば、MySQLが使える様になるかわかるかたいましたら教えてください。
よろしく、お願いいたします。
