BS320-80（小川 信人） <nogawa@xxxxxxxxxx> wrote:

> MySQL 4.1.8を、Windows XPにインストールしました。
> my.iniに、
> default-character-set=sjis
> と指定しています。
> この設定は、MySQLを使っているシステムの以前から
> の設定なので、変更できません。
> この設定で、コマンドラインから、日本語データを
> 登録すると、文字化けしてしまいます。
> 登録方法は、データ登録のSQL文のテキストファイルを、
> mysql -u root mydb < InsertUser.sql
> 等として登録しています。

mysql -u root mydb --default-character-set=sjis < InsertUser.sql
                   ^^^^^^^^^^^^^^^^^^^^^^^^^^^^
とすれば大丈夫かと思います。
