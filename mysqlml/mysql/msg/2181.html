私は SPARC Solaris-2.5.1 用の 3.22.32 バイナリは作成していません。
ということは、Solaris 2.6, 7, 8 用のバイナリを Solaris-2.5.1 上で
動かしてはいないでしょうか？
SunOS5.5, 5.6, 5.7 , 5.8 は library が別物だと思った方がいいです
使用している機種のバイナリが無い場合は、ソースからコンパイルしてください

公開しているバイナリは、sun4u アーキテクチャのものです(Sun Ultra-AXi で作成.)

次に Disk I/O ですが
例えば MySQL 以外の、UNIX bench や bonnie などで
一度 write のチェックをしてみてください
ハード側の原因も考えられます

3.22 の ISAM テーブルより、3.23 で採用されている MyISAM の方が
パフォーマンスもすこし良くなっているはずです
3.23 でテーブルを作り直してみる価値もあります
