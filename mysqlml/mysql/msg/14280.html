かわうそです。

date_format()を使用すると、上手く結果が返るようです。
（MySQL 5.0.46 Win)


mysql> select * from a;
+---------------------+
| c1                  |
+---------------------+
| NULL                |
| 2007-01-01 10:00:00 |
+---------------------+
2 rows in set (0.00 sec)

mysql> select * from a where date(c1) = '2007-01-01';
Empty set (0.00 sec)

mysql> select * from a where date_format(c1, '%Y-%m-%d') = '2007-01-01';
+---------------------+
| c1                  |
+---------------------+
| 2007-01-01 10:00:00 |
+---------------------+
1 row in set (0.00 sec)


以上です。
