> /usr の容量が100%になってしまいました。
> ここで質問なんですけれど、MySQLを/usr 以外に移動させることはできるのですか？

> 
> どなたかご存知でしたら、教えてください。

　以下の手順で出来ると思います。
　ただし、やる前に容量のチェックは怠らないように…

# 調査 容量
% df -h
% du -sh /usr/local/mysql

# 停止 MySQL
% mysql stop

# 移動 /usr2/mysql
% mv /usr/local/mysql /usr2

# 作成 シンボリックリンク
% ln -s /usr2/mysql /usr/local/mysql

# 開始 MySQL
% mysql start
