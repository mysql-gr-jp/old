Subject: [mysql 13636] MySQLが突然停止する

株式会社ビービットの玉越と申します。

mysqlサーバが突然停止することが頻繁に起こり、非常に困っております。
もし原因や解決法などをご存知の方がいらっしゃいましたらご教示いただけます
と幸いです。

■ハードウェア
・サードウェーブ
  ExPrime Server
  T-75-ECO
  D-XELT2SATA-SU7520

・Intel Xeon LV 2GHz
・メモリ 2GB
・HDD 2TB(RAID5構成)

■OS
・Fedora Core 5（5の最新版にはアップデート済み）
  kernel-2.6.18-2257

■MySQL
・5.0.27

■現象
・今までに何回か停止し、その都度現象が異なりました。

  ●1回目
　　停止直前に観測できたロードアベレージは2.39
　　筐体温度はそれほど高くない。
　　マウスカーソルは動くため、X server, kernelは生きている。
　　キーボード入力やマウスクリックは反応なし。
　　外部からのpingは応答なし。

  ●2回目
　　筐体温度はそれほど高くない。
　　画面に何も表示されていない。
　　外部からのpingは応答なし。

  ●3回目
　　mysqldは起動したまま、接続を受け付けない。
　　mysql-administratorの画面からサーバ停止をしても受け付けない。
　　コマンドプロンプトにてmysqladmin shutdownで停止。
　　OS再起動後は通常通り稼動。



　最初は熱暴走かと思いましたが、動いている部分もあるためそれだけではなさ
そうです。
　停止した原因の調査方法などもご存知でしたらご教示いただけますと幸いです。
