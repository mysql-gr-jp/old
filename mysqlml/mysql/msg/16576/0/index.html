MySQL NDB Clusterは、2020年1月に最新バージョン8.0がGAリリースとなりました。

MySQL NDB Clusterは開発当初は通信インフラ向けのインメモリデータベースでしたが、
その後の機能強化によりFintechサービスの基盤やオンラインゲームのバックエンド、
さらには軍事防衛システムなど高い信頼性と性能を求められるシステムでの採用が
広がっています。
