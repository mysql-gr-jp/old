結局，下記のupdate.batをWindowsのタスクスケジューラに入れることでうまく
運用できるようになりました．

もっとスマートな方法があるはずですし，タスクスケジューラだと一日一回しか
更新できないのが悔しいですが，自分なりにできる範囲でなんとか目的は達成で
きたといったところです．

まずは御礼申し上げます．今度ともよろしくお願い致します。


update.bat----------------------------------------------------------
dir /s /b Y:\ > C:\mysql\bin\y_filelist.txt
perl rplScript.pl
mysql filedata < batch.sql
mysql -e "LOAD DATA INFILE 'c:/mysql/bin/y_filedata.txt' REPLACE INTO TABLE t_url" filedata

y_filelist.txt------------------------------------------------------

Y:\hoge\hoge\foo\bar1.txt
Y:\hoge\hoge\foo\bar2.txt
Y:\hoge\hoge\foo\bar3.txt
        :

y_filedata.txt------------------------------------------------------

hoge\hoge\foo\bar1.txt
hoge\hoge\foo\bar2.txt
hoge\hoge\foo\bar3.txt
        :

rplScript.plc
open(INPUT, "y_filelist.txt");
open(OUTPUT, ">y_filedata.txt");
while (<INPUT>) {
		s/\\/\\\\/gi;
		s/^y:\\\\//gi;
		{
		print OUTPUT;
		}
}
close(OUTPUT);


batch.sql-----------------------------------------------------------

drop table t_url;
create table t_url(fullpath text);
