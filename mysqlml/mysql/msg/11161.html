MySQL 4.1.10a と 4.0.24 のリリース

MySQL 4.1.10aと MySQL 4.0.24の新しい版がリリースされました。
数多くのプラットフォーム用のソースとバイナリは以下のサイトやミラーから
ダウンロードできます。http://dev.mysql.com/downloads/
全てのミラーサイトがアップデートされているわけではありません
ので、後でチェックするか他のサイトをお試しください。

4.1.10a と 4.0.24のリリースは最近発見されたテンポラリー
テーブルのファイル名の生成とUser Defined Functions (UDFs)
の処理の際に生ずるセキュリティの弱点に対するフィックス
を含みます。Stefano Di Paola 氏(stefano.dipaola@xxxxxxxxxx)にこの
問題を発見しレポートしてくれたことに感謝いたします。

この変更はUser Defined Functions (UDF)がロードされる
方法に影響があります。マニュアルの
"User-defined Function Security Precautions" の
章を参照してください。
http://dev.mysql.com/doc/mysql/en/udf-security.html

MySQL 4.0.24リリースは保守フィックスも含みます。
詳細は http://bugs.mysql.com/
