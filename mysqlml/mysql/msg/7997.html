MySQL では join句に副問い合わせを使えません。
http://www.mysql.com/doc/en/JOIN.html

直接、対象となるカラムのカウント数を求めてはいかがでしょう？

select
  ATBL.*,
  count(BCNT.Code) as cnt
from
  ATBL
  left join BTBL as BCNT on ATBL.Code=BCNT.Code -- using(Code)
group by
  ATBL.code
;
