> 物の本には違う書き方をすればよろしいとあります。サブクエリーを使わないで
> も同様のことはできると思いますが。
　あっ　でも delete になると話が変わりますよね？
　さきほどの例では、selectで出してしまったのですが、実はdeleteで使いたかったの
です。

# delete分になると・・・

 delete from TBL2
 where ID in ( select ID from TBL1 where MID = 'target' )

　んで、MySQL だと・・・

 delete from TBL2,TBL1
 where TBL2.ID == TBL1.ID
   and TBL1.MID == 'target';

　だと、from TBL2,TBL1 の時点でだめ・・・
