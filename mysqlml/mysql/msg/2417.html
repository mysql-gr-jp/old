> explain の表示で 3.22 と 3.23 とでは、Extra 以外には違いは無いの
> でしょうか？

3.22のほうでは key_lenがnullとでますが、3.23の場合は数値が入ります。
その他の違いはありませんでした。



> 
> あとは、mysqladmin variables の出力結果を比べてみるとか、
> SHOW COLUMNS FROM table や SHOW INDEX FROM table の結果を比べてみるとか。


SHOW INDEX FROM table
を実行し、見比べてみると、
3.23のほうは全ての項目のCardinalityに数値が入っていますが、
3.22ではPRIMARYだけに数字が入っています。これは総件数でした。
