MySQL 4.1.x の問題と仮定した場合、経験的に次のようなことをチェックすれ
ば良いと思います。

・「show create table テーブル名」クエリを実行して、
  「DEFAULT CHARSET=ujis」になっているかどうかを調べる。

・「show variables」クエリを実行して、character_set_* とか collation_* 
  が ujis になっているかどうかを調べる。

・プログラムから使うときは、コネクション確立後に「set names ujis」クエ
  リを一回実行してみる。
