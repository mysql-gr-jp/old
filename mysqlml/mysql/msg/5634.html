もう少し懇切丁寧に説明すると。

> Warning: Supplied argument is not a valid MySQL result resource in
> /home/xxxx/public_html/webdb/simple.php on line 16
このエラーの意味は、「$MyLinkが有効な変数でない」
よってその前段階のコネクトで失敗している。と推測される。
そこでソースを見ると
> $MyLink = mysql_connect("localhost","X22","xxxx");
となっているので、これでは説明通り、ローカルホストでは
つながるが、他の環境ではエラーになるのはあたりまえ。

と言う事です。
