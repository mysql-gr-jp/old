goungoun様の「ローカル接続」というキーワードをヒントに
とりあえず問題が解決できました。
皆さんありがとうございました。

root権限でログインして、
mysql> select user,host from mysql.user;

よりユーザーテーブルを調べたところ、
+--------+--------------+
| user   | host         |
+--------+--------------+
| spadmi | %            |
| spadmi | 192.168.11.4 |
+--------+--------------+

と何故か、"spadmin"ではなく、"spadmi"という名称で作成されていることが
確認できました。
よって、spadmiというユーザ名からリモート接続を行ったところ、
正常に接続が行われました。

再度GRANT宣言で"spadminで何度か作成しましたが、"apadmi"となってしまいます。
