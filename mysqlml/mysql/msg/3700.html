php インストール時の、apxs による httpd.conf の変更項目は、

# cp -p /etc/httpd/conf/httpd.conf /etc/httpd/conf/httpd.conf.orig

などとしたあとで PHP を make install し、

$ diff -u /etc/httpd/conf/httpd.conf.orig /etc/httpd/conf/httpd.conf

すれば見られますよ。
