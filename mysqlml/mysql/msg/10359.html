MySQL 4.1 の製品版がリリースされました。
http://www.techweb.com/wire/software/51200864
2004年10月27日

MySQL AB は今日「製品版」のラベルを最新版のオープンソース
データベースサーバーに貼り付け出荷しました。4.1版の
新しい機能はサブクエリを通じての高度なクエリ機能、より
速くよりセキュアなクライアント・サーバ通信、ウインドウ・Linux用
のインストレーションとコンフィギュレーションのウイザード、
国際文字との地理的なデータのサポートだとMySQLは発表しました。

CEOのMarten Mickosは「私たちのコミュニティは何ヶ月にも
わたって、MySQL 4.1をテストしてきました。何百というビジネスが
この版用のプロダクション　アプリケーションを展開
しています。」と語った。

MySQL 4.1 がプロダクション品質であるということは,MySQLが
企業全般に展開でき、ミッション・クリティカル
なアプリケーションをサポートするのに十分に信頼性が
あると保障することです。

製品版のMySQL 4.1はWindows, Linux, Mac OS X, Solaris, FreeBSD,
HP-UX, IBM AIX, および他のOS上で作動します。データベースはオープンソース
のライセンスで、MySQLのサイトから無料でダウンロードできます。
サポートを含む、コマーシャル・ライセンスはサーバー毎に$595です。
