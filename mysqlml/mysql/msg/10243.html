MySQLの応用：小売、テレコム、旅行やホスピタリティ産業
MySQLは産業の枠を超えて競争に打つ勝つアプリを提供するお手伝い
をします。特に、小売、テレコム、旅行やホスピタリティ産業
にユニークな利点を提供します。

アメリカのスズキやSabre がどのようにMySQL使用しているか
学んでください。http://www.mysql.com/industry/

事例: craigslistは数百万個の広告を提供するのにMySQLを使います。
craigslist はアイディア、商品やサービスを交換するのを
促進するオンライン・コミュニティを作る際にその容易さと
速さでこの業界に革命を起こしました。Alexa
Traffic Rankingsによれば, craigslistは世界中でウエブのアクセス
では上位100番内に入るそうです。毎月craigslist には500万人の
異なった人からアクセスがあり、アクセスは全部で10億のページになります。
わずか４年のうちに、MySQLを使って当初25万ぺージだったものが
1日につき2700万ページのヒットとなりました。そのうちの殆ど
がMySQLを使って作られたものです。

詳細は
http://www.mysql.com/it-resources/case-studies/mysql-craigslist-casestudy.pdf

事例: EviteはMySQLを使って何百万の招待状を発送。
Eviteのウエブサイトは6百万の顧客を持ち、毎月9百万の
招待状を発行。 Evite はオーラクルを使用して４年間にわたりデータ管理の全ての
要求を満たしてきました。しかし、オーラクルのソフトが高価
であまり経験のないスタッフにとっては使用が困難である
ということが分かってきました。わずか１年でMySQL
は、高価なプロプリエタリの解の代わりに、EviteのＩＴのキーとなりました。

詳細は
http://www.mysql.com/it-resources/case-studies/mysql-evite-casestudy.pdf

MySQLの認定ホスティング - 特別
MySQL ホスティングはウエブ、メールやMySQLのデータベースの
アプリケーションをホストするには信頼性の高く、セキュアーでお得な
方法です。またこのホスティングだけがMySQLに認定されています。
LAMPスタックに基づいて構築され、MySQL用にセットされた高い
アベイラビリティのホストティングの構造で、あなたの
アプリのアップタイムを99.9%まで保障します。

今サインアップすると1年間のドメイン・ネームの登録が無料。

MySQL (4.1.5 Gamma) の新しいリリース

これは 4.1 gamma の２度目のリリースで製品版としての
リリースを近い将来に見据えて主に最近発見された
バグの修正版。

内容の１部は:

InnoDB: コンフィギュレーションのオプションである
innodb_autoextend_increment を追加して、InnoDB tablespace
が一杯になった時に拡張するサイズをメガバイトの大きさで
指定できるようになった。デフォルトは 8でこれは前の版で
8MBごとにしか設定できなかった8MBに対応。

WindowsインストラーのパッケージはWindows Installer (MSI)を利用
して新しいMySQL サーバー・インスタンス・コンフィギュレーション・ウイザード
を利用。これを使えば簡単にオプティマイズされたサーバー構築
が可能。

詳細はhttp://lists.mysql.com/announce/222
日本語は：　http://lists.mysql.com/mysql-ja/68

ソース・バイナリのダウンロードは
http://dev.mysql.com/downloads/mysql/4.1.html

MySQL Connector/J (3.1.4 ベータ) の新しいリリース
これは２回目のベータリリースでMySQL-4.1 または MySQL-5.0と
一緒に使用するのに適します。(MySQL-4.1.2かそれ以降
でサーバー側のprepared statementand を使う際は必要です。）
このリリースは：

VIEW用（5.0より）に DatabaseMetaData.getTables()とgetTableTypes() に
サポートを加えた。
'useOnlyServerErrorMessages'プロパティを追加、これはサーバから
送られたテキストのみを含むためにサーバが生成した
メッセージのテキストを作成。（これはSQLState's
'standard' の表現で、サーバーのエラーメッセージが続くのではなくて。）
このプロパティはデフォルトでは'true'

http://lists.mysql.com/announce/216
ダウンロードは：http://dev.mysql.com/downloads/connector/j/3.1.html

MySQL Connector/MXJ (1.0.5 アルファ）リリース
MySQL Connector/MXJ はMySQLを展開・管理するためのJMX MBeanです。
ユーザーのインストレーションの問題を軽減することで、Javaの開発者
が容易にデータベースを使用するアプリを展開できます。
Connector/MXJ はJBoss第４版などのJMX v1.2 準拠のMBeanコンテイナー が
必要です。JMX MBeanとして, Connector/MXJ はMySQL データベースをJavaベースの
コンポーネントのように扱います。

1.0.5-アルファ版での新しい機能は、スタートの際にMBeanがMySQLのデータベース
を立ち上げます。
またこのリリースでは、既にJBossをお使いでしたら、JBossライフサイクル
拡張方法を使って, JBossがシャットダウンした時にデータベースも
自動的に問題なくシャットダウンします。

http://lists.mysql.com/announce/221

ダウンロード：http://dev.mysql.com/downloads/connector/mxj/1.0.html

マイグレーション・ホワイトペーパ: AmazonTech's Flight Operations Management 
Solution
でMySQL(SQL Server から MySQL)にマイグレートした実例

FPAssistant Flight Operations Management (FOM)は
AmazonTechの全てのフライトのディスパッチの活動を１つの
使いやすい地図をベースにしたシステムにまとめる解です。
地図ベースの解は飛行機の管理、フライトの予定、ディスパッチの
コントロールとフライトのトラックも行います。このホワイトペーパは
FPAssistant FOMのデータを人気があるがライセンスの制限の
大きいマイクロソフトのSQLサーバ2000からMySQLにマイグレートする
に必要であった全てのステップに関して述べます。

http://www.mysql.com/it-resources/white-papers/mysql_amazontech_mssql2mysql_paper.pdf
