たてやんっす

TurboLinux 8 WorkStation の mysql を試してみました
バージョンはMySQL-3.23.52-3 でした

私の環境でも　local (UNIX socket) は接続ＯＫなのですが
TCP での接続をすると　Stack Dump で同じようにエラーになること確認しまし
た。(TurboLinux8 におけるバグかなぁ？）

ということで、 www.mysql.com から　rpm をダウンロードして
インストールしたところ正常に接続できることを確認できました
http://www.mysql.com/downloads/mysql-3.23.html

アップデート ( -Uvh ) が使えなかったので　アンインストール
rpm -e MySQL MySQL-shared MySQL-client MySQL-devel
してから、インストールしなおしました
rpm -ivh MySQL-3.23.56-1.i386.rpm MySQL-devel-3.23.56-1.i386.rpm \
 MySQL-shared-3.23.56-1.i386.rpm MySQL-client-3.23.56-1.i386.rpm

MySQL-devel-3.23.56-1.i386.rpm　は必要に応じて（ツールをコンパイルすると
きとか必要になります）
