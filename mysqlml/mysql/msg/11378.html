Subject: [mysql 11378] SUSE　Linux での　MySQL/Rubyの設定

遠藤と申します。

SUSE Linux9.3でデフォルトでインストールされる　MySQL4.1.10でMySQL/Rubyを使い
たいと考えています。
常識的な言葉なのだと思いますが、ヘッダファイル　と　ライブラリファイル　が具
体的にどのファイルをさすかがわからず立ち往生しています。

具体的には
MySQL/Rubyインストールの際に必要となる下記のパラメータの設定方法がわかりませ
ん。
ファイルの検索はできますのでヘッダファイル　と　ライブラリファイルそれぞれに
ついて具体的な事例を一個づつ教えていただけると検索して下記の設定ができます。

1.
--with-mysql-include=dir
MySQL のへッダファイルの場所として /usr/local/include の代わりのディレクトリ
を指定します。

2.
--with-mysql-lib=dir
MySQL のライブラリの場所として /usr/local/lib の代わりのディレクトリを指定し
ます。

なにとぞよろしくお願いします。
