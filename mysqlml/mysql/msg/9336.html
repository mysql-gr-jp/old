MySQL-server-4.1.1-1 付属の mysql.help_* テーブルではとみたさんの例の
help create index が表示されなかったのですが、snapshot のを拝借して
help テーブルを作り直したら表示されるようになりました。

  $ mysql -uroot mysql < mysql-4.1.2-alpha-nightly-20040509/scripts/fill_help_tables.sql

  mysql> help create
  Many help items for your request exist
  For more specific request please type 'help <item>' where item is one of next
  topics :
     CREATE DATABASE
     CREATE INDEX
     CREATE TABLE
     FUNCTION
     SHOW
     SPATIAL
  
  mysql> help create index
  Name: 'CREATE INDEX'
  Description:
  In MySQL 3.22 or later, CREATE INDEX is mapped to an
  ALTER TABLE statement to create indexes.
  (snip)
