試してみましたが、

LoadLibraryなどの使い方も判らなかったりしています。

そうしますと、やはり無謀な挑戦と言うか、VC++を所有していない
Windowsユーザーは、やってはいけなかったりするのでしょうか？
従って、もしそうおっしゃるのでしたらあきらめます。

やっているだけですので、返事が遅くなってもいっこうに気にしません。
#だからピンクの熊で有名なプロバイダのように43日後に返事が来ても
#特に問題はありませんです。

因みに、以下がArp3のソースコードです。

zip char(8),address varchar(50)
の構成となっております。

//---------------------------------------------------------------------------
#include <mysql/mysql.h>
#include <stdio.h>
int main()
{
/* (1) MYSQLの定義 */
	MYSQL mysql;
	MYSQL *mysqldb = NULL;
	MYSQL_RES *result;
	/* (2) MYSQLに接続 */
	mysqldb =
	mysql_connect(&mysql,
	 "192.168.0.1", "yusuke", "yusuke");
	if (!mysqldb) {
		puts("Connection Error.");
		exit(1);
	}
	/* (3) データベースを選択 */
	mysql_select_db(mysqldb, "db1");
	/* (4) ＳＱＬ文を発行 */
	mysql_query(
	mysqldb,
	"insert into postnumber values ('104-0061','東京都中央区銀座')");
}
