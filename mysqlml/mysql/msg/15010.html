坂井です

  今月も月初に MySQL 5.1 の最新バージョンがリリースされました。
  http://www.mysql.gr.jp/frame/modules/news/article.php?storyid=147

  今回のリリースには "InnoDB Plugin" が含まれています。
  
  MySQL 5.1 からはストレージエンジンもプラグイン可能になり、
  様々な人がストレージエンジンを開発しています（Spiderエンジンとか）。
  しかし InnoDB は従来通り MySQL本体に組み込まれていました。
  その InnoDB をもプラグイン化しよう、というのがこの InnoDB Plugin 
  です。
  従来のビルトイン(組み込み)のInnoDBももちろん従来通り含まれていま
  すし、InnoDB Plugin はまだベータレベルだとアナウンスされています
  ので実環境投入はもう少し先かな、と個人的には思いますが、ビルトイン
  型にはない試みもあるようなので期待したいと思います。
  
  InnoDBプラグイン 1.0.4 (今回含まれているもの) については松信さん
  の以下のブログも詳しいです。
  http://opendatabaselife.blogspot.com/2009/08/innodb-plugin-104-innodb.html
  
  
  その他 InnoDB プラグインについては、詳しい方ぜひフォローくださいm(_ _)m


Thank you
