苦心なさっておいでのようですが、普通のupdate文で一発でできますよ。
resttimeへの登録と件数カウントはこんなかんじかな。↓

$sql = "update faq_table set resttime = datediff(kigen, kyou) where kigen <> 0";
if (!mysql_query($sql)) {
  echo "期限登録失敗";
} else if (mysql_affected_rows() == 0) {
  echo "該当レコード無し";
} else {
  echo "期限を登録";
}
